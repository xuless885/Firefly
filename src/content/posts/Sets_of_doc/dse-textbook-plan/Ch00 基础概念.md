## 0.1　数理逻辑

### 0.1.1　定义

- **定义 (Definition)：** 对某个概念赋予陈述句进行解释。
- **对象 (Object)：** 不构成陈述句的事物（词语、符号如 $x, 1, 2, \dots$）。

核心术语：公理、定理、推论、引理。

### 0.1.2　命题逻辑

#### 命题与开放命题

- **命题 (Proposition)：** 一个客观描述只可能为真或假的陈述性语句。
- **开放命题：** 包含变量（可自由取值）的命题，记作 $P(x, y, \dots)$。赋予变量具体值前无真假之分。
- **真值：** 真命题记 $\mathbb{T}$，假命题记 $\mathbb{F}$。

#### 等号

- **等式：** $A = B$ 断言两对象一致，本身构成命题。$A \neq B$ 断言不一致。
- “$=$”是作用于对象上的工具。两命题逻辑上相同称为“等价”，不用“$=$”表示。

### 0.1.3　命题逻辑的运算

设 $P, Q$ 为命题。

| 运算 | 符号 | 读法 | 真值条件 |
|------|------|------|----------|
| 否定 | $\neg P$ | 非 $P$ | 与 $P$ 相反 |
| 合取 | $P \land Q$ | $P$ 及 $Q$ | 两者皆真才真 |
| 析取 | $P \lor Q$ | $P$ 或 $Q$ | 至少一真即真 |
| 条件句 | $P \to Q$ | 若 $P$，则 $Q$ | 仅 $P$ 真且 $Q$ 假时为假 |
| 双条件 | $P \leftrightarrow Q$ | $P$ 当且仅当 $Q$ | 两者真值一致时为真 |

**逻辑蕴含：** $P \implies Q$ 当 $P \to Q = \mathbb{T}$（$P$ 成立时 $Q$ 必成立）。

**逻辑等价：** $P \iff Q$ 当 $(P \to Q) \land (Q \to P) = \mathbb{T}$，即两命题真假一致。

### 0.1.4　全称量词与存在量词

设 $P(x)$ 为开放命题。

| 量词 | 符号 | 含义 | 何时为假 |
|------|------|------|----------|
| 全称量词 | $\forall x, P(x)$ | 对所有 $x$，$P(x)$ 为真 | 存在一个反例 |
| 存在量词 | $\exists x, P(x)$ | 存在某些 $x$ 使 $P(x)$ 为真 | 没有任何对象能使命题成立 |

### 0.1.5　现代数学的规则

#### 公理

- **公理 (Axiom)：** 被定义为真命题的命题。

**公理 0.1.1（等式公理）：** 对任何对象 $a, b, c$：
1. **反身性：** $a = a$
2. **对称性：** $a = b \iff b = a$
3. **传递性：** $(a = b) \land (b = c) \implies a = c$（可简写为 $a = b = c$）

#### 莱布尼茨定律

- **公理 0.1.2（同一者不可分原理）：** $\forall x, y$，$(x = y) \implies (P(x) \iff P(y))$

两个相同的对象具有完全相同的性质。凡对 $x$ 成立的命题，对等于 $x$ 的 $y$ 也同样成立。

### 0.1.6　定理与数学证明

- **定理 (Theorem)：** 在既定公理下的真命题。
- **数学证明 (Proof)：** 基于公理和已证定理推导出命题为真的逻辑过程。
- 定理通常写为：$(\text{前提}) \implies (\text{结论})$

**定理 0.1.2（假言三段论）：** 对命题 $X, Y, Z$，
$$(X \to Y) \land (Y \to Z) \implies (X \to Z)$$

- **证明：** 列出 $X, Y, Z$ 所有 $2^3 = 8$ 种真值组合，计算 $(X \to Y) \land (Y \to Z)$ 与 $(X \to Z)$ 的真值：

  | $X$ | $Y$ | $Z$ | $(X \to Y) \land (Y \to Z)$ | $X \to Z$ |
  |:---:|:---:|:---:|:---------------------------:|:---------:|
  | $\mathbb{T}$ | $\mathbb{T}$ | $\mathbb{T}$ | $\mathbb{T}$ | $\mathbb{T}$ |
  | $\mathbb{T}$ | $\mathbb{T}$ | $\mathbb{F}$ | $\mathbb{F}$ | $\mathbb{F}$ |
  | $\mathbb{T}$ | $\mathbb{F}$ | $\mathbb{T}$ | $\mathbb{F}$ | $\mathbb{T}$ |
  | $\mathbb{F}$ | $\mathbb{T}$ | $\mathbb{T}$ | $\mathbb{T}$ | $\mathbb{T}$ |
  | $\mathbb{T}$ | $\mathbb{F}$ | $\mathbb{F}$ | $\mathbb{F}$ | $\mathbb{F}$ |
  | $\mathbb{F}$ | $\mathbb{T}$ | $\mathbb{F}$ | $\mathbb{F}$ | $\mathbb{T}$ |
  | $\mathbb{F}$ | $\mathbb{F}$ | $\mathbb{T}$ | $\mathbb{T}$ | $\mathbb{T}$ |
  | $\mathbb{F}$ | $\mathbb{F}$ | $\mathbb{F}$ | $\mathbb{T}$ | $\mathbb{T}$ |

  观察：当前提 $(X \to Y) \land (Y \to Z) = \mathbb{T}$ 时（第 1、4、7、8 行），结论 $X \to Z$ 均为 $\mathbb{T}$。故 $((X \to Y) \land (Y \to Z)) \to (X \to Z)$ 恒真，即逻辑蕴含成立。$\square$

- **推论 (Corollary)：** 从定理直接推出的结论。
- **引理 (Lemma)：** 为证明主要定理而准备的辅助命题。
- **撰写证明的要点：** 从前提出发，每一步依据公理或已证定理，逐步推向结论。证明链的核心是 $\implies$ 的传递：若 $A \implies B$ 且 $B \implies C$，则 $A \implies C$。

## 0.2　基础算术

### 0.2.1　自然数

- **自然数 (Natural Number)：** 全体正整数 $\{1, 2, 3, \dots\}$（亦常包含 $0$）。源于计数行为。

### 0.2.2　数的运算

- 算式：只含有自然数及其运算的对象。“$=$”是逻辑符号而非运算。

#### 加法与减法

**公理 0.2.1（自然数加法公理）：** 对任意自然数 $a, b, c$：
1. **交换律：** $a + b = b + a$
2. **结合律：** $(a + b) + c = a + (b + c)$
3. **单位元：** $a + 0 = a$

- **减号定义：** $a = b + c \iff c = a - b$，称 $c$ 为 $a$ 与 $b$ 之差。同理 $b = a - c$。

**定理 0.2.1：** $a - a = 0$

- **证明：** 据减号定义，
  $$a - a = 0 \iff a = a + 0$$
  右端 $a = a + 0$ 由公理 0.2.1(3)（$a + 0 = a$）及等式反身性（$a = a$）知恒为真命题。故其等价命题 $a - a = 0$ 亦恒真。$\square$

**推论 0.2.1：** $0 + a = a$

- **证明：** 由公理 0.2.1(1)（加法交换律），
  $$0 + a = a + 0$$
  再由公理 0.2.1(3)（加法单位元），$a + 0 = a$。故 $0 + a = a$。$\square$

#### 乘法与除法

**公理 0.2.2：** 对任意自然数 $a, b, c$：
1. **交换律：** $a \times b = b \times a$
2. **单位元：** $a \times 1 = 1 \times a = a$
3. **结合律：** $(ab) \times c = a \times (bc)$
4. **分配律：** $a \times (b + c) = a \times b + a \times c$

- **整除定义：** 若 $n = p \times q$（$p, q \neq 0$），则 $n \div p = q$，记为 $p \mid n$。$0$ 可被所有非零自然数整除。

**推论 0.2.2：** 若 $n = p \times q$，则 $p, q$ 均为 $n$ 的因数。

- **证明：** 由整除定义，$n = p \times q$ 对应 $p \mid n$ 及 $q \mid n$。按因数定义（若 $p \mid n$ 则 $p$ 为 $n$ 的因数），即得 $p, q$ 均为 $n$ 的因数。$\square$

#### 乘方的记号

- 乘法记号：已知数之间用 $\times$ 或 $\cdot$；未知量之间可省略（如 $2x$、$nxyz$）。
- **乘方：** $a^n = \underbrace{a \cdot a \cdots a}_{n \text{ 个 } a}$
- $n^2$ 称平方，$n^3$ 称立方。
- $a^m \times a^n = a^{m+n}$
- $n^0 = 1$（$n \neq 0$）

#### 公因数与公倍数

- **因数：** 若 $p \mid n$，则 $p$ 为 $n$ 的因数（因子）。
- **公因数：** $p$ 同时整除 $n, m$。
- **H.C.F.（最大公因数）：** $\gcd(m, n)$，$n, m$ 所有公因数中最大者。
- **倍数：** 若 $p \mid n$，则 $n$ 为 $p$ 的倍数。
- **公倍数：** $n$ 同时为 $p, q$ 的倍数。
- **L.C.M.（最小公倍数）：** $\operatorname{lcm}(m, n)$，$n, m$ 所有公倍数中最小者。
- 计算方法：短除法。

#### 四则运算

- **四则运算**即加法、减法、乘法、除法四种基本运算。
- **表示法：** 乘法可用 $a \times b$、$a \cdot b$、$ab$；除法可用 $a \div b$、$a/b$、$\frac{a}{b}$。
- **运算优先级：** 先乘除、后加减；有括号先算括号内。

## 0.3　基础代数

- **核心区分：** 在代数中，**对象**与**命题**的界线尤为重要——代数式本身不含真假，而等式和方程则构成可判真假的命题。

### 0.3.1　对象与命题在代数中的体现

- **变量 (Variable)：** 用来表示未知数或可变数值的符号，通常用字母（$x, y, z, \dots$）表示。变量本身是一个**对象**。
- **代数式 (Algebraic Expression)：** 由数、变量和运算符号组成的式子（如 $3x + 5y - 2$）。代数式是一个**对象**——它不含真假，就如同词语“猫”不是陈述句一样。
- **等式 (Equality)：** 以“$=$”连接两个对象（代数式），断言两者在数值上一致。等式是一个**命题**（如同 $A = B$），具有真值，可判真假。
- **方程 (Equation)：** 含有变量的等式，即一个**开放命题**。例如 $2x - 5 = 11$ 是形如 $P(x)$ 的开放命题，其真值取决于 $x$ 的取值。
- **解方程 (Solving an Equation)：** 寻找能使该开放命题为真的变量取值。即求 $x$ 使得 $P(x) = \mathbb{T}$。解法核心是运用等式性质将未知数逐步分离。

- 回顾 0.1 的框架：代数式 $\leftrightarrow$ 对象，等式 $\leftrightarrow$ 命题，方程 $\leftrightarrow$ 开放命题，解方程 $\leftrightarrow$ 判断开放命题何时为真。

### 0.3.2　等式的性质

**定理 0.3.1（等式的性质）：**

1. **加减法不变性：** $a = b \implies a + c = b + c$
2. **乘除法不变性：** 若 $c \neq 0$，则 $a = b \iff ac = bc$

- **证明（加减法不变性）：** 设 $P(x)$ 为命题“$x + c = a + c$”。由莱布尼茨定律，$a = b \implies (P(a) \iff P(b))$，即
  $$(a + c = a + c) \iff (b + c = a + c)$$
  左端 $a + c = a + c$ 由等式反身性恒真，故右端 $b + c = a + c$ 亦真，即 $a + c = b + c$。$\square$

- **证明（乘除法不变性，正向）：** 设 $P(x)$ 为命题“$xc = ac$”。由莱布尼茨定律，$a = b \implies (ac = ac \iff bc = ac)$。左端由反身性恒真，故 $ac = bc$。$\square$

- **证明（乘除法不变性，反向）：** 设 $ac = bc$ 且 $c \neq 0$。由正向结论，以 $c$ 为乘数、两端分别除以 $c$ 即得 $a = b$。或以莱布尼茨定律：设 $Q(x)$ 为命题“$xc = bc$”，则由 $ac = bc$ 知 $Q(a) = \mathbb{T}$，而 $bc = bc$，代入可得 $a = b$。$\square$

### 0.3.3　简易代数运算

- 代数式中的四则运算同样遵循交换律、结合律、分配律。
- **代入法 (Substitution)：** 将具体数值或代数式替换公式中对应的变量，从而计算结果的过程。本质上是将开放命题中的变量赋予特定对象。

## 0.4　物理量与单位表示

- **物理量 (Physical Quantity)：** 由数值和单位共同组成（如 5 米、20 千克）。
- 不同单位的量不能直接加减（如 $3\text{ kg} + 5\text{ cm}$ 无意义）。
- 运算时单位也需参与：速度 $(\text{m/s}) \times$ 时间 $(\text{s}) =$ 路程 $(\text{m})$。

---

- **本章总结：** 命题 → 逻辑运算 → 公理 → 定理与证明。在此框架上，自然数的公理化定义支撑起算术四则运算，代数则将数推广为变量，以等式性质为核心工具。

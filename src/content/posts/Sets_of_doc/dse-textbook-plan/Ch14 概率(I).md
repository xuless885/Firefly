## 14.1 概率的基本概念 (Basic Concepts of Probability)

### 14.1.1 样本空间 (Sample Space)

- **定义 14.1 (随机试验 / Random Experiment)**
  一个**随机试验**是一个过程，其结果在试验进行前无法确定，但所有可能出现的结果是已知的。

- **定义 14.2 (样本空间 / Sample Space)**
  一个随机试验的**样本空间**，记为 $S$，是该试验所有可能结果的总体。
  - 例如：掷一枚硬币，$S$ 包含两个结果：正面（H）和反面（T）。
  - 例如：掷一枚六面骰子，$S$ 包含六个结果：$1, 2, 3, 4, 5, 6$。

- **定义 14.3 (事件 / Event)**
  样本空间 $S$ 中的**一部分结果**称为一个**事件**，记为 $E$。事件 $E$ 发生当且仅当试验的实际结果在 $E$ 所指的那部分结果之中。一个只包含单一结果的事件称为**基本事件**（Elementary Event）。

  其中两类极端情形尤为常用：
  - **必然事件 (Certain Event)** —— 包含样本空间中**所有**结果的事件。每次试验必定发生。
  - **不可能事件 (Impossible Event)** —— 不包含任何结果的事件。每次试验必定不发生。

- **方法 14.1 (列举样本空间 / Listing a Sample Space)**
  对于多步骤的随机试验，系统性地列举所有可能结果的方法：

  1. **列表法（Tabulation）：** 适用于两步骤试验。以第一步骤的结果为行、第二步骤的结果为列，列出所有有序对。例如，掷两枚骰子的样本空间可表示为 $6\times 6$ 的方格表，共 $36$ 个有序对 $(i, j)$，其中 $i, j$ 各可取 $1, 2, \dots, 6$。
  2. **树形图（Tree Diagram）：** 适用于三步骤或更多步骤的试验。从起点出发，每一步分支代表该步骤的可能结果，末端节点对应一个完整的试验结果序列。树形图的优点是能直观展示各步骤的先后顺序与分支结构，避免遗漏。

  **Remark:** 区分“有序”与“无序”的结果空间是常见陷阱。例如，同时掷两枚相同的硬币，若将“一正一反”与“两正”视为等可能则出错——必须意识到“一正一反”实际上对应两个有序结果 $(\text{H}, \text{T})$ 和 $(\text{T}, \text{H})$，而“两正”仅对应一个有序结果 $(\text{H}, \text{H})$。

### 14.1.2 古典概型 (Classical Probability Model)

- **定义 14.4 (古典概型 / Classical Probability Model)**
  一个随机试验若满足以下两个条件，则称为**古典概型**：
  1. **有限性（Finiteness）：** 样本空间 $S$ 中的结果数量是有限的。
  2. **等可能性（Equally Likely Outcomes）：** $S$ 中的每个基本结果发生的可能性相等。

  在古典概型下，对任何事件 $E$，其**概率** $P(E)$ 定义为：
  $$P(E) = \frac{n(E)}{n(S)} = \frac{\text{事件 } E \text{ 包含的结果数}}{\text{样本空间 } S \text{ 的结果总数}}$$

  其中 $n(E)$ 表示 $E$ 所包含的结果个数，$n(S)$ 表示 $S$ 所包含的结果总数。

- **定理 14.1 (概率的基本性质 / Basic Properties of Probability)**
  对于古典概型下的任意事件 $E$：
  1. $0 \le P(E) \le 1$。
  2. 不可能事件的概率为 $0$。
  3. 必然事件的概率为 $1$。

  **证明：**
  事件 $E$ 所包含的结果数 $n(E)$ 满足 $0 \le n(E) \le n(S)$，因此 $0 \le n(E)/n(S) \le 1$，即 $0 \le P(E) \le 1$。
  不可能事件不含任何结果，即 $n = 0$，故概率为 $0$。
  必然事件包含全部 $n(S)$ 个结果，即 $n(E) = n(S)$，故概率为 $1$。

## 14.2 期望值 (Expected Value)

- **定义 14.5 (期望值 / Expected Value)**
  设一个随机试验有 $k$ 种可能的结果，各结果对应的数值分别为 $x_1, x_2, \dots, x_k$，各结果发生的概率分别为 $p_1, p_2, \dots, p_k$（其中 $p_1 + p_2 + \cdots + p_k = 1$）。则该试验的**期望值** $E$ 定义为各数值与其对应概率的乘积之和：
  $$E = x_1 p_1 + x_2 p_2 + \cdots + x_k p_k = \sum_{i=1}^{k} x_i p_i$$

  **Remark:** 期望值本质上就是以概率为权重的**加权平均数**（见第 13 章定义 13.4）。它代表在大量重复该试验后的**长期平均结果**，而非单次试验的必然结果。换言之，期望值是一个“理论上的平均”，现实中的单次结果可能远离期望值。

- **定义 14.6 (公平游戏 / Fair Game)**
  考虑一个游戏：玩家支付成本 $c$ 元参与，游戏结果对应的回报金额的期望值为 $E$。则**净期望值**为 $E - c$（即回报期望值减去成本）。
  若净期望值为零，即 $E = c$，则称该游戏为**公平游戏**（Fair Game）。

  **Remark:** 若 $E < c$，游戏对玩家不利（unfavorable），长远而言玩家平均每次亏损；若 $E > c$，游戏对玩家有利（favorable）。现实中的商业游戏（如彩票、赌场）普遍设计为 $E < c$，以确保庄家的长期盈利。

## 第十四章总结

1. **样本空间** $S$ 是随机试验所有可能结果的总体。**事件** $E$ 是 $S$ 中的一部分结果。
2. **必然事件**包含全部结果，概率为 $1$；**不可能事件**不含任何结果，概率为 $0$。
3. **古典概型**需满足有限性与等可能性两个条件。在此条件下，$P(E) = n(E)/n(S)$。
4. 概率的基本性质：对任意事件 $E$，$0 \le P(E) \le 1$（定理 14.1）。
5. 列举样本空间的两种系统性方法：**列表法**（适用于两步骤试验）与**树形图**（适用于多步骤试验）（方法 14.1）。关键在于区分有序与无序结果。
6. **期望值** $E = \sum x_i p_i$ 是各结果数值以概率为权重的加权平均（与第 13 章的加权平均数概念一致），反映长期平均结果（定义 14.5）。
7. **公平游戏**的充要条件为回报期望值等于成本，即 $E = c$（定义 14.6）。

### 备课建议

1. **“等可能性”前提的强调：** 这是古典概型最核心的前提，也是学生最易出错的地方。务必用反例说明：例如“明天要么下雨、要么不下雨，所以下雨概率是 $1/2$”——错误，因为两种结果并非等可能。要求学生在每次计算概率前先陈述“等可能性”的依据。

2. **列表法与树形图的系统训练：** 对于两步骤试验（如掷两枚骰子），列表法直观且不易遗漏，建议优先教授。树形图用于三步骤或更多步骤时，注意引导学生逐层分支、在末端计数。可对比同一问题用两种方法的结果，以建立方法间的可互验性。

3. **有序 vs 无序的陷阱：** 同时掷两枚相同硬币时，许多初学者会错误地认为样本空间只有“两正、一正一反、两反”三种结果且等可能。需反复训练学生“想像硬币有不同颜色”的技巧——这样结果空间变成有序对，等可能性才成立。这是古典概型中最常见的错误思路，值得花时间专门处理。

4. **期望值的直观理解：** 学生常混淆“期望值”与“最可能值”——期望值甚至可能不是任何一个实际可能的取值（如掷一枚骰子的期望值是 $3.5$，但不可能掷出 $3.5$）。建议用“大量重复试验后的长期平均”的语言来建立直观，而非“期望下一次得到什么”。

5. **与后续章节的联系：** 本章的概率概念为高中第 35 章“概率(II)”中集合语言、加法定律、乘法定律和条件概率打下基础。期望值则为第 36 章“数列”中的应用（如分期付款现值计算）提供概率权重思维。教学时可暗示这种延伸关系，为后续章节铺垫。

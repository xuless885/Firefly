## 23.1 单变量多项式的记法与基本概念

### 23.1.1 单变量多项式的记法

回顾 Ch04：多项式由单项式的和构成。当多项式仅含一个变量时，即为**单变量多项式**。

- **定义 23.1 (单变量多项式 / Univariate Polynomial)**
  回顾 [Ch19 函数及其图像](/posts/ch19-函数及其图像/)，函数是集合间的对应 $f: A \to B$。一个**单变量多项式**是形如
  $$P: \mathbb{C} \to \mathbb{C}, \qquad P(z) = a_n z^n + a_{n-1} z^{n-1} + \cdots + a_1 z + a_0$$
  的函数，其中 $a_0, a_1, \dots, a_n \in \mathbb{C}$（或限定 $a_i \in \mathbb{R}$ 时为 $P: \mathbb{R} \to \mathbb{R}$）且 $n \in \mathbb{N}_0$。$a_i$ 为 $z^i$ 项的系数。全体单变量多项式构成的集合记为 $\mathbb{C}[z]$（或 $\mathbb{R}[z]$，取决于系数所属的域）。
  - 若 $a_n \neq 0$，则 $a_n z^n$ 称为**首项**（Leading Term），$a_n$ 为**首项系数**（Leading Coefficient）。
  - $a_0$ 为**常数项**。
  - 当所有 $a_i = 0$ 时，$P(z) = 0$ 称为**零多项式**（即常数函数 $0$）。

 > **将多项式视为函数的意义：** 多项式不只是一串系数的排列——它定义了一个从 $\mathbb{C}$（或 $\mathbb{R}$）到自身的映射。这使得「代入求值」$P(a)$ 成为函数求值的自然结果（而非代数式的形式替换），也解释了为何余式定理中 $P(a)$ 的计算与除法算法相容。日后在 [Ch25 指数函数及对数函数](/posts/ch25-指数函数及对数函数/) 等章节中，这种函数视角将统一各类函数的讨论。

  >**变量记号约定：** 本章以 $z$ 为通用变量，因所述定理对 $\mathbb{C}$ 和 $\mathbb{R}$ 均成立。当上下文明确限定变量与系数均在 $\mathbb{R}$ 上时（如 §23.4.2b 的 $\mathbb{R}$ 专属讨论），约定将 $z$ 写回 $x$，即 $P(x) \in \mathbb{R}[x]$，以示此时仅在实数域中考虑。

  **Remark:** 单变量多项式是多项式（定义 4.5）取变量个数为 $1$ 的特殊情形。定义 4.9 的次数在此简化为：非零多项式的次数为 $z$ 的最高幂次，记为 $\deg P$；零多项式定义 $\deg 0 = -1$。

- **定义 23.2 (首一多项式 / Monic Polynomial)**
  若非零单变量多项式 $P(z)$ 的首项系数为 $1$，即
  $$P(z) = z^n + a_{n-1} z^{n-1} + \cdots + a_1 z + a_0$$
  则称 $P(z)$ 为**首一多项式**。

  **Remark:** 任意非零单变量多项式 $P(z) = a_n z^n + \cdots + a_0$（$a_n \neq 0$）可通过提取首项系数化为首一形式：
  $$P(z) = a_n \left( z^n + \frac{a_{n-1}}{a_n} z^{n-1} + \cdots + \frac{a_0}{a_n} \right)$$
  在因式分解与 HCF/LCM 的讨论中，常将因式归一化为首一形式以便比较——这类似于整数质因数分解中取正质数的约定。

### 23.1.2 单变量多项式的相等

- **定理 23.2 (单变量多项式相等的等价条件)**
  设 $P(z) = \sum_{i=0}^n a_i z^i$，$Q(z) = \sum_{i=0}^m b_i z^i$ 均为最简表示。则 $P = Q$ 当且仅当：
  1. $n = m$（次数相同），及
  2. $a_i = b_i$ 对所有 $i$ 成立（各项系数相等）。

  **证明：** 此为定理 4.2（多项式相等的等价条件）在单变量情形下的特例，证明见 Ch04。□

## 23.2 单变量多项式的除法

> 类似整数的带余除法，单变量多项式也存在唯一的除法算法。这是本章的核心结构定理。

### 23.2.1 除法算法

- **定理 23.3 (多项式除法算法 / Division Algorithm for Polynomials)**
  设 $P(z), D(z)$ 为两个单变量多项式，$D(z) \neq 0$。则存在唯一的单变量多项式 $Q(z)$ 和 $R(z)$，使得
  $$P(z) = D(z) \cdot Q(z) + R(z)$$
  其中 $\deg R < \deg D$ 或 $R(z) = 0$。
  - $P(z)$ 称为**被除式**（Dividend），$D(z)$ 为**除式**（Divisor）。
  - $Q(z)$ 称为**商式**（Quotient），$R(z)$ 为**余式**（Remainder）。
  - 当 $R(z) = 0$ 时，$D(z)$ 为 $P(z)$ 的一个因式。（整除的正式定义见下文定义 23.3。）

  **证明（存在性）：** 对 $\deg P$ 进行归纳。
  - 基础情形：若 $\deg P < \deg D$，取 $Q = 0$，$R = P$ 即满足。
  - 归纳步骤：设 $\deg P \ge \deg D$，令 $p = \deg P$，$d = \deg D$。写出首项 $P(z) = a_p z^p + \cdots$，$D(z) = b_d z^d + \cdots$（$b_d \neq 0$）。构造
    $$Q_1(z) = \frac{a_p}{b_d} z^{p-d}$$
    令 $P_1(z) = P(z) - Q_1(z) D(z)$，则 $\deg P_1 < p$。对 $P_1$ 应用归纳假设，存在 $Q_2, R$ 使得 $P_1 = D Q_2 + R$（$\deg R < \deg D$）。令 $Q = Q_1 + Q_2$ 即得 $P = DQ + R$。

  **唯一性：** 假设 $P = D Q_1 + R_1 = D Q_2 + R_2$，其中 $\deg R_1, \deg R_2 < \deg D$（或为零）。则 $D(Q_1 - Q_2) = R_2 - R_1$。若 $Q_1 \neq Q_2$，则左边次数 $\ge \deg D$，但右边次数 $< \deg D$，矛盾。故 $Q_1 = Q_2$，进而 $R_1 = R_2$。□

- **定义 23.3 (整除 / Divisibility)**
  设 $P(z), D(z)$ 为单变量多项式，$D(z) \neq 0$。若存在多项式 $Q(z)$ 使得
  $$P(z) = D(z) \cdot Q(z)$$
  则称 $D(z)$ **整除** $P(z)$（或 $P(z)$ 可被 $D(z)$ 整除），记为 $D(z) \mid P(z)$。此时 $D(z)$ 称为 $P(z)$ 的**因式**（Factor），$P(z)$ 称为 $D(z)$ 的**倍式**（Multiple）。
  
  若不存在这样的 $Q(z)$，则记为 $D(z) \nmid P(z)$。

  **Remark:** 定理 23.3（除法算法）为整除判定提供了等价刻画：$D(z) \mid P(z)$ 当且仅当 $P(z)$ 除以 $D(z)$ 的余式 $R(z) = 0$。这与整数中 $a \mid b \iff b \div a$ 的余数为 $0$ 完全平行。

### 23.2.2 长除法

- **方法 (长除法 / Long Division)**
  利用除法算法的构造性证明，可按以下步骤执⾏多项式长除法：
  1. 将 $P(z)$ 和 $D(z)$ 按 $z$ 的降幂排列。
  2. 以 $P(z)$ 的首项除以 $D(z)$ 的首项，得商式的下一项。
  3. 将该项乘以 $D(z)$，从 $P(z)$ 中减去。
  4. 以所得新多项式为新的被除式，重复步骤 2–3，直至余式的次数小于 $D(z)$ 的次数。

  **Remark:** 长除法与整数长除法（如 $1234 \div 11$）结构完全平行——整数按“位值”（$10$ 的幂）排列，多项式按 $z$ 的幂排列。

## 23.3 余式定理

> 当除式为一次式 $(z-a)$ 时，除法算法有一个极为简洁的推论。

### 23.3.1 余式定理

- **定理 23.4 (余式定理 / Remainder Theorem)**
  设 $P(z)$ 为单变量多项式。$P(z)$ 除以 $(z-a)$ 的余式等于 $P(a)$。

  **证明：** 由除法算法，存在 $Q(z)$ 和常数 $R$，使得
  $$P(z) = (z-a) \cdot Q(z) + R$$
  （因 $\deg (z-a) = 1$，故 $\deg R < 1$，即 $R$ 为常数。）
  代入 $z = a$：
  $$P(a) = (a-a) \cdot Q(a) + R = 0 + R = R$$
  故 $R = P(a)$。□

- **推论 23.1**
  设 $P(z)$ 为单变量多项式，$a \neq 0$。$P(z)$ 除以 $(az-b)$ 的余式等于 $P\!\left(\dfrac{b}{a}\right)$。

  **证明：** $P(z) = (az-b) Q(z) + R = a\left(z-\dfrac{b}{a}\right) Q(z) + R$。代入 $z = \dfrac{b}{a}$ 得 $P\!\left(\dfrac{b}{a}\right) = R$。□

### 23.3.2 余式定理的意义

余式定理提供了一种无需执行长除法即可计算余式的方法——仅需代入 $z = a$ 计算 $P(a)$。这在除式为一次式时极大简化了计算。

## 23.4 因式与根的关系

> §23.4.1 建立因式定理——因式与根的基本双向等价。§23.4.2 深入讨论根的结构：重根、因式的整除性质、$\mathbb{R}$ 上的根数上界与结构定理，然后提升至 $\mathbb{C}$——代数基本定理、完全分解、共轭根定理、以及回到 $\mathbb{R}$ 的不可约因式分类。§23.4.3 将这些理论应用于系统化分解多项式。

###  23.4.1 因式定理

- **定理 23.5 (因式定理 / Factor Theorem)**
  设 $P(z)$ 为单变量多项式。$(z-a)$ 是 $P(z)$ 的因式（即 $(z-a) \mid P(z)$），当且仅当 $P(a) = 0$。

  **证明：** 由余式定理，$P(z)$ 除以 $(z-a)$ 的余式为 $P(a)$。因此：
  $$(z-a) \mid P(z) \iff \text{余式 } R = 0 \iff P(a) = 0$$
  □

- **定义 23.4 (多项式的根 / Root of a Polynomial)**
  若 $P(a) = 0$，则称 $a$ 为多项式 $P(z)$ 的一个**根**（或**零点**）。

  因式定理建立了根与因式之间的第一层联系——每个根对应一个一次因式。然而，一个多项式可能有多个根；同一个根也可能“重复出现”。下一节将系统地深化这层关系。

---

### 23.4.2 根与因式的深层关系

> 因式定理告诉我们“一个根 $\iff$ 一个一次因式”。本节分两个层次深化这层关系：首先在实数域 $\mathbb{R}$ 上讨论重根、根数上界及结构定理，然后提升至复数域 $\mathbb{C}$（回顾 [Ch17 数系(II)](/posts/ch17-数系(ii)/)）——借助代数基本定理，在 $\mathbb{C}$ 上多项式可**完全分解**为一次因式之积，从而彻底解决根的结构问题。最后回到 $\mathbb{R}$，推导实系数多项式在 $\mathbb{R}$ 上的不可约因式分类。

---

#### 23.4.2a 重根

- **定义 23.5 (重根与重数 / Multiple Root and Multiplicity)**
  设 $P(z)$ 为单变量多项式，$a$ 为 $P(z)$ 的一个根。若 $(z-a)^m \mid P(z)$ 但 $(z-a)^{m+1} \nmid P(z)$，则称 $a$ 为 $P(z)$ 的 **$m$ 重根**（Root of Multiplicity $m$）。$m$ 称为该根的**重数**。
  - $m = 1$：**单根**（Simple Root）。
  - $m = 2$：**二重根**（Double Root）。
  - $m \ge 2$：统称**重根**（Multiple Root）。

  **Remark:** 因式定理给出的是 $m = 1$ 的情况。重根概念是对因式定理的自然推广。此定义对 $\mathbb{R}$ 和 $\mathbb{C}$ 均适用（$a$ 可为实数或复数）。

---

#### 23.4.2b 因式的整除性质

> 以下定理描述了一次因式在整除乘积时的行为——它是多项式版本的“欧几里得引理”。

- **定理 23.6 (因式的整除传递性)**
  设 $P(z) = A(z) \cdot B(z)$ 为两个多项式的乘积。若 $(z-a) \mid P(z)$ 但 $(z-a) \nmid A(z)$，则 $(z-a) \mid B(z)$。

  **证明：** 由 $(z-a) \mid P(z)$ 及因式定理，$P(a) = 0$，即 $A(a) \cdot B(a) = 0$。在 $\mathbb{C}$（或 $\mathbb{R}$）中，乘积为零蕴含至少一个因子为零。由 $(z-a) \nmid A(z)$ 及因式定理，$A(a) \neq 0$。因此 $B(a) = 0$，再由因式定理得 $(z-a) \mid B(z)$。□

  **Remark (与数论的对照):** 此定理是数论中**欧几里得引理**（Euclid's Lemma，见 [Ch01 数系(I)](/posts/ch01-数系(i)/)）的多项式版本：

  | | 数论（$\mathbb{Z}$） | 多项式（$\mathbb{C}[z]$ 或 $\mathbb{R}[z]$） |
  |:---|:---|:---|
  | 不可约元素 | 质数 $p$ | 一次因式 $(z-a)$ |
  | 命题 | 若 $p \mid ab$ 且 $p \nmid a$，则 $p \mid b$ | 若 $(z-a) \mid A(z)B(z)$ 且 $(z-a) \nmid A(z)$，则 $(z-a) \mid B(z)$ |
  | 证明工具 | 贝祖等式（$\gcd(p,a)=1 \implies \exists x,y: px+ay=1$） | 因式定理（$P(a)=0 \iff (z-a) \mid P(z)$） |

  欧几里得引理是整数质因数分解唯一性的基石；定理 23.6 在多项式因式分解中扮演完全平行的角色——保证了一次因式的“质数性”：若一个一次因式整除乘积，则它必整除其中至少一个因子。

#### 23.4.2c 在实数域 $\mathbb{R}$ 上的结构

- **定理 23.7 ($n$ 次多项式的根数上界)**
  设 $P(x)$ 为非零单变量多项式，$\deg P = n \ge 0$。则 $P(x)$ 在 $\mathbb{R}$（或任意域）中至多有 $n$ 个不同的根。

  **证明：** 假设 $P(x)$ 有 $k$ 个不同的根 $a_1, a_2, \dots, a_k$。对每个 $a_i$，由因式定理知 $(x-a_i) \mid P(x)$。由于 $a_1, \dots, a_k$ 两两不同，诸多项式 $(x-a_1), (x-a_2), \dots, (x-a_k)$ 两两互质，故乘积 $\prod_{i=1}^k (x-a_i) \mid P(x)$。因此
  $$k = \deg \prod_{i=1}^k (x-a_i) \le \deg P = n$$
  即 $k \le n$。□

  **推论：** 若一个 $n$ 次多项式的根多于 $n$ 个，则它必为零多项式。

- **定理 23.8 (实数域上的根—因式结构)**
  设 $P(x)$ 为非零单变量多项式，$\deg P = n$，首项系数为 $a_n$。若 $P(x)$ 在 $\mathbb{R}$ 中的所有根为 $a_1, a_2, \dots, a_k$，且对应的重数分别为 $m_1, m_2, \dots, m_k$，则存在多项式 $Q(x)$ 满足：
  $$P(x) = a_n \cdot (x-a_1)^{m_1} (x-a_2)^{m_2} \cdots (x-a_k)^{m_k} \cdot Q(x)$$
  其中 $Q(x)$ 在 $\mathbb{R}$ 上无根（即 $Q(x)$ 在 $\mathbb{R}$ 上不可再分解出一次因式），且 $\deg Q = n - (m_1 + m_2 + \cdots + m_k)$。

  **证明思路：** 对 $P(x)$ 反复应用因式定理及除法算法：对每个根 $a_i$，连续除以 $(x-a_i)$ $m_i$ 次，每次降次 $1$，直至除尽。商式 $Q(x)$ 不再含任何 $(x-a)$ 形式的因式。□

  **Remark:** 定理 23.8 是“尽可能分解”的结果——在 $\mathbb{R}$ 上只能走到这一步。$Q(x)$ 在 $\mathbb{R}$ 上无根，但并不意味 $Q(x)$ 一定不可再分解（例如 $x^4+1 = (x^2+\sqrt{2}x+1)(x^2-\sqrt{2}x+1)$ 在 $\mathbb{R}$ 上无根却可分解为两个二次式）。要彻底解决分解的终点问题，必须提升至复数域。

#### 23.4.2d 提升至复数域 $\mathbb{C}$

> 回顾 [Ch17 数系(II)](/posts/ch17-数系(ii)/)，复数系 $\mathbb{C}$ 是 $\mathbb{R}$ 的扩张，使得每个非常数多项式都有根。这彻底改变了根与因式的结构。

- **定理 23.9 (代数基本定理 / Fundamental Theorem of Algebra)**
  设 $P(z)$ 为次数 $n \ge 1$ 的复系数多项式。则 $P(z)$ 在 $\mathbb{C}$ 中至少有一个根。

  **证明：** 超出 DSE 范围（需复分析或拓朴学工具），此处作为已知事实接受。□

- **推论 23.3 ($\mathbb{C}$ 上的完全因式分解)**
  设 $P(z)$ 为次数 $n \ge 1$ 的复系数多项式，首项系数为 $a_n$。则 $P(z)$ 可**完全分解**为 $n$ 个一次因式的乘积：
  $$P(z) = a_n (z - z_1)(z - z_2) \cdots (z - z_n)$$
  其中 $z_1, z_2, \dots, z_n \in \mathbb{C}$ 为 $P(z)$ 的所有根（计重数）。

  **证明思路：** 反复应用定理 23.9 及除法算法：由代数基本定理，$P(z)$ 在 $\mathbb{C}$ 中有根 $z_1$，因式定理给出 $P(z) = (z-z_1) P_1(z)$，$\deg P_1 = n-1$。对 $P_1(z)$ 继续应用定理 23.9，直至次数降为 $0$。每次降次 $1$，共 $n$ 次，故得到 $n$ 个一次因式。□

  将重根合并后，可写为：
  $$P(z) = a_n (z - z_1)^{m_1} (z - z_2)^{m_2} \cdots (z - z_k)^{m_k}, \qquad \sum_{i=1}^k m_i = n$$

  **对照定理 23.8：** 在实数域上 $Q(x)$ 未必为常数；在复数域上，推论 23.3 断言 $Q(z) = a_n$（常数）——一切非常数多项式在 $\mathbb{C}$ 上均可彻底分解为一次因式之积。

- **定理 23.10 (共轭根定理 / Conjugate Root Theorem)**
  设 $P(x)$ 为**实系数**多项式（即所有系数 $a_i \in \mathbb{R}$）。若复数 $z \in \mathbb{C} \setminus \mathbb{R}$ 是 $P(x)$ 的一个根，则其共轭 $\overline{z}$ 也是 $P(x)$ 的根，且两者的重数相同。

  **证明：** 设 $P(x) = \sum_{i=0}^n a_i x^i$，其中 $a_i \in \mathbb{R}$。则 $\overline{a_i} = a_i$。若 $P(z) = 0$，则
  $$P(\overline{z}) = \sum a_i \overline{z}^i = \sum \overline{a_i} \, \overline{z^i} = \overline{\sum a_i z^i} = \overline{P(z)} = \overline{0} = 0$$
  故 $\overline{z}$ 也是根。由对称性，重数相同（证明略，可由除法算法及唯一性推得）。□

  **Remark:** 共轭根定理是理解“实系数多项式在 $\mathbb{R}$ 上的因式结构”的关键。它告诉我们：实系数多项式的非实复根总是**成对**出现。

- **推论 23.4 (实系数多项式在 $\mathbb{R}$ 上的不可约因式)**
  设 $P(x)$ 为实系数多项式。则 $P(x)$ 在 $\mathbb{R}$ 上的不可约因式仅有两类：
  1. **一次因式** $(x-a)$，其中 $a \in \mathbb{R}$（对应实根）；
  2. **二次因式** $x^2 + px + q$，其中判别式 $\Delta = p^2 - 4q < 0$（对应一对共轭复根 $z, \overline{z}$，因 $(x-z)(x-\overline{z}) = x^2 - 2\operatorname{Re}(z)x + |z|^2$）。

  **证明思路：** 由推论 23.3，$P(x)$ 在 $\mathbb{C}$ 上可完全分解为一次因式。由共轭根定理，非实复根成对出现，每对 $z, \overline{z}$ 对应的乘积 $(x-z)(x-\overline{z})$ 为实系数二次式，判别式 $<0$。将所有一次实因式与二次共轭对因式相乘，即得 $P(x)$ 在 $\mathbb{R}$ 上的因式分解。□

  **Remark:** 这解释了定理 23.8 中“$Q(x)$ 无实根”的情形——$Q(x)$ 必可分解为若干个判别式 $<0$ 的二次因式的乘积。例如 $x^4+1$ 在 $\mathbb{C}$ 上有四个根 $\frac{\pm1\pm i}{\sqrt{2}}$，两两共轭成对，在 $\mathbb{R}$ 上分解为两个二次式。

---

#### 23.4.2e 小结：从 $\mathbb{R}$ 到 $\mathbb{C}$

| 域 | 根的存在性 | 因式分解结构 | 不可约因式 |
|:---|:---|:---|:---|
| $\mathbb{R}$ | $n$ 次多项式未必有实根（如 $x^2+1$） | $P(x) = a_n \prod (x-a_i)^{m_i} \cdot Q(x)$，$Q$ 无实根 | 一次式及 $\Delta<0$ 的二次式 |
| $\mathbb{C}$ | 每个非常数多项式必有根（定理 23.9） | $P(z) = a_n \prod (z-z_i)$，完全分解（推论 23.3） | 仅有一次式 |

三种表示方式对照（更新）：

| 表示方式 | 形式 | 适用的域 |
|:---|:---|:---|
| 标准展开式 | $P(x) = a_n x^n + \cdots + a_0$ | $\mathbb{R}$ 或 $\mathbb{C}$ |
| $\mathbb{R}$ 上的根—因式式 | $P(x) = a_n \prod (x-a_i)^{m_i} \cdot Q(x)$ | $\mathbb{R}$ |
| $\mathbb{C}$ 上的完全分解 | $P(z) = a_n \prod (z-z_i)$（计重数） | $\mathbb{C}$ |
| 任意因式分解 | $P(x) = f_1(x) f_2(x) \cdots$ | $\mathbb{R}$（含二次不可约因式）或 $\mathbb{C}$（仅一次因式） |

### 23.4.3 因式定理与除法算法的协同：分解多项式

> 因式定理负责“检测”——判断 $(z-a)$ 是否为因式；除法算法（定理 23.3）负责“提取”——将该因式从多项式中分离出来。两者协同，构成了分解单变量多项式的核心机制。

因式定理与除法算法的分工如下：

| 步骤 | 所用定理 | 作用 |
|:---|:---|:---|
| 1. 检测 | 因式定理（定理 23.5） | 计算 $P(a)$：若 $P(a)=0$，则 $(z-a)$ 为因式 |
| 2. 提取 | 除法算法（定理 23.3） | 计算 $P(z) \div (z-a)$，得商式 $Q(z)$，满足 $P(z) = (z-a) \cdot Q(z)$ |

这是一个“检测 → 提取”的迭代循环：每找到一个因式，立即用除法算法将其提取出来，降低多项式次数，然后在新的商式上继续检测。正因为除法算法保证了 $\deg Q = \deg P - 1$（定理 23.3），每次迭代都使问题规模严格缩小，循环必然在有限步内终止。

- **方法 23.1 (利用除法算法分解多项式)**
  1. **寻找候选根：** 对于整系数多项式 $P(z) = a_n z^n + \cdots + a_0$，有理根必为 $\pm \dfrac{p}{q}$ 的形式，其中 $p \mid a_0$，$q \mid a_n$。（有理根定理，可视为选学拓展。）实务上亦可从 $P(0), P(\pm 1), P(\pm 2), \dots$ 逐个测试。
  2. **检测（因式定理）：** 代入候选值 $a$，计算 $P(a)$。若 $P(a) \neq 0$，则 $a$ 不是根，返回步骤 1 继续搜寻。若 $P(a) = 0$，则 $(z-a)$ 为因式，进入步骤 3。
  3. **提取（除法算法）：** 以 $P(z)$ 为被除式，$(z-a)$ 为除式，执行长除法（或综合除法），由定理 23.3 得：
     $$P(z) = (z-a) \cdot Q(z) + 0$$
     其中商式 $Q(z)$ 满足 $\deg Q = \deg P - 1$。
  4. **检查重数：** 对商式 $Q(z)$ 再次检测 $Q(a)$：若 $Q(a)=0$，则 $a$ 为重根——以 $Q(z)$ 为新的被除式，返回步骤 3，继续除以 $(z-a)$；否则 $a$ 的处理完成。
  5. **迭代：** 以最终得到的降次多项式为新的 $P(z)$，返回步骤 1 继续搜寻其他根。

     循环终止条件取决于所处的域：
     - 在 $\mathbb{R}$ 上：无法再找到有理根，或剩余多项式为二次式可直接用 Ch04 的十字相乘法处理；若剩余为高次且无实根，则由推论 23.4 可分解为判别式 $<0$ 的二次因式之积。
     - 在 $\mathbb{C}$ 上：由推论 23.3，循环会一直进行直至商式为常数，即多项式完全分解为一次因式之积。

  **Remark:** 此方法的效率来自除法算法的降次性质——每次成功提取一个一次因式，多项式次数严格减少 $1$，确保了算法的有限终止性。这与整数的质因数分解完全平行。在 $\mathbb{R}$ 上，分解终止于无实根的 $Q(z)$（可进一步分解为二次不可约因式）；在 $\mathbb{C}$ 上，由代数基本定理（定理 23.9）保证分解可进行至所有因式均为一次为止。

## 23.5 最大公因式与最小公倍式

> 将整数的 HCF 与 LCM 概念迁移至单变量多项式。核心方法仍依赖于**完全因式分解**——在 $\mathbb{R}$ 上分解至一次及不可约二次因式，在 $\mathbb{C}$ 上分解至一次因式（推论 23.4 / 推论 23.3）。

### 23.5.1 最大公因式

回顾定义 4.12（公因式与最大公因式）适用于一般多项式。在单变量多项式语境下，我们可给出更具操作性的定义：

- **定义 23.6 (单变量多项式的最大公因式 / HCF)**
  设 $P(x), Q(x)$ 为两个非零单变量多项式，其完全因式分解为（将所有因式归一化为首一形式）：
  $$P(x) = k \cdot \prod_i f_i(x)^{p_i}, \qquad Q(x) = \ell \cdot \prod_i f_i(x)^{q_i}$$
  其中 $f_i(x)$ 为不可再分解的首一多项式，$p_i, q_i \ge 0$（若 $P$ 不含 $f_i$，则 $p_i = 0$）。则 $P$ 与 $Q$ 的**最大公因式**为：
  $$\text{HCF}(P, Q) = \prod_i f_i(x)^{\min(p_i, q_i)}$$
  HCF 的常数部分取 $k, \ell$ 的最大公因数。

- **方法 (求最大公因式):**
  1. 将每个多项式**完全因式分解**（运用 Ch04 的因式分解方法、本章的因式定理及 Ch06 的恒等式）。
  2. 将所有因式归一化为首一形式。
  3. 找出所有**公共的因式**。
  4. 取每个公共因式的**最低次幂**相乘。
  5. 系数部分取各多项式首项系数的最大公因数。

### 23.5.2 最小公倍式

- **定义 23.7 (单变量多项式的最小公倍式 / LCM)**
  设 $P(x), Q(x)$ 为两个非零单变量多项式，其完全因式分解同上。则 $P$ 与 $Q$ 的**最小公倍式**为：
  $$\text{LCM}(P, Q) = \prod_i f_i(x)^{\max(p_i, q_i)}$$
  系数部分取各多项式首项系数的最小公倍数。

- **方法 (求最小公倍式):**
  1. 将每个多项式**完全因式分解**。
  2. 将所有因式归一化为首一形式。
  3. 找出所有出现过的**不同因式**。
  4. 取每个不同因式的**最高次幂**相乘。
  5. 系数部分取各多项式首项系数的最小公倍数。

### 23.5.3 HCF 与 LCM 的关系

- **定理 23.11**
  对任意两个非零单变量多项式 $P(x), Q(x)$，
  $$\text{HCF}(P, Q) \cdot \text{LCM}(P, Q) = c \cdot P(x) \cdot Q(x)$$
  其中 $c$ 为常数因子（来自系数归一化）。若忽略常数倍数的差异，可简记为：
  $$\text{HCF}(P, Q) \cdot \text{LCM}(P, Q) \sim P(x) \cdot Q(x)$$

  **证明思路：** 对每个因式 $f_i$，$\min(p_i, q_i) + \max(p_i, q_i) = p_i + q_i$。将各因式相乘即得结论。□

### 23.5.4 与整数的类比

多项式 HCF/LCM 与整数的 HCF/LCM 在结构上完全平行：

| 概念 | 整数 | 单变量多项式 |
|:---|:---|:---|
| 基本构成单元 | 质数 | 不可约首一多项式（一次因式等） |
| 规范化约定 | 取正质数 | 归一化为首一形式（定义 23.2） |
| HCF | 公共质因数取最低次幂 | 公共因式取最低次幂 |
| LCM | 所有质因数取最高次幂 | 所有因式取最高次幂 |
| HCF × LCM | $= a \times b$ | $\sim P(x) \cdot Q(x)$ |

## 第二十三章总结

1. **单变量多项式**：仅含一个变量的多项式，记为 $P(z) = a_n z^n + \cdots + a_0$。相等条件为次数相同且各项系数相等（定理 23.2）。**首一多项式**（定义 23.2）为首项系数 $=1$ 者，因式分解中常将因式归一化为首一形式以便比较。

2. **除法算法与整除**：对任意 $P(z), D(z)$（$D \neq 0$），存在唯一的 $Q(z), R(z)$ 使得 $P = DQ + R$，$\deg R < \deg D$（定理 23.3）。长除法是执行此算法的标准方法。**整除**（定义 23.3）：$D(z) \mid P(z) \iff$ 存在 $Q(z)$ 使 $P = DQ$ $\iff$ 余式 $R = 0$。

3. **余式定理**：$P(z)$ 除以 $(z-a)$ 的余式 $= P(a)$（定理 23.4）。推论：除以 $(az-b)$ 的余式 $= P(b/a)$。

4. **因式定理**：$(z-a) \mid P(z) \iff P(a) = 0$（定理 23.5）。建立了因式与根之间的基本双向联系。与除法算法（定理 23.3）形成“检测 $\to$ 提取”的协同机制——因式定理判断因式，除法算法将其分离，每次降次 $1$，迭代至分解完成（方法 23.1）。

5. **根与因式的深层关系**（分 $\mathbb{R}$ 和 $\mathbb{C}$ 两个层次）：
   - **重根**（定义 23.5）：$(z-a)^m \mid P$ 但 $(z-a)^{m+1} \nmid P$，$m$ 为重数。适用于 $\mathbb{R}$ 和 $\mathbb{C}$。
   - **因式的整除性质**（定理 23.6）：若 $(z-a) \mid A(z)B(z)$ 且 $(z-a) \nmid A(z)$，则 $(z-a) \mid B(z)$。对应数论中欧几里得引理（Ch01）：一次因式在多项式中扮演质数的角色。
   - **在 $\mathbb{R}$ 上**：
     - 根数上界（定理 23.7）：$n$ 次多项式至多有 $n$ 个不同的根。
     - 结构定理（定理 23.8）：$P(x) = a_n \prod (x-a_i)^{m_i} \cdot Q(x)$，$Q(x)$ 无实根。
   - **提升至 $\mathbb{C}$**（回顾 Ch17）：
     - 代数基本定理（定理 23.9）：非常数复系数多项式必有复根。
     - 完全分解（推论 23.3）：$P(z) = a_n \prod (z-z_i)$——在 $\mathbb{C}$ 上所有一次因式均可分解到底。
     - 共轭根定理（定理 23.10）：实系数多项式的非实复根成对出现。
     - 回到 $\mathbb{R}$（推论 23.4）：实系数多项式的不可约因式仅有一次式及 $\Delta<0$ 的二次式。

6. **HCF 与 LCM**：将多项式完全因式分解并归一化为首一形式后，公共因式取最低次幂得 HCF（定义 23.6），所有因式取最高次幂得 LCM（定义 23.7）。满足 $\text{HCF} \cdot \text{LCM} \sim P \cdot Q$（定理 23.11）。

**模块目标：** Ch18 §4 建立了换元法的一般框架——通过代换 $u = f(x)$ 将特定形式的方程化为一元二次方程。本章将换元法推广至超越方程（指数、对数、三角），并结合 Ch25 与 Ch26 中建立的函数理论，系统处理指数方程、对数方程及三角方程的解法。核心策略有三：利用函数单射性化简（同底法、对数合并法）、直接使用对数定义（取对数法、转为指数式）、以及换元法化为二次方程。

---

## 27.1 指数方程

> **学习目标：** 利用指数函数的单射性（Ch25 定理 25.7）及对数的定义（Ch25 定义 25.5），建立指数方程的三类标准解法。

### 27.1.1 同底法

指数函数 $E_a: x \mapsto a^x$（$a > 0$，$a \neq 1$）为严格单调函数，故为单射——不同的输入产生不同的输出。此性质直接导出同底法。

*   **定理 27.1（同底法 / Same Base Method）：**
    设 $a > 0$，$a \neq 1$，$f(x)$ 与 $g(x)$ 为定义于 $\mathbb{R}$ 上的实值表达式。则：
    $$a^{f(x)} = a^{g(x)} \iff f(x) = g(x)$$

    **证明：** 由 Ch25 定理 25.7，$E_a: x \mapsto a^x$ 为单射。故 $E_a(f(x)) = E_a(g(x)) \iff f(x) = g(x)$，此即所需。

*   **方法 27.1（同底法的一般步骤）：**
    1.  **化同底：** 利用指数运算律（Ch25 定理 25.5）将方程两边写成同一底数的幂。
    2.  **比较指数：** 由定理 27.1，令两边的指数相等，得代数方程。
    3.  **求解：** 解出代数方程即得原指数方程的解。

    **常见转化：**
    *   $4^x = 8 \implies (2^2)^x = 2^3 \implies 2^{2x} = 2^3 \implies 2x = 3$
    *   $\left(\dfrac{1}{3}\right)^x = 9 \implies 3^{-x} = 3^2 \implies -x = 2$
    *   $\sqrt{a^x} = a^{x/2}$

> **注：** 底数的选择不必唯一。上述 $4^x = 8$ 亦可以 $4$ 为底：$4^x = 4^{3/2} \implies x = 3/2$。结果一致。

---

### 27.1.2 取对数法

当方程两边的底数无法化为相同时，同底法失效。此时可利用对数的定义直接求解。

*   **定理 27.2（取对数法——定义形式）：**
    设 $a > 0$，$a \neq 1$，$b > 0$。则：
    $$a^x = b \iff x = \log_a b$$

    **证明：** 即为 Ch25 定义 25.5（$\log_a N = x \iff a^x = N$）的直接改写。

    推广至含函数表达式的情形：
    $$a^{f(x)} = b \iff f(x) = \log_a b \quad (\text{要求 } b > 0)$$

*   **方法 27.2（取对数法——不同底数情形）：**
    设方程形如 $a^{f(x)} = b^{g(x)}$（$a, b > 0$，$a, b \neq 1$，且 $a \neq b$，无法化同底），则两边取常用对数 $\lg$（或以任意底数 $c$ 取对数）：
    $$\lg(a^{f(x)}) = \lg(b^{g(x)}) \implies f(x) \lg a = g(x) \lg b$$
    此为关于 $x$ 的代数方程。

    **原理：** 对数函数 $L_c: x \mapsto \log_c x$ 为单射（Ch25 定理 25.11），故两边取对数为等价变换（因原方程两边均为正数——$a^{f(x)} > 0$，$b^{g(x)} > 0$）。

*   **推论 27.1（取对数法——取以 $a$ 为底的情形）：**
    对 $a^{f(x)} = b$（$b > 0$），直接取以 $a$ 为底的对数：
    $$\log_a(a^{f(x)}) = \log_a b \implies f(x) = \log_a b$$
    这实为定理 27.2 的直接应用。若 $b \leq 0$，则方程无实数解（因 $a^{f(x)} > 0$ 恒成立，Ch25 定理 25.6）。

---

### 27.1.3 换元法——可化为二次方程

当方程中指数项仅以 $a^x$ 的幂次出现时，可透过换元将方程化为关于 $u = a^x$ 的二次方程。此为 Ch18 §4 中换元法框架在指数方程中的应用。

*   **方法 27.3（指数换元法 / Substitution for Exponential Equations）：**
    设方程形如：
    $$A \cdot k^{2x} + B \cdot k^x + C = 0 \quad (A \neq 0,\; k > 0,\; k \neq 1)$$
    1.  **换元：** 令 $u = k^x$。注意到 $k^{2x} = (k^x)^2 = u^2$，代入得：
        $$A u^2 + B u + C = 0$$
    2.  **解二次方程：** 用 Ch18 的方法解出 $u$ 的值。
    3.  **回代：** 对每个 $u$ 值，解 $k^x = u$。由定理 27.2，$x = \log_k u$（要求 $u > 0$）。
    4.  **检验约束：** $u = k^x > 0$ 对所有实数 $x$ 恒成立（因 $k > 0$，Ch25 定理 25.6）。若二次方程有根 $u \leq 0$，则该根对应的 $k^x = u$ 无实数解，应予舍弃。

*   **定理 27.3（指数换元法的解的存在性）：**
    方程 $A \cdot k^{2x} + B \cdot k^x + C = 0$（$A \neq 0$，$k > 0$，$k \neq 1$）有实数解，当且仅当二次方程 $A u^2 + B u + C = 0$ 存在正实根。

    **证明：** 换元 $u = k^x$ 建立了 $x \in \mathbb{R}$ 与 $u \in (0, \infty)$ 之间的一一对应（因 $E_k: \mathbb{R} \to (0, \infty)$ 为双射——严格单调且满射至正实数集）。故原方程在 $\mathbb{R}$ 中的解与二次方程在 $(0, \infty)$ 中的正根一一对应。

*   **推论 27.2（多项式型指数方程的推广）：**
    更一般地，若方程形如 $\sum_{i=0}^n A_i (k^x)^i = 0$（即关于 $k^x$ 的多项式方程），均可以 $u = k^x$ 换元，化为关于 $u$ 的多项式方程求解。DSE 范围内常见者为 $n = 2$ 的二次情形。

## 27.2 对数方程

> **学习目标：** 利用对数的定义（Ch25 定义 25.5）、运算律（Ch25 定理 25.9）及对数函数的单射性（Ch25 定理 25.11），建立对数方程的标准解法。**重点**：对数方程必须验根——确保解满足所有真数大于零、底数大于零且不等于一的约束。

### 27.2.1 转化为指数式（定义法）

对数的定义直接提供了最简单的解法：将对数式还原为等价的指数式。

*   **定理 27.4（定义法 / Definition Approach）：**
    设 $a > 0$，$a \neq 1$。则：
    $$\log_a f(x) = b \iff f(x) = a^b \quad (\text{其中 } f(x) > 0 \text{ 自动由 } a^b > 0 \text{ 满足})$$

    **证明：** 由 Ch25 定义 25.5，$\log_a N = b \iff N = a^b$。取 $N = f(x)$ 即得。注意 $a^b > 0$（$a > 0$），故 $f(x) = a^b > 0$ 自动成立，无需额外验证。

---

### 27.2.2 对数合并法

当方程两边各为对数表达式时，可利用对数运算律将方程合并为 $\log_a F(x) = \log_a G(x)$ 的形式，再利用对数函数的单射性化简。

*   **定理 27.5（对数合并法 / Condensing Logarithms）：**
    设 $a > 0$，$a \neq 1$。若 $F(x) > 0$ 且 $G(x) > 0$，则：
    $$\log_a F(x) = \log_a G(x) \iff F(x) = G(x)$$

    **证明：** 由 Ch25 定理 25.11，$L_a: x \mapsto \log_a x$ 为单射。故 $L_a(F(x)) = L_a(G(x)) \iff F(x) = G(x)$。前提条件 $F(x) > 0$，$G(x) > 0$ 确保 $\log_a F(x)$ 与 $\log_a G(x)$ 有定义。

*   **方法 27.4（对数合并法的一般步骤）：**
    给定含多个对数项的方程：
    1.  **合并对数：** 利用对数运算律（Ch25 定理 25.9）将各项合并：
        *   $\log_a M + \log_a N = \log_a(MN)$
        *   $\log_a M - \log_a N = \log_a(M/N)$
        *   $k \log_a M = \log_a(M^k)$
    2.  **化为 $\log_a F(x) = \log_a G(x)$：** 合并后方程两边各为一个对数式（同底）。
    3.  **去对数：** 由定理 27.5，得 $F(x) = G(x)$。
    4.  **求解与验根：** 解出代数方程的根，逐一验证是否满足原方程所有真数条件。

    **关键注意：** 步骤 3 中“去对数”的前提是 $F(x) > 0$ 且 $G(x) > 0$。这些约束来自原方程中各对数项的定义域要求，必须在求解后检验（见 §27.2.4）。

---

### 27.2.3 换元法——可化为二次方程

当对数方程中仅以 $\log_a x$（或 $\log_a f(x)$）的幂次出现时，可透过换元化为二次方程。

*   **方法 27.5（对数换元法 / Substitution for Logarithmic Equations）：**
    设方程形如：
    $$A (\log_a x)^2 + B (\log_a x) + C = 0 \quad (A \neq 0)$$
    1.  **换元：** 令 $u = \log_a x$，代入得 $A u^2 + B u + C = 0$。
    2.  **解二次方程：** 用 Ch18 的方法解出 $u$ 的值。
    3.  **回代：** 对每个 $u$ 值，由定义 $u = \log_a x \iff x = a^u$。
    4.  **验根：** 由于 $x = a^u > 0$ 自动成立（$a > 0$），此处的回代不产生增根。但需注意原方程可能含有定义域更复杂的对数项（如 $\log_a (x-1)$ 等）。

*   **推论 27.3（推广至一般真数表达式）：**
    若方程形如 $A (\log_a f(x))^2 + B (\log_a f(x)) + C = 0$，令 $u = \log_a f(x)$，解 $u$ 后回代解 $f(x) = a^u$。此情形下 $f(x) > 0$ 的约束必须在回代后检验。

---

### 27.2.4 验根原则

对数方程中，以下操作可能引入增根（extraneous root），必须逐一验证：

*   **定理 27.6（对数方程的增根来源）：**
    在解对数方程过程中，以下每一步均必须确保对应的真数条件成立，否则所得之根可能为增根：
    1.  **合并对数：** $\log_a M + \log_a N = \log_a(MN)$ 成立的前提是 $M > 0$ 且 $N > 0$。若原方程写为 $\log_a (MN)$ 的形式，则 $MN > 0$ 的条件较原式 $\log_a M + \log_a N$ 中的 $M > 0$、$N > 0$ 更弱——可能引入使 $M < 0$ 且 $N < 0$ 但 $MN > 0$ 的增根。
    2.  **去对数：** 由 $\log_a F(x) = \log_a G(x)$ 化为 $F(x) = G(x)$，丢弃了 $F(x) > 0$ 与 $G(x) > 0$ 的约束。
    3.  **定义转换：** $\log_a f(x) = b$ 化为 $f(x) = a^b$ 为等价变换——不产生增根（因 $a^b > 0$ 自动保证 $f(x) > 0$）。

    **证明（1. 的说明）：**
    $\log_a M + \log_a N = \log_a(MN)$ 的等式左边要求 $M > 0$ 且 $N > 0$；右边仅要求 $MN > 0$。当 $M < 0$ 且 $N < 0$ 时，$MN > 0$ 满足右边的定义域，但左边无定义。因此从 $\log_a (MN) = \cdots$ 的形式无法逆推出 $\log_a M + \log_a N$ 的原始形式——后者对 $M, N$ 各自有额外的正性要求。

*   **方法 27.6（对数方程的验根步骤）：**
    对解出的每一个 $x$ 值，代入**原方程**（非化简后的代数方程），逐一检查：
    1.  底数 $a$ 是否满足 $a > 0$ 且 $a \neq 1$（若底数为常数，此步自动跳过）。
    2.  每个对数项的真数（即 $\log_a (\text{真数})$ 中的“真数”）是否**严格大于零**。
    3.  若某项真数 $\leq 0$，则该 $x$ 为增根，予以舍弃。

## 27.3 三角方程

> **学习目标：** 在 Ch26 建立的三角方程通解理论（定理 26.7）基础上，拓展至可透过换元法化为二次方程的三角方程，以及需利用恒等式转换为单一三角函数的混合方程。

### 27.3.1 基本三角方程回顾

以下为 Ch26 中三角方程通解的核心结论，本章直接引用。

*   **定理 26.7（三角方程的通解公式，见 Ch26）：**
    设 $\alpha \in \mathbb{R}$：
    1.  若 $\sin x = \sin \alpha$，则 $x = \alpha + 2k\pi$ 或 $x = \pi - \alpha + 2k\pi$（$k \in \mathbb{Z}$）。
    2.  若 $\cos x = \cos \alpha$，则 $x = \pm \alpha + 2k\pi$（$k \in \mathbb{Z}$）。
    3.  若 $\tan x = \tan \alpha$，则 $x = \alpha + k\pi$（$k \in \mathbb{Z}$）。

*   **方法 26.1（参考角法，见 Ch26）：** 将方程化为 $\sin x = k$（或 $\cos x = k$、$\tan x = k$），求参考角 $\alpha$，由 $k$ 的符号及 ASTC（定理 26.4）确定象限，写出 $[0, 2\pi]$ 内的解，再用通解公式推至全 $\mathbb{R}$。

*   **方法 26.2（恒等式转换法，见 Ch26）：** 利用 $\sin^2 x + \cos^2 x = 1$、$\tan x = \sin x / \cos x$ 等恒等式，将含多种三角函数的方程统一为同一种三角函数的代数方程后求解。

---

### 27.3.2 可化为二次的三角方程

三角方程中，若三角函数值仅以 $\sin x$（或 $\cos x$、$\tan x$）的幂次形式出现，则可透过换元化为二次方程。换元的关键差异在于：换元变量 $u = \sin x$ 的取值范围受限于 $[-1, 1]$。

*   **方法 27.7（三角换元法 / Substitution for Trigonometric Equations）：**
    设方程形如：
    $$A \sin^2 x + B \sin x + C = 0 \quad (A \neq 0)$$
    1.  **换元：** 令 $u = \sin x$，代入得 $A u^2 + B u + C = 0$。
    2.  **解二次方程并筛选：** 解出 $u$ 的值，**仅保留满足 $|u| \leq 1$ 的根**（因 $\sin x$ 的值域为 $[-1, 1]$，定理 26.3）。
    3.  **回代解三角方程：** 对每个保留的 $u$ 值，解 $\sin x = u$。利用方法 26.1 先求 $[0, 2\pi]$ 内的解，再由定理 26.7 写出通解。
    4.  **指定区间截取：** 若题目指定 $x$ 的范围（常见 $[0, 2\pi]$ 或 $[0^\circ, 360^\circ]$），从通解中截取满足范围的解。

    **同理：** $\cos x$ 换元时 $|u| \leq 1$；$\tan x$ 换元时 $u \in \mathbb{R}$（无额外范围限制）。

*   **定理 27.7（三角换元法的约束条件）：**
    设 $A \neq 0$。方程 $A \sin^2 x + B \sin x + C = 0$（或对 $\cos x, \tan x$ 的类似形式）在指定区间 $I \subseteq \mathbb{R}$ 内的解，与二次方程 $A u^2 + B u + C = 0$ 中满足以下条件的根 $u$ 有关：
    1.  若换元为 $u = \sin x$ 或 $u = \cos x$，则 $u$ 必须满足 $|u| \leq 1$；
    2.  若换元为 $u = \tan x$，则 $u \in \mathbb{R}$（无额外限制）。

    对于每个符合条件的 $u$，解 $\sin x = u$（或 $\cos x = u$、$\tan x = u$）后截取 $I$ 中的解。

    **证明：** 第一条来自 $\sin$ 与 $\cos$ 的值域 $[-1, 1]$（定理 26.3）；第二条来自 $\tan$ 的值域 $\mathbb{R}$（定义 26.5）。

---

### 27.3.3 利用恒等式统一为同一种三角函数

当方程同时含有 $\sin x$ 和 $\cos x$（或 $\tan x$），且可透过恒等式 $\sin^2 x + \cos^2 x = 1$ 或 $\tan x = \sin x / \cos x$ 转换后统一为一种三角函数，则可将方程化为 §27.3.2 中可换元的形式。

*   **方法 27.8（恒等式统一法 / Unification via Identity）：**
    设方程含 $\sin x$ 与 $\cos x$（或 $\tan x$）的混合项：
    1.  **选择目标函数：** 观察方程结构，选定一种三角函数（通常为 $\sin x$ 或 $\cos x$），目标是将所有项表示为该函数的幂次。
    2.  **恒等式转换：**
        *   以 $\sin$ 为目标：$\cos^2 x = 1 - \sin^2 x$
        *   以 $\cos$ 为目标：$\sin^2 x = 1 - \cos^2 x$
        *   含 $\tan$ 时：$\tan x = \sin x / \cos x$，若方程中同时有 $\sin x$ 和 $\cos x$，可先化为 $\sin x$ 与 $\cos x$ 的方程，再选一者为目标。
    3.  **换元化为二次：** 统一后方程形如 $A (\sin x)^2 + B \sin x + C = 0$，按方法 27.7 处理。
    4.  **验根：** 转换过程中若涉及除法（如去分母 $x \neq 0$），须检验解是否满足原方程的定义域。若原方程含 $\tan x$，需排除使 $\cos x = 0$ 的 $x$ 值。

*   **定理 27.8（恒等式转换的等价性）：**
    利用 $\sin^2 x + \cos^2 x = 1$ 进行代换（如 $\cos^2 x = 1 - \sin^2 x$）为**等价变换**——不引入增根，不遗漏解。
    利用 $\tan x = \sin x / \cos x$ 进行代换时，需注意原方程须排除 $\cos x = 0$ 的点。若化简过程中约去了含 $\cos x$ 的因式，可能遗漏 $\cos x = 0$ 的解，须单独检验。

    **证明：** $\sin^2 x + \cos^2 x = 1$ 对所有 $x \in \mathbb{R}$ 成立（定理 26.3），故以 $1 - \sin^2 x$ 取代 $\cos^2 x$（或反之）在实数集上为恒等变形——不改变方程的解集。$\tan x = \sin x / \cos x$ 仅在 $\cos x \neq 0$ 处成立，以该式进行代换引入的“等价”仅在 $\cos x \neq 0$ 时有效。

## Ch27 总结

1.  **指数方程（§27.1）：** 三种策略——同底法（利用 $E_a$ 的单射性，定理 27.1）；取对数法（利用对数定义 $\log_a b = x \iff a^x = b$，定理 27.2）；换元法（令 $u = a^x$，$u > 0$，化为二次方程后回代，定理 27.3）。同底法与取对数法均为等价变换，不产生增根。

2.  **对数方程（§27.2）：** 三种策略——定义法（$\log_a f(x) = b \iff f(x) = a^b$，定理 27.4，等价变换）；对数合并法（利用运算律合并后以单射性去对数，定理 27.5）；换元法（令 $u = \log_a x$ 或 $u = \log_a f(x)$，方法 27.5）。**对数方程的核心挑战是增根**（定理 27.6）：合并对数时放松了真数条件，去对数时丢弃了正性约束——所有解必须代回原方程检验（方法 27.6）。

3.  **三角方程（§27.3）：** 在 Ch26 通解公式（定理 26.7）的基础上拓展——换元法（令 $u = \sin x$ 等，注意 $|u| \leq 1$ 的范围约束，定理 27.7）；恒等式统一法（利用 $\sin^2 x + \cos^2 x = 1$ 将混合三角方程化为单一三角函数后换元，方法 27.8）。$\sin^2 x + \cos^2 x = 1$ 为等价变换；含 $\tan x$ 的代换须单独检验 $\cos x = 0$ 的点（定理 27.8）。

> **衔接：** 本章所建立的换元法思维——将看似不可解的方程透过代换化为熟悉的二次方程——是 DSE 代数解题的核心策略。关于分式方程及含根式方程化为二次方程的处理，详见 Ch18 §4。

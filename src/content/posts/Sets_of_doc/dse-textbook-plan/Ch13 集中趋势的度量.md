## 13.1 不分组数据中的统计 (Statistics for Ungrouped Data)

### 13.1.1 算术平均数、中位数、众数 (Arithmetic Mean, Median, Mode)

- **定义 13.1 (算术平均数 / Arithmetic Mean)**
  设一组数据为 $x_1, x_2, \dots, x_n$（$n \ge 1$），则其**算术平均数**（简称**平均数**）$\bar{x}$ 定义为：
  $$\bar{x} = \frac{x_1 + x_2 + \cdots + x_n}{n} = \frac{1}{n}\sum_{i=1}^{n} x_i$$

- **定义 13.2 (中位数 / Median)**
  将一组数据按大小顺序排列后，位于正中间位置的数值。
  - 设排序后的数据为 $x_{(1)} \le x_{(2)} \le \cdots \le x_{(n)}$。
  - 若 $n$ 为**奇数**，中位数为第 $\frac{n+1}{2}$ 个数据：$\text{Median} = x_{(\frac{n+1}{2})}$。
  - 若 $n$ 为**偶数**，中位数为第 $\frac{n}{2}$ 个与第 $\frac{n}{2}+1$ 个数据的算术平均数：$\text{Median} = \dfrac{x_{(\frac{n}{2})} + x_{(\frac{n}{2}+1)}}{2}$。

- **定义 13.3 (众数 / Mode)**
  数据中出现频数最高的数值。
  - 一组数据可以有一个众数、多个众数，或没有众数（当所有数值的频数均相同时）。
  - 众数是唯一可用于非数值型数据（如类别数据）的集中趋势度量。

- **性质 13.1 (平均数的离差之和为零)**
  设 $\bar{x}$ 为数据 $x_1, x_2, \dots, x_n$ 的算术平均数，则各数据与平均数的离差之和为零：
  $$\sum_{i=1}^{n} (x_i - \bar{x}) = 0$$
  **证明：**
  $$\sum_{i=1}^{n} (x_i - \bar{x}) = \sum_{i=1}^{n} x_i - \sum_{i=1}^{n} \bar{x} = n\bar{x} - n\bar{x} = 0$$

- **性质 13.2 (极端值的影响)**
  算术平均数受数据中的极端值（outliers）影响较大，因为每个数据均参与总和的计算。
  中位数仅依赖排序后的位置，不受极端值影响，具有**稳健性（Robustness）**。
  众数仅依赖频数，亦不受极端值影响。

### 13.1.2 加权平均数 (Weighted Mean)

- **定义 13.4 (加权平均数 / Weighted Mean)**
  设数据为 $x_1, x_2, \dots, x_k$，对应的非负权重为 $w_1, w_2, \dots, w_k$（其中至少有一个 $w_i > 0$），则**加权平均数** $\bar{x}_w$ 定义为：
  $$\bar{x}_w = \frac{w_1 x_1 + w_2 x_2 + \cdots + w_k x_k}{w_1 + w_2 + \cdots + w_k} = \frac{\displaystyle\sum_{i=1}^{k} w_i x_i}{\displaystyle\sum_{i=1}^{k} w_i}$$

  **Remark:** 
  - 当所有权重相等（即 $w_1 = w_2 = \cdots = w_k$）时，加权平均数退化为普通的算术平均数。
  - 权重反映各数据在整体中的重要性或占比。常见应用包括：按学分加权的平均绩点、按购买数量加权的平均单价。

## 13.2 分组数据中的统计 (Statistics for Grouped Data)

> 当数据以频数分布表的形式呈现时，无法获取每个数据的准确值，因此所有统计量均为**估计量**。

### 13.2.1 众数组 (Modal Class)

- **定义 13.5 (众数组 / Modal Class)**
  在分组数据的频数分布表中，频数最高的组别（class interval）称为**众数组**。
  - 众数组反映数据集中出现的大致范围，而非一个精确的数值。
  - 可在直方图中直接识别：最高频数的条形所对应的组别即为众数组。

### 13.2.2 分组数据的平均数（估算） (Estimated Mean for Grouped Data)

- **定义 13.6 (组中点 / Class Mark)**
  对于组别 $[L, U)$（或 $[L, U]$），其**组中点** $m$ 定义为：
  $$m = \frac{L + U}{2}$$
  组中点作为该组中所有数据的代表值。

- **定义 13.7 (分组数据的估算平均数)**
  设数据分为 $k$ 组，第 $i$ 组的组中点为 $m_i$，频数为 $f_i$（$i = 1, 2, \dots, k$），总频数 $N = \sum f_i$。则**估算平均数**为：
  $$\bar{x}_{\text{est}} = \frac{f_1 m_1 + f_2 m_2 + \cdots + f_k m_k}{f_1 + f_2 + \cdots + f_k} = \frac{\displaystyle\sum_{i=1}^{k} f_i m_i}{N}$$

  **Remark:** 此为加权平均数的特殊情形，其中每组的组中点为 $x$ 值，频数为 $w$ 权重。

### 13.2.3 分组数据的中位数（估算） (Estimated Median for Grouped Data)

- **定义 13.8 (累积频数 / Cumulative Frequency)**
  将频数分布表中各组的频数由第一组开始逐组累加，所得数列称为**累积频数**。
  - 累积频数用于定位中位数所在的组别。

- **定义 13.9 (分组数据的估算中位数)**
  设总数据个数为 $N$，中位数的位置为第 $\frac{N}{2}$ 个数据。
  在累积频数分布中定位中位数所在的组别（即累积频数首次达到或超过 $\frac{N}{2}$ 的组）。
  假设该组数据在组距内均匀分布，则**估算中位数**由线性插值给出：
  $$\text{Median}_{\text{est}} = L + \left(\frac{\frac{N}{2} - F}{f}\right) \times w$$

  其中：
  - $L$ = 中位数所在组的下限
  - $F$ = 中位数所在组之前一组的累积频数
  - $f$ = 中位数所在组的频数
  - $w$ = 组距（class width）

- **方法 13.1 (以频数直接定位中位数组 / Locating Median Class by Frequency Summation)**

  可跳过“累积频数”的计算，直接用各组频数定位中位数所在组：

  1. 计算总频数 $N = f_1 + f_2 + \cdots + f_k$。
  2. 计算中位数位置 $\dfrac{N}{2}$。
  3. 自第一组起，将频数逐一累加：$f_1,\ f_1+f_2,\ f_1+f_2+f_3,\ \dots$，**首次达到或超过 $\dfrac{N}{2}$ 的组**即为中位数所在组。
  4. 记中位数所在组的下限为 $L$，该组频数为 $f$，在该组之前的所有组的频数之和为 $f_{\text{prev}}$（即第 3 步中使累加值首次 $\ge N/2$ 时，**前一组**的累积频数），组距为 $w$。
  5. 代入插值公式：
  $$\text{Median}_{\text{est}} = L + \frac{\frac{N}{2} - f_{\text{prev}}}{f} \times w$$

- **例：**

  某次考试分数的频数分布如下，求估算中位数：

  | 分数区间 | 频数 $f$ |
  |:---:|:---:|
  | 0–10 | 5 |
  | 10–20 | 12 |
  | 20–30 | 20 |
  | 30–40 | 10 |
  | 40–50 | 3 |

  - $N = 5+12+20+10+3 = 50$；中位数位置 = $25$。
  - 累加频数：$5 < 25$；$5+12 = 17 < 25$；$17+20 = 37 \ge 25$ → 中位数在 20–30 组。
  - $L = 20$，$f_{\text{prev}} = 17$，$f = 20$，$w = 10$。
  $$\text{Median}_{\text{est}} = 20 + \frac{25-17}{20} \times 10 = 20 + 4 = 24$$

## 第十三章总结

1. **算术平均数**：$\bar{x} = \frac{1}{n}\sum x_i$。各数据的离差之和为零（性质 13.1）。对极端值敏感。
2. **中位数**：排序后居中的值。奇数 $n$ 取第 $\frac{n+1}{2}$ 个，偶数 $n$ 取中间两个的平均。具稳健性，不受极端值影响。
3. **众数**：频数最高的值。可有多个或没有。唯一可用于非数值型数据的集中趋势度量。
4. **加权平均数**：$\bar{x}_w = \frac{\sum w_i x_i}{\sum w_i}$。当权重相等时退化为普通平均数。
5. **数据变换的影响**：
   - 加常数 $c$：平均数、中位数、众数均加 $c$（定理 13.1）。
   - 乘常数 $k$：平均数、中位数、众数均乘 $k$（定理 13.2）。
6. **分组数据**：
   - 众数组为频数最高的一组（定义 13.5）。
   - 估算平均数以组中点 $m_i$ 为代表值：$\bar{x}_{\text{est}} = \frac{\sum f_i m_i}{N}$（定义 13.7）。
   - 估算中位数以线性插值计算：$\text{Median}_{\text{est}} = L + (\frac{N/2 - F}{f}) \times w$（定义 13.9）。

### 备课建议

1. **平均数 vs 中位数的选择：** 用实际案例（如收入分布）说明何时用中位数比平均数更合理——当数据呈偏态分布或有极端值时，中位数更能反映“典型”水平。这是 DSE 常考的论述题。

2. **计算中位数的常见错误：** 学生最容易忘记“先排序”。必须反复强调：中位数的第一步永远是排序。偶数个数据时，是中间两个数据的平均，而非任取其一。

3. **加权平均数的分母：** 学生常误将分母写为“项目的个数 $k$”而非“权重之和 $\sum w_i$”。可用“总分 = 各科得分 × 学分之和，除以总学分”的比喻强化记忆。

4. **数据变换的逻辑：** 定理 13.1 和 13.2 看似简单，但学生在实际应用中容易出错——特别是乘负数时中位数的排序逆转。建议用少量数值的简单例题让学生“感受”变换前后的变化，再建立抽象理解。

5. **分组数据中位数的插值公式：** 这是本节中公式最长、学生最易背错的部分。建议不要让学生死记公式，而是理解其几何意义：在组距内按比例“走”到中位数的位置，即 $\frac{N/2 - F}{f}$ 表示在该组中走了多少比例。

6. **与后续章节的联系：** 本章的集中趋势与第 33 章“离差的度量”形成互补——前者描述数据的“中心”，后者描述数据的“散布程度”。教学时可以暗示这种对称关系，为后续章节铺垫。

## 28.1 一元不等式组 (Systems of Inequalities in One Variable)

> 本节建立不等式组的逻辑基础——“且”对应集合交集、“或”对应集合并集——并以此为框架，系统处理一元一次不等式组与一元二次不等式的求解。

### 28.1.1 解集的概念 (Concept of Solution Set)

- **定义 28.1 (不等式的解集 / Solution Set of an Inequality)**
  不等式所有解的集合，称为该不等式的**解集**，记为 $S$。
  - 与方程不同——方程的解通常为有限个（甚至唯一），而不等式的解集通常为**区间**或其并集，含有无穷多个元素。
  - Ch11 引入的区间记号（定义 11.3）及数轴图示（方法 11.2）是表示解集的主要工具。

- **定义 28.2 (一元不等式组 / System of Inequalities in One Variable)**
  由若干个含有同一未知数的一元不等式以逻辑连词“且”（$\land$）或“或”（$\lor$）联结而成的复合命题。其标准形式为：
  $$
  \begin{cases}
  f_1(x) \; \square_1 \; 0 \\
  f_2(x) \; \square_2 \; 0 \\
  \;\vdots \\
  f_n(x) \; \square_n \; 0
  \end{cases}
  $$
  其中每个 $\square_i$ 为 $>$、$<$、$\ge$ 或 $\le$ 之一。

  - **Remark：** 用花括号 $\begin{cases} \end{cases}$ 列出各不等式时，默认各不等式之间以“且”联结（即要求 $x$ 同时满足所有条件）。若需使用“或”，须以文字明确标示。

- **定理 28.1 (不等式组的解集 / Solution Set of a System)**
  设不等式组由 $n$ 个不等式 $I_1, I_2, \dots, I_n$ 以“且”联结而成，各不等式的解集分别为 $S_1, S_2, \dots, S_n$。则不等式组的解集为：
  $$S = S_1 \cap S_2 \cap \cdots \cap S_n$$
  即 $x$ 必须**同时满足**所有不等式。

  **证明：** $x$ 是不等式组的解 $\iff$ 对每个 $i = 1, 2, \dots, n$，$x$ 满足 $I_i$ $\iff$ $x \in S_i$ 对所有 $i$ 成立 $\iff$ $x \in \bigcap_{i=1}^n S_i$。

- **方法 28.1 (数轴法 / Number Line Method for Compound Inequalities)**
  对于一元一次不等式组（各不等式均为一次），最直观的求解方式是在数轴上画出各不等式的解区间，寻找它们的**公共部分（重叠区域）**。

  **步骤：**
  1. 分别解出每个不等式的解集，在数轴上以空心圆或实心点及箭头标出。
  2. 观察各区间的重叠部分——此即为不等式组的解集。
  3. 以区间记号写出最终答案。

- **方法 28.2 (二元不等式组的口诀法 / Mnemonic for Two-Inequality Systems)**
  设不等式组形如 $\begin{cases} x > a \\ x < b \end{cases}$ 或 $\begin{cases} x > a \\ x > b \end{cases}$ 等（其中 $a < b$），则可不必画数轴，直接由口诀得出：

  | 情形 | 不等式组 | 解集 | 口诀 |
  |:----:|:--------:|:----:|:----:|
  | (1) | $\begin{cases} x > a \\ x > b \end{cases}$ | $x > b$ | 同大取大 |
  | (2) | $\begin{cases} x < a \\ x < b \end{cases}$ | $x < a$ | 同小取小 |
  | (3) | $\begin{cases} x > a \\ x < b \end{cases}$ | $a < x < b$ | 大小小大中间找 |
  | (4) | $\begin{cases} x < a \\ x > b \end{cases}$ | $\varnothing$（无解） | 大大小小无处找 |

  **Remark：** 口诀法的本质是集合交集的直观判断。情形 (1) 中 $x > a$ 与 $x > b$ 的交集为 $(a, \infty) \cap (b, \infty) = (b, \infty) = \{x : x > b\}$。其余情形同理。口诀仅为记忆辅助，理解背后的集合逻辑更为重要。

- **方法 28.3 (包含等号的推广)**
  上述口诀对 $\ge$ / $\le$ 同样适用，只需将 $>$ 替换为 $\ge$、$<$ 替换为 $\le$ 即可。例如：
  - $\begin{cases} x \ge a \\ x \ge b \end{cases} \implies x \ge \max(a, b)$
  - $\begin{cases} x \ge a \\ x \le b \end{cases} \implies a \le x \le b$

  **Remark：** 当 $a = b$ 时，情形 (3) 退化为 $x = a$（单点解集）。口诀应理解为由区间交集所导出的结论，而非机械套用。

### 28.1.2 逻辑连词对集的运算 (Set Operations for Logical Connectives)

> 上一节专注于“且”（交集）的应用。本节推广至“或”（并集），并将逻辑连词的集合对应应用于一元二次不等式的求解——二次不等式的解本质上是两个（或一个、或零个）区间的“并集”或“交集”，其结构由二次函数图像与 $x$ 轴的相交情况决定。

- **定理 28.2 (逻辑连词的集合对应 / Logical Connectives as Set Operations)**
  设命题 $P(x)$ 与 $Q(x)$ 的真值集分别为 $S_P = \{x : P(x)\}$ 与 $S_Q = \{x : Q(x)\}$。则：
  1. “$P(x)$ 且 $Q(x)$”的真值集为 $S_P \cap S_Q$。
  2. “$P(x)$ 或 $Q(x)$”的真值集为 $S_P \cup S_Q$。

  **证明：**
  1. $x$ 使“$P(x)$ 且 $Q(x)$”为真 $\iff$ $P(x)$ 为真且 $Q(x)$ 为真 $\iff$ $x \in S_P$ 且 $x \in S_Q$ $\iff$ $x \in S_P \cap S_Q$。
  2. $x$ 使“$P(x)$ 或 $Q(x)$”为真 $\iff$ $P(x)$ 为真或 $Q(x)$ 为真 $\iff$ $x \in S_P$ 或 $x \in S_Q$ $\iff$ $x \in S_P \cup S_Q$。

  **Remark：** 此定理是贯穿全章的统一框架。不等式组（以“且”联结）的解集为交集；以“或”联结的不等式之解集为并集。一元二次不等式的求解（见下文）亦归结为此框架——将二次式因式分解后转化为一次因式的符号分析，实质上是在计算若干开区间的并集。

#### 一元二次不等式 (Quadratic Inequalities in One Variable)

> Ch21 建立了二次函数 $f(x) = ax^2 + bx + c$ 的图像（抛物线）及其与 $x$ 轴相交情况（由判别式 $\Delta$ 决定）。本节将此几何知识转化为不等式求解的工具。

- **定义 28.3 (一元二次不等式 / Quadratic Inequality in One Variable)**
  形如 $ax^2 + bx + c > 0$、$ax^2 + bx + c < 0$、$ax^2 + bx + c \ge 0$ 或 $ax^2 + bx + c \le 0$ 的不等式，其中 $a, b, c \in \mathbb{R}$ 且 $a \neq 0$。

- **定理 28.3 (一元二次不等式的图解法 / Graphical Method for Quadratic Inequalities)**
  设 $f(x) = ax^2 + bx + c$（$a \neq 0$），$\Delta = b^2 - 4ac$。

  | 情况 | 条件 | $a > 0$ 时 $f(x) > 0$ 的解 | $a > 0$ 时 $f(x) < 0$ 的解 |
  |:----:|:----:|:--------------------------:|:--------------------------:|
  | $\Delta > 0$ | 两不等实根 $\alpha < \beta$ | $x < \alpha$ **或** $x > \beta$ | $\alpha < x < \beta$ |
  | $\Delta = 0$ | 重根 $\alpha$ | $x \neq \alpha$ | $\varnothing$（无解） |
  | $\Delta < 0$ | 无实根 | $\mathbb{R}$（全体实数） | $\varnothing$（无解） |

  若 $a < 0$，则 $f(x) > 0$ 与 $f(x) < 0$ 的解对调（因开口向下时，$f(x)$ 的符号分布与 $a > 0$ 时相反）。

  **证明框架：** 由 Ch21 定理 21.4，$f(x) = 0$ 的根对应抛物线与 $x$ 轴的交点。
  - 若 $\Delta > 0$ 且 $a > 0$，抛物线开口向上，与 $x$ 轴交于 $\alpha, \beta$（$\alpha < \beta$）。此时 $f(x) > 0$ 对应图像在 $x$ 轴**上方**的部分——即 $x < \alpha$ 及 $x > \beta$ 两段；$f(x) < 0$ 对应图像在 $x$ 轴**下方**的部分——即 $\alpha < x < \beta$。
  - 若 $\Delta = 0$ 且 $a > 0$，抛物线开口向上，与 $x$ 轴相切于 $\alpha$。$f(x) \ge 0$ 对所有 $x$ 成立（等号仅在 $x = \alpha$ 时成立），故 $f(x) > 0 \iff x \neq \alpha$，而 $f(x) < 0$ 无解。
  - 若 $\Delta < 0$ 且 $a > 0$，整个抛物线位于 $x$ 轴上方，$f(x) > 0$ 对所有 $x \in \mathbb{R}$ 成立，$f(x) < 0$ 无解。

  **Remark：** 图解法的优势在于直观——学生可借由画出草图（开口方向 + 与 $x$ 轴交点）迅速判断 $f(x)$ 何处为正、何处为负。其理论基础为实函数的连续性与介值定理，惟 DSE 阶段无需严格证明，以图像直观理解即可。

- **定理 28.4 (一元二次不等式的代数方法——因式分解法 / Algebraic Method via Factorization)**
  设一元二次不等式 $ax^2 + bx + c \; \square \; 0$（$\square$ 为 $>$、$<$、$\ge$ 或 $\le$），且 $ax^2 + bx + c$ 可在 $\mathbb{R}$ 上分解为：
  $$a(x - \alpha)(x - \beta)$$
  其中 $\alpha \le \beta$。

  以 $\alpha, \beta$ 将实数轴分为三个区间：$(-\infty, \alpha)$、$(\alpha, \beta)$、$(\beta, \infty)$（若 $\alpha = \beta$ 则仅有两个区间）。在每个区间内取一测试点代入各因式，确定各因式及乘积的符号，从而读出不等式成立的区间。

  **Remark：** 因式分解法的核心是将二次式转化为一次因式的乘积，利用“正正得正”等符号规则将问题分解为若干区间的分析。此方法本质上是定理 28.2 中集合运算的应用——二次不等式的解集是若干开区间的并集。

- **方法 28.4 (因式分解法——步骤 / Factorization Method — Steps)**
  对一元二次不等式 $ax^2 + bx + c > 0$（以 $>$ 为例，$<$、$\ge$、$\le$ 同理）：

  1. **分解：** 将 $ax^2 + bx + c$ 因式分解为 $a(x - \alpha)(x - \beta)$（设 $\alpha \le \beta$）。若判别式 $\Delta < 0$，则无法在 $\mathbb{R}$ 上分解，需改用定理 28.3 的图解法直接判断。
  2. **定根：** $\alpha$ 及 $\beta$ 为 $ax^2 + bx + c = 0$ 的根（实根或重根）。
  3. **划分区间：** 以 $\alpha, \beta$ 为边界点，划分实数轴为若干区间。
  4. **判符号：** 在每个区间内取一个测试点，代入 $a(x - \alpha)(x - \beta)$，判断整个表达式的符号。亦可利用 $a$ 的符号及“最右侧区间符号与 $a$ 相同，每经一个单根符号翻转”的规律快速判断。
  5. **取并集：** 收集所有满足不等式的区间，以“或”（$\cup$）联结，写出解集。

- **方法 28.5 (含绝对值的一元二次不等式——平方转化法)**
  若不等式含有绝对值，但可透过平方化去绝对值后转化为一元二次不等式，则先平方再求解。常见形式如 $|ax + b| > |cx + d|$，两边平方后得 $(ax + b)^2 > (cx + d)^2$，展开合并后即为一元二次不等式。

  **原理：** 对于实数 $u, v$，$|u| > |v| \iff u^2 > v^2$（因平方函数 $t \mapsto t^2$ 在 $[0, \infty)$ 上严格递增）。

- **推论 28.1 ($\Delta < 0$ 时二次三项式的恒符号性)**
  设 $f(x) = ax^2 + bx + c$，$a \neq 0$。
  - 若 $\Delta < 0$ 且 $a > 0$，则 $f(x) > 0$ 对所有 $x \in \mathbb{R}$ 成立。
  - 若 $\Delta < 0$ 且 $a < 0$，则 $f(x) < 0$ 对所有 $x \in \mathbb{R}$ 成立。

  **证明：** $\Delta = b^2 - 4ac < 0$ 表示 $f(x) = 0$ 无实根，故抛物线不与 $x$ 轴相交。由 $a$ 的符号决定开口方向——$a > 0$ 时整条抛物线在 $x$ 轴上方（$f(x) > 0$），$a < 0$ 时整条抛物线在 $x$ 轴下方（$f(x) < 0$）。这直接由 Ch21 定理 21.4 及配方结果 $f(x) = a(x + b/(2a))^2 - \Delta/(4a)$ 得出：当 $\Delta < 0$ 时，$-\Delta/(4a)$ 与 $a$ 同号。

## 28.2 联立二元一次不等式 (Systems of Linear Inequalities in Two Variables)

> 本节将不等式的讨论从一维数轴拓展至二维坐标平面。一条二元一次不等式对应一个半平面；多条不等式的联立则定义了这些半平面的交集——即可行域。

### 28.2.1 二元一次不等式 (Linear Inequality in Two Variables)

- **定义 28.4 (二元一次不等式 / Linear Inequality in Two Variables)**
  形如 $Ax + By + C > 0$、$Ax + By + C < 0$、$Ax + By + C \ge 0$ 或 $Ax + By + C \le 0$ 的不等式，其中 $A, B, C \in \mathbb{R}$ 且 $A, B$ 不同时为 $0$。

  $Ax + By + C = 0$ 为此不等式对应的**边界直线 (Boundary Line)**。

- **定理 28.5 (半平面定理 / Half-Plane Theorem)**
  直线 $L: Ax + By + C = 0$ 将坐标平面划分为两个**开放半平面**：
  $$H_+ = \{(x, y) : Ax + By + C > 0\}, \qquad H_- = \{(x, y) : Ax + By + C < 0\}$$
  这两个半平面互为补集（以 $L$ 为边界）：
  $$\mathbb{R}^2 = H_+ \cup L \cup H_-$$
  且 $H_+ \cap H_- = \varnothing$，$H_+ \cap L = \varnothing$，$H_- \cap L = \varnothing$。

  **证明思路：** 对于平面上任一点 $(x, y)$，表达式 $Ax + By + C$ 的取值必为正、负或零恰一成立。取值为零的点全体构成 $L$。取值为正的点构成 $H_+$，取值为负的点构成 $H_-$。由 $\mathbb{R}$ 的三歧性（公理 1.4.1），此三分划互不相交且覆盖全平面。

- **方法 28.6 (边界的绘制规则 / Boundary Drawing Rules)**
  在坐标平面上表示二元一次不等式的解区域时：
  - 若不等式含等号（$\ge$ 或 $\le$）：边界直线属于解区域，画**实线 (Solid Line)**。
  - 若不等式不含等号（$>$ 或 $<$）：边界直线不属于解区域，画**虚线 (Dashed Line)**。

  **Remark：** 此规则与 Ch11 方法 11.2 中数轴端点“实心点 $\bullet$ vs 空心圆 $\circ$”的规则完全一致——实线对应“包含边界”，虚线对应“不包含边界”。这是 DSE 作图题最频繁的扣分点之一。

- **方法 28.7 (选点测试法 / Test Point Method)**
  画出边界直线后，选择平面上**不在该直线上**的任意一点（通常取原点 $(0, 0)$，若原点恰在直线上则另选他点），代入不等式检验：
  - 若代入后不等式成立，则该点所在的一侧为所求半平面。
  - 若代入后不等式不成立，则该点所在的一侧不是所求半平面，应选取**另一侧**。

  **原理：** 由定理 28.5，$H_+$ 与 $H_-$ 不相交且覆盖（除去 $L$ 后的）全平面。任一不在 $L$ 上的点必属于 $H_+$ 或 $H_-$ 恰一者。测试该点的符号即可判定其所在半平面与不等式所求半平面的关系。

  **Remark：** 选原点 $(0, 0)$ 是最方便的选择——代入后仅剩常数项 $C$。若 $C > 0$ 且不等式为 $Ax + By + C > 0$，则原点所在侧即为所求；若 $C < 0$ 且不等式为 $Ax + By + C > 0$，则需取原点的对侧。

- **推论 28.2 (系数方向的判定准则 / Coefficient Sign Rule)**
  设二元一次不等式已化为 $y$ 的显式（即 $B \neq 0$ 时，将不等号两边整理为 $y$ 与 $x$ 的形式）。若不等式形如：
  $$y > mx + c \quad \text{或} \quad y \ge mx + c$$
  则解区域为直线 $y = mx + c$ 的**上方**。
  若不等式形如：
  $$y < mx + c \quad \text{或} \quad y \le mx + c$$
  则解区域为直线 $y = mx + c$ 的**下方**。

  **证明思路：** 直线 $y = mx + c$ 将平面分为 $y > mx + c$（上方）与 $y < mx + c$（下方）两半平面。对直线上同一 $x$ 坐标，上方点的 $y$ 坐标更大。

  **Remark：** 此推论的前提是将不等式化为 $y$ 的显式（即 $y$ 的系数化为 $+1$）。若原不等式中 $y$ 的系数为负（例如 $-2y + x > 0$），须先变号整理（注意：乘除负数时不等号反转！），再应用“上大下小”准则。

### 28.2.2 二元一次不等式组 (Systems of Linear Inequalities in Two Variables)

- **定义 28.5 (二元一次不等式组 / System of Linear Inequalities in Two Variables)**
  由若干个二元一次不等式以“且”联结而成的复合条件：
  $$
  \begin{cases}
  A_1 x + B_1 y + C_1 \; \square_1 \; 0 \\
  A_2 x + B_2 y + C_2 \; \square_2 \; 0 \\
  \;\vdots \\
  A_n x + B_n y + C_n \; \square_n \; 0
  \end{cases}
  $$
  其中每个 $\square_i$ 为 $\ge$、$\le$、$>$ 或 $<$。

- **定理 28.6 (可行域定理 / Feasible Region Theorem)**
  二元一次不等式组的解区域为各不等式所对应半平面的**交集**：
  $$R = R_1 \cap R_2 \cap \cdots \cap R_n$$
  此交集称为此不等式组的**可行域 (Feasible Region)**。

  **证明：** 直接由定理 28.1 推广至二元情形：点 $(x, y)$ 满足不等式组 $\iff$ 对每个 $i$，$(x, y)$ 满足第 $i$ 条不等式 $\iff$ $(x, y) \in R_i$ 对所有 $i$ 成立 $\iff$ $(x, y) \in \bigcap_{i=1}^n R_i$。

- **定理 28.7 (可行域的形状 / Shape of the Feasible Region)**
  由有限个二元一次不等式定义的可行域为**凸多边形区域（Convex Polygonal Region）**，可能为有限（闭凸多边形）或无限（无界凸多边形区域）。

  **说明：** 每个二元一次不等式定义一个半平面（凸集）。凸集的交集仍为凸集。由若干半平面的交集所得之凸集为凸多面体（在二维即凸多边形区域）。DSE 阶段无需给出严格证明，理解几何直观即可——若干条直线像“刀”一样切割平面，留下的公共部分必为由直线段围成的多边形区域。

- **方法 28.8 (画出可行域的步骤 / Steps for Sketching a Feasible Region)**
  1. **画各边界直线：** 对每条不等式，画出其对应的边界直线（实线或虚线取决于是否含等号）。
  2. **定各半平面：** 以选点测试法（方法 28.7）确定每条不等式对应的半平面。
  3. **取交集：** 在各不等式对应的半平面内以箭头或阴影标记，所有标记重叠的区域即为可行域。
  4. **标注：** 在可行域内标注“$R$”或打上斜线，边界的实线/虚线状态须清楚可辨。

  **Remark：** DSE 作图题通常要求将可行域以斜线或阴影清楚标出。若边界为虚线，表示该边界不属于可行域，需特别留意。

## 28.3 线性规划 (Linear Programming)

> 线性规划是二元一次不等式组的**应用延伸**——在可行域的限制下，求一个二元一次表达式（目标函数）的最大值或最小值。这是 DSE 中少数需要综合运用代数、几何与逻辑推理的课题，也是现实中资源分配、生产排程等最优化问题的数学模型。

- **定义 28.6 (线性规划问题 / Linear Programming Problem)**
  在给定的**约束条件 (Constraints)**（由若干二元一次不等式组成的系统）下，求**目标函数 (Objective Function)** $z = px + qy$（$p, q$ 为常数，不同时为 $0$）的最大值或最小值。

  **标准形式：**
  $$
  \begin{aligned}
  \text{Maximize / Minimize} \quad & z = px + qy \\
  \text{Subject to} \quad &
  \begin{cases}
  A_1 x + B_1 y + C_1 \; \square_1 \; 0 \\
  A_2 x + B_2 y + C_2 \; \square_2 \; 0 \\
  \;\vdots \\
  A_n x + B_n y + C_n \; \square_n \; 0
  \end{cases}
  \end{aligned}
  $$

- **定义 28.7 (可行解与最优解 / Feasible Solution & Optimal Solution)**
  - **可行解 (Feasible Solution)：** 满足所有约束条件的 $(x, y)$。
  - **最优解 (Optimal Solution)：** 在所有可行解中，使目标函数取得最大值（或最小值）的 $(x, y)$。

- **定理 28.8 (顶点原理 / Vertex Principle)**
  若线性规划问题的可行域为非空的有界凸多边形（闭区域），且目标函数 $z = px + qy$ 在此可行域上可取得最大值（或最小值），则最值必在可行域的**某个顶点**处取得。

  **证明思路（几何直观）：** 目标函数 $z = px + qy$ 可改写为 $y = -\dfrac{p}{q}x + \dfrac{z}{q}$（设 $q \neq 0$）。固定 $z$ 时，这是一条斜率为 $-p/q$ 的直线；不同 $z$ 值对应此直线族中的不同直线——$z$ 越大，直线的 $y$ 截距越大（若 $q > 0$）。将此直线像一把“尺子”沿可行域的方向平移：当直线首次接触可行域时 $z$ 最小，最后脱离可行域时 $z$ 最大。由于可行域为凸多边形，首次及最后接触点必为多边形的**顶点**。

  **Remark：** 若最优解出现在整条边界上（例如目标函数的直线斜率恰好与可行域某一边界平行），则该边界上的所有点均为最优解。但该边界的端点（即顶点）亦是可行解，代入目标函数可得相同最值。因此，仅检查顶点即已足够。

  **Remark：** 对于无界可行域，最大值或最小值可能不存在（目标函数可无限增大或减小）。此时需结合几何直观判断。

- **方法 28.9 (线性规划的标准解题步骤 / Standard Procedure for Linear Programming)**
  1. **设变量：** 明确 $x$、$y$ 分别代表什么（常见为两种产品的产量）。
  2. **列约束：** 将题目中的所有限制条件（资源限制、非负条件 $x \ge 0, y \ge 0$ 等）转化为二元一次不等式。
  3. **写目标：** 将题目要求的最大化/最小化量写成 $z = px + qy$ 的形式。
  4. **画可行域：** 在坐标平面上绘出所有约束条件对应的半平面，标出可行域 $R$（方法 28.8）。
  5. **求顶点：** 计算可行域各顶点的坐标。顶点为两条边界直线的交点——解该两直线的联立方程即得。若可行域为三角形，需求三个顶点；为四边形则需求四个。
  6. **代目标：** 将各顶点坐标代入 $z = px + qy$。
  7. **比较得结论：** 比较各顶点的目标函数值，取最大值或最小值。以文字写出最终答案（例如“当 $x = \underline{\quad}, y = \underline{\quad}$ 时，$z$ 的最大值为 $\underline{\quad}$”）。

  **Remark：** DSE 线性规划应用题通常要求以**整数**为答案（因为实际问题中产量等不可为分数）。若最优顶点的坐标非整数，需检查邻近的整数格点是否仍在可行域内，选取使目标函数最优的整数点。

- **推论 28.3 (目标函数的几何意义)**
  目标函数 $z = px + qy$ 的值对应一直线族的 $y$ 截距（以 $z/q$ 表示，设 $q > 0$）。因此：
  - 求 $z$ 的最大值 $\iff$ 在可行域内，将斜率为 $-p/q$ 的直线尽可能向 $y$ 截距**增大**的方向平移，求最后一个接触点。
  - 求 $z$ 的最小值 $\iff$ 在可行域内，将斜率为 $-p/q$ 的直线尽可能向 $y$ 截距**减小**的方向平移，求第一个接触点。

  **Remark：** 此几何观点有助于检查答案的合理性——若目标函数直线的斜率介于可行域某两条边界直线的斜率之间，则最优解必在某一顶点；若斜率与某一边界平行，则该边界上所有点均可为最优解。

## 第二十八章总结

1. **不等式组的逻辑基础（定理 28.1–28.2）：** “且”对应集合交集 $\cap$、“或”对应集合并集 $\cup$。一元不等式组（以“且”联结）的解集为各不等式解集的交集。数轴法（方法 28.1）与口诀法（方法 28.2）是求解一元一次不等式组的两种互补工具——前者直观，后者快捷。

2. **一元二次不等式（定理 28.3–28.4）：** 有两条求解路径。图解法利用 Ch21 中二次函数抛物线的图像——由 $\Delta$ 及 $a$ 的符号决定 $f(x)$ 何处为正、何处为负。代数法将二次式因式分解后进行符号分析，本质上是定理 28.2 的应用——二次不等式的解为若干区间的并集。$\Delta < 0$ 时二次三项式符号恒定（推论 28.1）。

3. **二元一次不等式（定理 28.5、方法 28.6–28.7）：** 直线 $Ax + By + C = 0$ 将平面划分为两个半平面。边界绘制规则——含等号画实线、不含等号画虚线——与 Ch11 数轴端点的“实心点 vs 空心圆”规则一致。选点测试法（取原点）是用以判定所求半平面的基本方法。

4. **联立二元一次不等式（定理 28.6–28.7、方法 28.8）：** 不等式组的解为各半平面的交集——即**可行域**，必为凸多边形区域。绘制可行域需依次画各边界直线、定各半平面方向、取交集标出。

5. **线性规划（定理 28.8、方法 28.9）：** 在约束条件定义的可行域上求目标函数 $z = px + qy$ 的最值。**顶点原理**是核心——若可行域为有界闭凸多边形，最优解必在顶点处取得。解题步骤依次为：设变量 $\to$ 列约束 $\to$ 写目标 $\to$ 画可行域 $\to$ 求顶点 $\to$ 代目标 $\to$ 比较得结论。几何上，目标函数对应一族平行线——求最值即相当于将此直线沿可行域平移，寻找首次/最后接触点。

### 备课建议

1. **“且”与“或”的区分是首要难点：**
   学生容易混淆不等式组（默认“且”，求交集）与分类讨论（使用“或”，求并集）。建议在引入时直接以集合语言建立框架——“且 $\to \cap$，或 $\to \cup$”——并以具体数字及数轴反复演示。一元二次不等式 $x^2 - 4 > 0$ 的解为 $x < -2$ **或** $x > 2$（两个区间的并集），而联立不等式组 $\begin{cases} x > -2 \\ x < 2 \end{cases}$ 的解为 $-2 < x < 2$（一个区间，即交集）。这组对比例可帮助学生体会“两个区间的并集”与“两个区间的交集”之别。

2. **一元二次不等式——先图后代：**
   建议教学顺序为：先教图解法（利用已学的抛物线图像），让学生建立“$f(x) > 0$ 即图像在 $x$ 轴上方”的几何直觉，再引入因式分解的符号表分析法。两种方法并行教授，让学生理解它们是同一事物的不同视角。对于 $\Delta < 0$ 的恒正/恒负情况，建议以具体数字验证（例如 $x^2 + 1 > 0$ 对所有 $x$ 成立），避免死记结论。

3. **边界的实线/虚线——DSE 高频扣分点：**
   二元一次不等式的作图题中，实线（含等号）与虚线（不含等号）的区分是最频繁的扣分项目。建议在课堂练习中反复强调，并设计“给定不等式 $\to$ 判断边界应为实线还是虚线”的简短速练。可类比 Ch11 数轴上的空心圆 $\circ$ 与实心点 $\bullet$。

4. **选点测试——以原点为默认首选：**
   除非边界直线通过原点，否则一律建议学生以 $(0, 0)$ 为测试点。代入 $(0, 0)$ 后不等式化为 $C > 0$（或 $<$ 等），极易判断。若原点恰在直线上，可选 $(1, 0)$ 或 $(0, 1)$（亦易计算）。

5. **线性规划的“翻译”能力——文字转数学模型：**
   DSE 线性规划题目通常以长文字题形式出现，学生的首要障碍往往是将文字中的限制条件转化为不等式。建议系统训练以下常见的文字-数学转化：

   | 文字描述 | 数学表达 |
   |:--------|:--------|
   | “$x$ 不少于 $5$” | $x \ge 5$ |
   | “$y$ 最多为 $10$” | $y \le 10$ |
   | “总数不超过 $100$” | $x + y \le 100$ |
   | “$A$ 的两倍加 $B$ 至少为 $30$” | $2x + y \ge 30$ |
   | “非负条件” | $x \ge 0,\; y \ge 0$ |

6. **线性规划——顶点法 vs 平移法：**
   教材以顶点法（方法 28.9 步骤 6–7）为标准解题步骤——求各顶点坐标后代入比较，最稳妥且不易出错。但建议在讲解时辅以平移法的几何直观（推论 28.3），让学生理解“为什么最优解在顶点”而非机械记忆步骤。若有条件，使用动态几何软件（如 GeoGebra）演示直线族平移的过程效果极佳。

7. **整数解问题——线性规划的常见变体：**
   当题目要求 $x, y$ 为整数（例如“生产整数件产品”）而最优顶点非整数时，学生需检查邻近的整数格点。建议提醒学生：不可简单地四舍五入——须逐一检查附近的整数点是否仍在可行域内，再比较目标函数值。此为 DSE 常见考法。

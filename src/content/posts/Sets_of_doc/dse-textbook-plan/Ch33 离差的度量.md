**模块目标：** 理解数据离散程度的概念；掌握分布域、四分位数间距、标准差与方差等离差度量的定义、计算方法及性质；能绘制并阐释框线图；理解标准分的概念及其性质；掌握数据线性变换对离差度量的影响法则。

> **记号约定（多重集与数据集）：** 本章以第 19 章的函数语言与多重集记号描述数据及其变换。将一组 $N$ 个数据值记为多重集：
> $$X = \{x_1, x_2, \dots, x_N\}$$
> 其中元素允许重复，其计数函数 $\nu_X$（即统计中的频数分布）满足 $\sum_{x} \nu_X(x) = N$。$N$ 称为 $X$ 的**容量**。设 $f$ 为函数，定义
> $$f(X) = \{\, f(x) \mid x \in X \,\} = \{f(x_1), f(x_2), \dots, f(x_N)\}$$
> 为 $X$ 在 $f$ 下的**多重集像**（见第 19 章），其计数函数 $\nu_{f(X)}(y) = \sum_{f(x)=y} \nu_X(x)$。在不引起混淆时，$\mu_X$ 与 $\sigma_X$ 分别记 $X$ 的平均数与标准差；变换后数据集 $f(X)$ 的对应统计量则记为 $\mu_{f(X)}$ 与 $\sigma_{f(X)}$。

## 33.1 数据的离散度量

> 本节定义四种基本的离散程度度量：分布域、四分位数间距、标准差（及方差）和标准分。其中标准差是统计学中最核心的离差指标；标准分则消除了数据的单位和量纲，使来自不同数据集的数值可以公平比较其相对位置。

### 33.1.1 分布域

- **定义 33.1（分布域 / Range）**
  设一组数据为 $x_1, x_2, \dots, x_n$（$n \ge 1$），记 $x_{\max} = \max\{x_1, \dots, x_n\}$，$x_{\min} = \min\{x_1, \dots, x_n\}$，则其**分布域**（亦称**极差**）$R$ 定义为：
  $$R = x_{\max} - x_{\min}$$

- **性质 33.1（分布域的基本性质）**
  1. $R \ge 0$，且 $R = 0$ 当且仅当所有数据相等。
  2. 分布域计算简便，仅依赖数据的两个极端值。
  3. **缺点：** 分布域对极端值（outliers）极为敏感——单一个异常值即可使 $R$ 大幅改变，不能反映数据主体的波动情况。

### 33.1.2 百分位数与四分位数间距

- **定义 33.2（百分位数 / Percentile）**
  设 $n$ 个数据按升序排列为 $x_{(1)} \le x_{(2)} \le \cdots \le x_{(n)}$。对 $0 < k < 100$，第 $k$ **百分位数** $P_k$ 是满足以下条件的值：数据中约有 $k\%$ 的数值不大于 $P_k$，且约有 $(100-k)\%$ 的数值不小于 $P_k$。

  对**不分组数据**，$P_k$ 通过位置公式求取：计算 $\frac{k}{100}(n+1)$。
  - 若结果为整数 $m$，则 $P_k = x_{(m)}$。
  - 若结果非整数，设其整数部分为 $m$、小数部分为 $d$（$0 < d < 1$），则以线性插值法：
    $$P_k = x_{(m)} + d \cdot (x_{(m+1)} - x_{(m)})$$

  对**分组数据**，$P_k$ 从累积频数多边形（或累积频数曲线）上读取：在纵轴上定位累积频数 $\frac{k}{100}N$（$N$ 为总频数），对应的横轴值即为 $P_k$。

- **定义 33.3（四分位数 / Quartiles）**
  三个特殊的百分位数——第 25、第 50、第 75 百分位数——分别称为**第一四分位数**（下四分位数）、**第二四分位数**（中位数）、**第三四分位数**（上四分位数），记为 $Q_1$, $Q_2$, $Q_3$：
  $$Q_1 = P_{25}, \quad Q_2 = P_{50}, \quad Q_3 = P_{75}$$

  $Q_2$ 即为第 13 章所定义的中位数（定义 13.2）。四分位数将有序数据等分为四个部分，每部分约含 $25\%$ 的数据。

- **定义 33.4（四分位数间距 / Interquartile Range, IQR）**
  上四分位数与下四分位数之差称为**四分位数间距**：
  $$\mathrm{IQR} = Q_3 - Q_1$$

- **性质 33.2（IQR 的基本性质）**
  1. $\mathrm{IQR} \ge 0$。
  2. IQR 涵盖了中间 $50\%$ 数据的范围。
  3. **稳健性（Robustness）：** 与分布域不同，IQR 只依赖数据的中间部分，不受两端极端值的干扰——即使数据的最大值或最小值发生剧烈变化，只要 $Q_1$ 和 $Q_3$ 不变，IQR 保持不变。因此 IQR 比分布域更适合描述有离群值的数据集的离散程度。

### 33.1.3 标准差与方差

> 标准差是统计学中最重要的离差度量。其核心思想是：衡量每个数据点与平均数的“平均距离”。为避免正负离差相互抵消（性质 13.1），先将离差平方后再取平均，最后开方以还原量纲。

设一组数据为 $x_1, x_2, \dots, x_N$，其算术平均数为 $\mu = \frac{1}{N}\sum_{i=1}^{N} x_i$。

- **定义 33.5（总体方差 / Population Variance）**
  各数据与平均数之差的平方的算术平均数：
  $$\sigma^2 = \frac{1}{N}\sum_{i=1}^{N} (x_i - \mu)^2$$

- **定义 33.6（总体标准差 / Population Standard Deviation）**
  方差的非负平方根：
  $$\sigma = \sqrt{\frac{1}{N}\sum_{i=1}^{N} (x_i - \mu)^2}$$

  > 标准差的量纲与原数据相同（方差为原数据量纲的平方），因此标准差在实际解释中更为直观。

- **性质 33.3（标准差的基本性质）**
  1. $\sigma \ge 0$，且 $\sigma = 0$ 当且仅当所有数据相等（即 $x_1 = x_2 = \cdots = x_N = \mu$）。
  2. $\sigma$ 越大 $\implies$ 数据越分散（波动大）；$\sigma$ 越小 $\implies$ 数据越集中（波动小）。
  3. 标准差对极端值敏感——每个数据点都参与计算，因此异常值会通过平方项显著影响标准差。

- **定理 33.1（方差的计算公式 / Computational Formula for Variance）**
  $$\sigma^2 = \frac{1}{N}\sum_{i=1}^{N} x_i^{\,2} - \mu^2 = \frac{1}{N}\sum_{i=1}^{N} x_i^{\,2} - \left(\frac{1}{N}\sum_{i=1}^{N} x_i\right)^{\!2}$$

  **证明：**
  $$\begin{aligned}
  \sigma^2 &= \frac{1}{N}\sum_{i=1}^{N} (x_i - \mu)^2 \\[4pt]
  &= \frac{1}{N}\sum_{i=1}^{N} \bigl(x_i^{\,2} - 2\mu x_i + \mu^2\bigr) \\[4pt]
  &= \frac{1}{N}\sum_{i=1}^{N} x_i^{\,2} - 2\mu \cdot \frac{1}{N}\sum_{i=1}^{N} x_i + \mu^2 \\[4pt]
  &= \frac{1}{N}\sum_{i=1}^{N} x_i^{\,2} - 2\mu^2 + \mu^2 \\[4pt]
  &= \frac{1}{N}\sum_{i=1}^{N} x_i^{\,2} - \mu^2
  \end{aligned}$$
  将 $\mu = \frac{1}{N}\sum_{i=1}^{N} x_i$ 代入即得第二等式。$\square$

  > 定理 33.1 在实际计算中极为实用：只需计算 $\sum x_i$ 和 $\sum x_i^{\,2}$ 即可求得方差和标准差，无需逐一计算每个数据的离差。这也是科学计算器 `STAT` 模式背后的原理。

### 33.1.4 标准分

> 标准分消除了数据的单位和量纲，使来自不同数据集（甚至不同科目、不同尺度）的数值可以公平比较其相对位置。

- **定义 33.7（标准分 / Standard Score / z-score）**
  设数据的总体平均数为 $\mu$、标准差为 $\sigma$（$\sigma > 0$），则数值 $x$ 的**标准分** $z$ 定义为：
  $$z = \frac{x - \mu}{\sigma}$$

  $z$ 表示 $x$ 偏离平均数 $\mu$ 多少个标准差：$z > 0$ 表示 $x$ 在平均数之上；$z < 0$ 表示 $x$ 在平均数之下。

- **性质 33.4（标准分的基本性质）**
  对任意数据集 $\{x_1, \dots, x_N\}$，其对应的标准分 $\{z_1, \dots, z_N\}$（其中 $z_i = (x_i - \mu)/\sigma$）满足：
  1. 标准分的平均数恒为 $0$：$\displaystyle\frac{1}{N}\sum_{i=1}^{N} z_i = 0$
  2. 标准分的标准差（及方差）恒为 $1$：$\sigma_z = 1$，$\sigma^2_z = 1$

  **证明：**
  1. $\sum z_i = \sum \frac{x_i - \mu}{\sigma} = \frac{1}{\sigma}\sum (x_i - \mu) = 0$（由性质 13.1）。
  2. $\sigma_z^2 = \frac{1}{N}\sum (z_i - 0)^2 = \frac{1}{N}\sum \left(\frac{x_i - \mu}{\sigma}\right)^2 = \frac{1}{\sigma^2} \cdot \frac{1}{N}\sum (x_i - \mu)^2 = \frac{\sigma^2}{\sigma^2} = 1$，故 $\sigma_z = 1$。

  > 性质 33.4 说明：将任何一组数据标准化后，得到的新数据集自动具有平均数 $0$ 和标准差 $1$——这正是标准分名称的由来。

## 33.2 框线图

> 框线图（Box-and-Whisker Plot）以五个关键数值概括数据的分布，是一种简洁而资讯丰富的统计图。它同时展示了数据的中心位置（中位数）、离散程度（IQR 及全距）和偏态方向。

- **定义 33.8（五数概括 / Five-Number Summary）**
  一组数据的**五数概括**由以下五个统计量组成：
  $$\text{Min},\quad Q_1,\quad Q_2,\quad Q_3,\quad \text{Max}$$
  即最小值、下四分位数、中位数、上四分位数、最大值。

- **定义 33.9（框线图 / Box-and-Whisker Plot）**
  基于五数概括绘制的统计图：
  1. 以 $Q_1$ 和 $Q_3$ 为边界绘制一个矩形**箱体**（Box），箱体的长度即为 $\mathrm{IQR} = Q_3 - Q_1$。
  2. 在箱体内以一条垂直线标记中位数 $Q_2$ 的位置。
  3. 从箱体两端各引出一条线段（**触须** / Whisker），分别延伸至最小值 Min 和最大值 Max。

  > 框线图不显示数据的具体分布细节（如数据点密度、多峰等），但能直观展示数据的中心趋势、离散范围和偏态。

- **框线图的形态分析：**
  - **对称分布（Symmetric）：** 中位数 $\approx$ 箱体中央，两侧触须长度相近。此时 $\text{Mean} \approx \text{Median}$。
  - **正偏（Positively Skewed / Right-Skewed）：** 中位数偏近 $Q_1$，上方触须较下方触须长。数据的尾部向右延伸。此时 $\text{Mean} > \text{Median}$。
  - **负偏（Negatively Skewed / Left-Skewed）：** 中位数偏近 $Q_3$，下方触须较上方触须长。数据的尾部向左延伸。此时 $\text{Mean} < \text{Median}$。

  > 框线图的偏态判断基于箱体中线的位置与触须长度的对比，而非精确的偏态系数。在 DSE 中，框线图常用于并列比较两组或以上数据的分布差异（如比较不同班级的考试成绩分布）。

## 33.3 数据变换对统计量的影响

> 本节运用第 19 章的集合像记号 $f(X)$，统一讨论线性变换对集中趋势度量与离差度量两类统计量的影响：先分别给出平移与缩放对两类统计量的独立效应，再整合为涵盖所有统计量的线性变换定理。

设原数据集为 $X = \{x_1, x_2, \dots, x_N\}$，其平均数为 $\mu_X$、中位数为 $\mathrm{Med}_X$、众数为 $\mathrm{Mo}_X$、标准差为 $\sigma_X$。

- **定理 33.2（平移对集中趋势的影响 / Effect of Translation on Central Tendency）**
  设 $f(x) = x + c$（$c$ 为常数），则 $f(X) = \{x_1 + c,\; \dots,\; x_N + c\}$，且：
  - $\mu_{f(X)} = \mu_X + c$
  - $\mathrm{Med}_{f(X)} = \mathrm{Med}_X + c$
  - $\mathrm{Mo}_{f(X)} = \mathrm{Mo}_X + c$

  **证明（平均数）：** $\displaystyle\mu_{f(X)} = \frac{1}{N}\sum_{i=1}^{N} (x_i + c) = \frac{1}{N}\Bigl(\sum_{i=1}^{N} x_i + Nc\Bigr) = \mu_X + c$。

  **证明（中位数）：** 加上常数不改变数据的相对大小排序——$x_{(i)} + c$ 对所有 $i$ 仍保持升序。因此各百分位数的位置不变、数值均平移 $c$；特别地，中位数亦平移 $c$。

  **证明（众数）：** 加上常数不改变各数值的重数分布：$\nu_{f(X)}(x + c) = \nu_X(x)$，故频数最高的数值亦平移 $c$。

- **定理 33.3（缩放对集中趋势的影响 / Effect of Scaling on Central Tendency）**
  设 $g(x) = k x$（$k \neq 0$），则 $g(X) = \{k x_1,\; \dots,\; k x_N\}$，且：
  - $\mu_{g(X)} = k \mu_X$
  - $\mathrm{Med}_{g(X)} = k \cdot \mathrm{Med}_X$
  - $\mathrm{Mo}_{g(X)} = k \cdot \mathrm{Mo}_X$

  **证明（平均数）：** $\displaystyle\mu_{g(X)} = \frac{1}{N}\sum_{i=1}^{N} k x_i = k \cdot \frac{1}{N}\sum_{i=1}^{N} x_i = k \mu_X$。

  **证明（中位数）：** 若 $k > 0$，排序不变，$g(x_{(i)}) = k x_{(i)}$，中位数随之缩放 $k$ 倍。若 $k < 0$，排序逆转——$g(x_{(i)}) = k x_{(N-i+1)}$——但中位数的位置经变换后仍对应原中位数的 $k$ 倍。

  **证明（众数）：** 乘以 $k$ 仅改变数值刻度，不改变重数分布（$\nu_{g(X)}(k x) = \nu_X(x)$），故众数亦缩放 $k$ 倍。

- **定理 33.4（标准差的平移不变性 / Translation Invariance of SD）**
  设 $f(x) = x + c$，则：
  $$\sigma_{f(X)} = \sigma_X, \qquad \sigma^2_{f(X)} = \sigma^2_X$$

  **证明：** 由定理 33.2，$\mu_{f(X)} = \mu_X + c$。对每个 $x_i \in X$：
  $$f(x_i) - \mu_{f(X)} = (x_i + c) - (\mu_X + c) = x_i - \mu_X$$
  因此 $\displaystyle\sum_{i=1}^{N} (f(x_i) - \mu_{f(X)})^2 = \sum_{i=1}^{N} (x_i - \mu_X)^2$。代回定义即得 $\sigma^2_{f(X)} = \sigma^2_X$，从而 $\sigma_{f(X)} = \sigma_X$。$\square$

  > 平移不改变数据之间的相对距离——$f(X)$ 中任意两点的差与 $X$ 中对应两点的差相等。这与定理 33.2 对照：平移使集中趋势“移位”但不改离散程度。

- **定理 33.5（标准差的缩放性质 / Scaling Property of SD）**
  设 $g(x) = k x$（$k \neq 0$），则：
  $$\sigma_{g(X)} = |k| \cdot \sigma_X, \qquad \sigma^2_{g(X)} = k^2 \cdot \sigma^2_X$$

  **证明：** 由定理 33.3，$\mu_{g(X)} = k \mu_X$。对每个 $x_i \in X$：
  $$g(x_i) - \mu_{g(X)} = k x_i - k \mu_X = k(x_i - \mu_X)$$
  因此：
  $$\sigma^2_{g(X)} = \frac{1}{N}\sum_{i=1}^{N} k^2 (x_i - \mu_X)^2 = k^2 \cdot \frac{1}{N}\sum_{i=1}^{N} (x_i - \mu_X)^2 = k^2 \sigma^2_X$$
  开方即得 $\sigma_{g(X)} = |k| \sigma_X$（因标准差非负）。$\square$

- **定理 33.6（线性变换对统计量的统一影响 / Effect of Linear Transformation on All Statistics）**
  将定理 33.2–33.5 合并，推广至所有统计量。设 $h(x) = a x + b$（$a \neq 0$），则 $h(X) = \{a x_1 + b,\; \dots,\; a x_N + b\}$，且：

  | 统计量 | 原数据集 $X$ | 变换后 $h(X)$ | 类别 |
  |--------|-------------|-------------|------|
  | 平均数 $\mu$ | $\mu_X$ | $\mu_{h(X)} = a \mu_X + b$ | 集中趋势 |
  | 中位数 $\mathrm{Med}$ | $\mathrm{Med}_X$ | $\mathrm{Med}_{h(X)} = a \cdot \mathrm{Med}_X + b$ | 集中趋势 |
  | 众数 $\mathrm{Mo}$ | $\mathrm{Mo}_X$ | $\mathrm{Mo}_{h(X)} = a \cdot \mathrm{Mo}_X + b$ | 集中趋势 |
  | 分布域 $R$ | $R_X$ | $R_{h(X)} = \|a\| \cdot R_X$ | 离差 |
  | 四分位数间距 $\mathrm{IQR}$ | $\mathrm{IQR}_X$ | $\mathrm{IQR}_{h(X)} = \|a\| \cdot \mathrm{IQR}_X$ | 离差 |
  | 标准差 $\sigma$ | $\sigma_X$ | $\sigma_{h(X)} = \|a\| \cdot \sigma_X$ | 离差 |
  | 方差 $\sigma^2$ | $\sigma^2_X$ | $\sigma^2_{h(X)} = a^2 \cdot \sigma^2_X$ | 离差 |

  **证明：** 集中趋势部分由定理 33.2（平移）和定理 33.3（缩放）合并即得——先乘 $a$ 再平移 $b$。

  $\blacktriangleright$ **分布域：** 设 $x_{\max}, x_{\min}$ 为 $X$ 的最大值与最小值。
  - 若 $a > 0$：$h(x_{\max}) = a x_{\max} + b$，$h(x_{\min}) = a x_{\min} + b$，因此 $R_{h(X)} = a(x_{\max} - x_{\min}) = a R_X$。
  - 若 $a < 0$：乘以负数使大小顺序逆转——$h(X)$ 的最大值为 $a x_{\min} + b$，最小值为 $a x_{\max} + b$，因此 $R_{h(X)} = -a R_X = |a| R_X$。

  $\blacktriangleright$ **四分位数间距：** $Q_{k,\,h(X)} = a Q_{k,X} + b$（$k = 1, 2, 3$），从而有：
  $$\mathrm{IQR}_{h(X)} = (a Q_{3,X} + b) - (a Q_{1,X} + b) = a(Q_{3,X} - Q_{1,X})$$
  考虑大小顺序，最终 $\mathrm{IQR}_{h(X)} = |a| \cdot \mathrm{IQR}_X$。

  $\blacktriangleright$ **标准差与方差：** 注意到 $h = g \circ f$，其中 $f(x) = x + b$（平移）、$g(x) = a x$（缩放），即 $h(X) = g(f(X))$。由定理 33.4，$\sigma_{f(X)} = \sigma_X$；再由定理 33.5，$\sigma_{g(f(X))} = |a| \cdot \sigma_{f(X)} = |a| \cdot \sigma_X$。因此 $\sigma_{h(X)} = |a| \sigma_X$，$\sigma^2_{h(X)} = a^2 \sigma^2_X$。$\square$

  > **核心记忆法则：** 加减只变位置，乘除才改波动。
  > - 平移 $b$：集中趋势 $+b$；离差度量**不变**。
  > - 缩放 $a$：集中趋势 $\times a$；含量纲离差度量 $\times |a|$；方差 $\times a^2$。

- **推论 33.1（标准分在正线性变换下的不变性 / Invariance of z-score under Positive Linear Transformation）**
  设 $h(x) = a x + b$ 且 $a > 0$。对任意 $x \in X$，其在原数据集 $X$ 中的标准分与在变换后数据集 $h(X)$ 中的标准分相等：
  $$z_{h(X)}(h(x)) = z_X(x)$$

  **证明：** 由定理 33.6，$\mu_{h(X)} = a \mu_X + b$，$\sigma_{h(X)} = a \sigma_X$（因 $a > 0$，$|a| = a$）。则：
  $$z_{h(X)}(h(x)) = \frac{h(x) - \mu_{h(X)}}{\sigma_{h(X)}} = \frac{(a x + b) - (a \mu_X + b)}{a \sigma_X} = \frac{a(x - \mu_X)}{a \sigma_X} = z_X(x)$$
  $\square$

  > 推论 33.1 的直观意义是：对数据进行任何“加分”或“正向缩放”操作，每个数值的相对排名（标准分）保持不变——因为所有人在同一把尺下被同时拉伸和平移。这解释了为何标准分在比较不同科目或不同年份的考试成绩时如此有用。

1. **样本与总体的区分：** 本章使用 $\sigma^2$ 和 $\sigma$ 表示总体方差和总体标准差（分母为 $N$）。在统计推断中，若数据为样本，则样本方差的分母为 $N-1$（记为 $s^2$）。DSE 考试中，除非题目明确指明为“样本”，否则预设使用总体公式。
2. **计算器使用：** 本章涉及的计算量较大（特别是标准差的计算），建议熟练使用科学计算器的 `STAT` 模式：输入数据后可直接读取 $\bar{x}$、$\sigma x$（总体标准差）和 $sx$（样本标准差）。
3. **与后续章节的关联：** 四分位数与百分位数的概念在第 34 章“排列与组合”及第 35 章“概率(II)”中处理分位数相关概率问题时会再次用到。数据线性变换的性质在第 34 章及第 35 章的期望值与方差计算中亦会反复出现。

## 26.1 弧度法 (Radian Measure)

> **学习目标：** 建立角度与实数之间的一一对应——这是将三角函数定义为 $\mathbb{R} \to \mathbb{R}$ 实值函数的先决条件。

Ch09 中讨论的角以“度”（$^\circ$）为单位。度是将周角分为 $360$ 等份的几何度量，本身并非实数。要将三角函数定义为以实数为自变量的函数，必须建立角度与实数之间的自然对应——这正是弧度制的目的。

### 26.1.1 弧度的定义

*   **定义 26.1 (弧度制 / Radian Measure):**
    在半径为 $r$ 的圆中，若一段弧的长度等于 $r$，则该弧所对的圆心角定义为 **$1$ 弧度**（$1$ radian，记作 $1$ rad）。
    一般地，弧长为 $s$ 的弧所对的圆心角为：
    $$\theta = \frac{s}{r} \quad \text{(rad)}$$

    弧度是**弧长与半径的比值**——两个长度之比，因此是一个**纯数**（无量纲）。这正是弧度能与实数建立一一对应的根本原因：每一个实数 $\theta$ 对应唯一一个弧度为 $\theta$ 的角，反之亦然。

> **核心意义（仅为有微积分基础或正在学习M1,M2的学生说明）：** 弧度制将“角”这一几何概念转化为实数，使三角函数成为真正的 $\mathbb{R} \to \mathbb{R}$ 实值函数。在微积分中，$(\sin x)' = \cos x$ 等公式仅在弧度制下成立——其根源在于 $\lim_{x\to 0}\frac{\sin x}{x}=1$ 的证明依赖于弧度的定义。

### 26.1.2 弧度与角度的转换

*   **定理 26.1 (弧度与角度的转换 / Degree-Radian Conversion):**
    整个圆周的弧长 $s = 2\pi r$，对应的弧度为 $2\pi$ rad，而角度为 $360^\circ$，故：
    $$\pi \text{ rad} = 180^\circ$$
    转换公式：
    *   角度 $\to$ 弧度：$\theta_{\text{rad}} = \dfrac{\pi}{180^\circ} \times \theta_{\text{deg}}$
    *   弧度 $\to$ 角度：$\theta_{\text{deg}} = \dfrac{180^\circ}{\pi} \times \theta_{\text{rad}}$

*   **常用对照表：**

    | 角度 | $0^\circ$ | $30^\circ$ | $45^\circ$ | $60^\circ$ | $90^\circ$ | $180^\circ$ | $270^\circ$ | $360^\circ$ |
    | :---: | :---: | :---: | :---: | :---: | :---: | :---: | :---: | :---: |
    | **弧度** | $0$ | $\frac{\pi}{6}$ | $\frac{\pi}{4}$ | $\frac{\pi}{3}$ | $\frac{\pi}{2}$ | $\pi$ | $\frac{3\pi}{2}$ | $2\pi$ |

### 26.1.3 扇形的弧长与面积

*   **定理 26.2 (扇形的弧长与面积 / Arc Length and Sector Area):**
    半径为 $r$、圆心角为 $\theta$ rad 的扇形：
    *   弧长：$s = r\theta$
    *   面积：$A = \dfrac{1}{2}r^2\theta = \dfrac{1}{2}rs$

    **证明：** 弧长由 $\theta = s/r$ 直接推出。面积由比例：扇形面积 $/\;\pi r^2 = \theta / 2\pi$，化简得 $A = \frac{1}{2}r^2\theta$。

> **自本章起：** 三角函数的自变量以弧度计。角度制仅在实际计算（如 $\sin 30^\circ = 1/2$）中作为简记使用——理解为 $\sin(\pi/6)$ 的约定俗成写法。

## 26.2 任意角的三角比 (Trigonometric Ratios of Any Angle)

> **学习目标：** 在弧度制的基础上，将 Ch09 中的锐角三角比推广至定义域为 $\mathbb{R}$ 的三角函数。

### 26.2.1 任意角的概念

*   **定义 26.2 (任意角 / General Angle):**
    在平面中，一条射线从初始位置 $OA$（始边 / initial side）绕端点 $O$ 旋转到终止位置 $OB$（终边 / terminal side），所形成的旋转量称为一个**角**。
    *   逆时针旋转 $\implies$ 角取**正值**；顺时针 $\implies$ **负值**。
    *   旋转量为零 $\implies$ **零角**。
    *   角的大小可超过 $2\pi$（多圈旋转）。

*   **定义 26.3 (终边相同的角 / Coterminal Angles):**
    对任意角 $\theta$，所有与 $\theta$ 终边相同的角可表示为：
    $$\theta + 2k\pi \quad (k \in \mathbb{Z})$$

将角的顶点置于原点、始边与 $x$ 轴正半轴重合，称该角处于**标准位置**。本章自此起，所有角均置于标准位置讨论。

### 26.2.2 单位圆定义

Ch09 中的直角三角形定义仅适用于锐角（$0 < \theta < \pi/2$）。为了定义任意角的三角函数，使用单位圆：

*   **定义 26.4 (单位圆定义 / Unit Circle Definition):**
    以原点 $O$ 为心、半径 $1$ 的圆称为**单位圆**。设角 $\theta$ 处于标准位置，其终边与单位圆交于点 $P(x, y)$，则：
    *   $\sin \theta = y$，$\quad \cos \theta = x$，$\quad \tan \theta = \dfrac{y}{x} \;\; (x \neq 0)$

> **一致性验证：** 当 $\theta$ 为锐角时，$P(\cos\theta, \sin\theta)$ 恰为 Ch09 中直角三角形定义的对应点。单位圆定义是锐角定义的自然推广。

### 26.2.3 三角函数

在弧度制下，$\theta$ 是实数。单位圆上每一点 $P(\cos\theta, \sin\theta)$ 对应唯一的 $\theta$（模 $2\pi$），由此定义了从 $\mathbb{R}$ 到 $[-1,1]$ 的函数。

*   **定义 26.5 (三角函数 / Trigonometric Functions):**
    *   $\sin: \mathbb{R} \longrightarrow [-1, 1]$，$x \longmapsto \sin x$
    *   $\cos: \mathbb{R} \longrightarrow [-1, 1]$，$x \longmapsto \cos x$
    *   $\tan: \mathbb{R} \setminus \{\frac{\pi}{2} + k\pi \mid k \in \mathbb{Z}\} \longrightarrow \mathbb{R}$，$x \longmapsto \tan x$

    自变量 $x$ 为弧度（实数）。角度记号（如 $\sin 30^\circ$）约定俗成地理解为 $\sin(\pi/6)$。

*   **定理 26.3 (基本恒等式 / Fundamental Identities):**
    对所有 $x \in \mathbb{R}$（使 $\tan x$ 有定义者）：
    1.  $\sin^2 x + \cos^2 x = 1$
    2.  $\tan x = \dfrac{\sin x}{\cos x}$

    **证明：** (1) $P(\cos x, \sin x)$ 在单位圆上，距原点 $\cos^2 x + \sin^2 x = 1$。(2) 由定义 $\tan x = y/x$。

### 26.2.4 各象限的符号（ASTC）

*   **定理 26.4 (各象限符号 / ASTC Rule):**

    | 象限 | $\cos x$ | $\sin x$ | $\tan x$ | **为正的函数** |
    | :---: | :---: | :---: | :---: | :--- |
    | I（$0 < x < \pi/2$） | $+$ | $+$ | $+$ | **A**ll |
    | II（$\pi/2 < x < \pi$） | $-$ | $+$ | $-$ | **S**ine |
    | III（$\pi < x < 3\pi/2$） | $-$ | $-$ | $+$ | **T**angent |
    | IV（$3\pi/2 < x < 2\pi$） | $+$ | $-$ | $-$ | **C**osine |

### 26.2.5 化锐角公式

*   **定理 26.5 (化锐角公式 / Reduction Formulas):**
    设 $\alpha \in (0, \pi/2)$ 为锐角：

    | 角 | $\sin$ | $\cos$ | $\tan$ |
    | :---: | :---: | :---: | :---: |
    | $-\alpha$ | $-\sin\alpha$ | $\cos\alpha$ | $-\tan\alpha$ |
    | $\pi - \alpha$ | $\sin\alpha$ | $-\cos\alpha$ | $-\tan\alpha$ |
    | $\pi + \alpha$ | $-\sin\alpha$ | $-\cos\alpha$ | $\tan\alpha$ |
    | $2\pi - \alpha$ | $-\sin\alpha$ | $\cos\alpha$ | $-\tan\alpha$ |
    | $\pi/2 - \alpha$ | $\cos\alpha$ | $\sin\alpha$ | $1/\tan\alpha$ |
    | $\pi/2 + \alpha$ | $\cos\alpha$ | $-\sin\alpha$ | $-1/\tan\alpha$ |

    **证明思路：** 通过单位圆上对称点的坐标关系证明。例如 $\pi - \alpha$ 的终边点为 $P(-\cos\alpha, \sin\alpha)$。

### 26.2.6 三角函数的周期性

*   **定理 26.6 (三角函数的周期 / Periodicity):**
    1.  $\sin(x + 2\pi) = \sin x$，$\cos(x + 2\pi) = \cos x$——最小正周期 $T = 2\pi$
    2.  $\tan(x + \pi) = \tan x$——最小正周期 $T = \pi$

    **证明：** 旋转 $2\pi$ 后回到单位圆上同一点；旋转 $\pi$ 后 $P(-x, -y)$，$\tan$ 值不变。

周期性是解三角方程的理论基础。

*   **定理 26.7 (三角方程的通解公式 / General Solutions):**
    1.  若 $\sin x = \sin \alpha$，则 $x = \alpha + 2k\pi$ 或 $x = \pi - \alpha + 2k\pi$（$k \in \mathbb{Z}$）
    2.  若 $\cos x = \cos \alpha$，则 $x = \pm \alpha + 2k\pi$
    3.  若 $\tan x = \tan \alpha$，则 $x = \alpha + k\pi$

    **证明：** 单位圆上 $y = \sin\alpha$ 的点有两个（对应角 $\alpha$ 及 $\pi-\alpha$）；$x = \cos\alpha$ 的点有两个（$\pm\alpha$）；$\tan$ 周期 $\pi$。

*   **方法 26.1 (参考角法 / Reference Angle Method):**
    1.  化方程为 $\sin x = k$（或 $\cos x = k$、$\tan x = k$）。
    2.  求参考角 $\alpha \in [0, \pi/2]$：满足 $\sin\alpha = |k|$（或对应）的锐角。
    3.  由 $k$ 的符号及 ASTC（定理 26.4）确定 $x$ 所在的象限。
    4.  写出各象限解：I: $x = \alpha$；II: $x = \pi - \alpha$；III: $x = \pi + \alpha$；IV: $x = 2\pi - \alpha$。
    5.  限于指定范围（通常 $[0, 2\pi]$）取解。

*   **方法 26.2 (恒等式转换法 / Solving via Identity Substitution):**
    用 $\sin^2 x + \cos^2 x = 1$ 等恒等式将方程统一为同一种三角函数的代数方程，解出三角函数值后再用方法 26.1 求角度。检验增根。

## 26.3 三角函数的图像 (Graphs of Trigonometric Functions)

> **学习目标：** 掌握三角函数的图像特征，通过图像直观理解周期性、对称性与图像变换。

### 26.3.1 正弦、余弦、正切的图像

*   **定理 26.8 (正弦函数的图像性质 / Graph of Sine):**
    函数 $\sin: \mathbb{R} \longrightarrow [-1, 1]$：
    1.  值域 $[-1, 1]$；周期 $2\pi$
    2.  奇函数：$\sin(-x) = -\sin x$（关于原点对称）
    3.  零点：$x = k\pi$；极大值 $1$ 于 $x = \pi/2 + 2k\pi$；极小值 $-1$ 于 $x = 3\pi/2 + 2k\pi$

    | $x$ | $0$ | $\pi/2$ | $\pi$ | $3\pi/2$ | $2\pi$ |
    | :---: | :---: | :---: | :---: | :---: | :---: |
    | $\sin x$ | $0$ | $1$ | $0$ | $-1$ | $0$ |

*   **定理 26.9 (余弦函数的图像性质 / Graph of Cosine):**
    函数 $\cos: \mathbb{R} \longrightarrow [-1, 1]$：
    1.  值域 $[-1, 1]$；周期 $2\pi$
    2.  偶函数：$\cos(-x) = \cos x$（关于 $y$ 轴对称）
    3.  零点：$x = \pi/2 + k\pi$；极大 $1$ 于 $x = 2k\pi$；极小 $-1$ 于 $x = \pi + 2k\pi$

    | $x$ | $0$ | $\pi/2$ | $\pi$ | $3\pi/2$ | $2\pi$ |
    | :---: | :---: | :---: | :---: | :---: | :---: |
    | $\cos x$ | $1$ | $0$ | $-1$ | $0$ | $1$ |

*   **定理 26.10 (正切函数的图像性质 / Graph of Tangent):**
    函数 $\tan: \mathbb{R} \setminus \{\pi/2 + k\pi\} \longrightarrow \mathbb{R}$：
    1.  值域 $\mathbb{R}$；周期 $\pi$
    2.  奇函数；垂直渐近线 $x = \pi/2 + k\pi$
    3.  零点 $x = k\pi$；在每个区间 $(-\pi/2 + k\pi,\; \pi/2 + k\pi)$ 内严格递增

### 26.3.2 周期性在图像上的表现

*   **推论 26.1 (相位关系 / Phase Relationship):**
    $$\cos x = \sin\!\left(x + \frac{\pi}{2}\right), \qquad \sin x = \cos\!\left(x - \frac{\pi}{2}\right)$$
    余弦曲线是正弦曲线向左平移 $\pi/2$ 的结果。

    **证明：** 由定理 26.5 中 $\pi/2 + \alpha$ 行：$\sin(\pi/2 + x) = \cos x$。

*   **定理 26.11 (图像变换 / Graph Transformations):**
    对于 $y = A \sin(Bx + C) + D$（$A, B \neq 0$）：
    *   $|A|$ 为**振幅**（amplitude）
    *   周期 $T = 2\pi/|B|$
    *   相位平移 $-C/B$（水平方向）
    *   垂直平移 $D$

    **证明思路：** $y = A\sin(B(x + C/B)) + D$，依次变换。

> **注：** 余弦变换规律一致。正切 $y = A\tan(Bx+C)+D$ 的周期为 $\pi/|B|$，无振幅概念。

## 26.4 反三角函数 (Inverse Trigonometric Functions)

> **学习目标：** 理解三角函数因周期性非单射，需限制定义域方可定义反函数。掌握 $\arcsin$, $\arccos$, $\arctan$ 的定义域、值域及基本恒等式。

### 26.4.1 定义

$\sin$, $\cos$, $\tan$ 作为 $\mathbb{R}$ 上的函数均**非单射**（周期性导致同一 $y$ 值对应无穷多个 $x$）。为定义反函数，需将定义域限制到适当的**单调区间**。

*   **定义 26.6 (反正弦函数 / Arcsine Function):**
    将 $\sin$ 限制于 $[-\pi/2, \pi/2]$（严格递增区间），其反函数为：
    $$\arcsin: [-1, 1] \longrightarrow \left[-\frac{\pi}{2}, \frac{\pi}{2}\right], \quad x \longmapsto \arcsin x$$
    亦记作 $\sin^{-1} x$。$\arcsin x$ 是**唯一**满足 $\sin(\arcsin x) = x$ 且 $\arcsin x \in [-\pi/2, \pi/2]$ 的实数。
    > **注意：** $\sin^{-1} x \neq \frac{1}{\sin x}$——$-1$ 是反函数记号，非指数。

*   **定义 26.7 (反余弦函数 / Arccosine Function):**
    将 $\cos$ 限制于 $[0, \pi]$（严格递减区间），其反函数为：
    $$\arccos: [-1, 1] \longrightarrow [0, \pi], \quad x \longmapsto \arccos x$$
    亦记作 $\cos^{-1} x$。

*   **定义 26.8 (反正切函数 / Arctangent Function):**
    将 $\tan$ 限制于 $(-\pi/2, \pi/2)$（严格递增区间），其反函数为：
    $$\arctan: \mathbb{R} \longrightarrow \left(-\frac{\pi}{2}, \frac{\pi}{2}\right), \quad x \longmapsto \arctan x$$
    亦记作 $\tan^{-1} x$。

> **与 Ch09 的关系：** Ch09 定义 9.7 中的 $\sin^{-1}$ 等为锐角范围内的角度记号。本章将反三角函数定义为完整的 $\mathbb{R} \to \mathbb{R}$（或其子集）实值函数。

### 26.4.2 反三角恒等式

*   **定理 26.12 (基本互逆关系 / Fundamental Inverse Relations):**
    1.  $\sin(\arcsin x) = x$（$x \in [-1, 1]$）；$\arcsin(\sin x) = x$（$x \in [-\pi/2, \pi/2]$）
    2.  $\cos(\arccos x) = x$（$x \in [-1, 1]$）；$\arccos(\cos x) = x$（$x \in [0, \pi]$）
    3.  $\tan(\arctan x) = x$（$x \in \mathbb{R}$）；$\arctan(\tan x) = x$（$x \in (-\pi/2, \pi/2)$）

    **关键陷阱：** $x$ 超出主值区间时 $\arcsin(\sin x) \neq x$。例如 $\arcsin(\sin(2\pi/3)) = \arcsin(\sqrt{3}/2) = \pi/3$，而非 $2\pi/3$。

*   **推论 26.2 (互补关系 / Complementary Relation):**
    $$\arcsin x + \arccos x = \frac{\pi}{2} \quad (x \in [-1, 1])$$

    **证明：** 令 $\alpha = \arcsin x \in [-\pi/2, \pi/2]$，$\cos(\pi/2 - \alpha) = \sin\alpha = x$ 且 $\pi/2 - \alpha \in [0, \pi]$，故 $\arccos x = \pi/2 - \alpha$。

*   **推论 26.3 (负角关系 / Negative Argument Relations):**
    $$\arcsin(-x) = -\arcsin x, \qquad \arctan(-x) = -\arctan x$$
    $\arcsin$ 与 $\arctan$ 为奇函数。$\arccos(-x) = \pi - \arccos x$。

## Ch26 总结

1.  **弧度法（§26.1）：** $\theta = s/r$（纯数），建立角度与实数的一一对应。$\pi$ rad $= 180^\circ$（定理 26.1）。扇形弧长 $s = r\theta$、面积 $A = \frac{1}{2}r^2\theta$（定理 26.2）。
2.  **三角函数的定义（§26.2.1–26.2.3）：** $\sin: \mathbb{R} \longrightarrow [-1, 1]$，$\cos: \mathbb{R} \longrightarrow [-1, 1]$，$\tan: \mathbb{R}\setminus\{\pi/2 + k\pi\} \longrightarrow \mathbb{R}$（定义 26.5）。自变量为弧度（实数）。单位圆定义与 Ch09 锐角定义一致。
3.  **基本恒等式与符号（§26.2.4–26.2.5）：** $\sin^2 x + \cos^2 x = 1$，$\tan x = \sin x / \cos x$（定理 26.3）。ASTC 法则（定理 26.4）。化锐角公式（定理 26.5）。
4.  **周期性与解方程（§26.2.6）：** $\sin, \cos$ 周期 $2\pi$，$\tan$ 周期 $\pi$（定理 26.6）。通解公式（定理 26.7）；参考角法（方法 26.1）；恒等式转换法（方法 26.2）。
5.  **图像与变换（§26.3）：** $\sin$ 为奇函数，$\cos$ 为偶函数，$\tan$ 有垂直渐近线（定理 26.8–26.10）。相位关系 $\cos x = \sin(x + \pi/2)$（推论 26.1）。$y = A\sin(Bx+C)+D$：振幅 $|A|$，周期 $2\pi/|B|$，相位平移 $-C/B$，垂直平移 $D$（定理 26.11）。
6.  **反三角函数（§26.4）：** $\arcsin: [-1,1] \to [-\pi/2, \pi/2]$，$\arccos: [-1,1] \to [0,\pi]$，$\arctan: \mathbb{R} \to (-\pi/2, \pi/2)$（定义 26.6–26.8）。互逆关系仅在主值区间成立（定理 26.12）。$\arcsin x + \arccos x = \pi/2$（推论 26.2）。$\arcsin$, $\arctan$ 为奇函数（推论 26.3）。

### 备课建议

1.  **弧度制必须先行：** 务必在引入三角函数图像之前建立弧度制。弧度是连接“几何角”与“实变函数”的桥梁。让学生理解 $\theta = s/r$ 给出的是纯数——这才是 $\sin: \mathbb{R} \to \mathbb{R}$ 的数学根基。
2.  **从锐角到任意角的过渡：** 单位圆定义是核心桥梁。从“$\sin(2\pi/3) = ?$”出发引导学生追问第二象限符号，从单位圆的对称性推导 ASTC，而非死记口诀。
3.  **解方程的“解遗漏”陷阱：** 学生最常见错误是 $\sin x = k$ 只写一个解。养成习惯：每次解 $\sin$ 方程后问“ASTC 中哪些象限 $\sin$ 为正/负？”，确保不遗漏。
4.  **图像变换的系统教学：** $y = A\sin(Bx+C) + D$ 建议配合 Desmos/GeoGebra 互动滑杆演示——学生亲手拖动 $A$ 观察振幅，拖动 $B$ 观察周期压缩。
5.  **反三角函数的符号陷阱：** $\sin^{-1} x \neq 1/\sin x$。$\arcsin(\sin(2\pi/3)) = \pi/3 \neq 2\pi/3$ 是理解主值区间的经典例题。
6.  **与 Ch32 的衔接铺垫：** 正弦定理、余弦定理及三角形面积公式将在 Ch32 中系统引入。本章聚焦于三角函数作为“函数”的性质。

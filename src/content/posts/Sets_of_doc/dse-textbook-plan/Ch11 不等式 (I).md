## 11.1 实数的序性质回顾 (Review of Order Properties of Real Numbers)

> 本节回顾 Ch01§1.4 中建立的序公理及相关定理。详见 [Ch01§1.4 比较不同的数大小的方法](/posts/ch01-数系(i)#1.4/)，此处仅列出本章所需的核心结论。

- **公理 1.4.1（序公理）回顾**
  对任意实数 $x, y$：
  1. （三歧性）$x > y$、$x < y$、$x = y$ 恰一成立。
  2. （乘法性质）$x > 0,\; y > 0 \implies xy > 0$。
  3. $x > y \iff y < x$。
  4. （加法性质）$x > y \implies x + z > y + z$。
  5. （传递性）$x > y,\; y > z \implies x > z$。

- **定理 1.4.1 – 1.4.5 回顾**
  | 定理 | 内容 | 本章用途 |
  |------|------|---------|
  | 1.4.1–1.4.2 | $a > b \iff a - b > 0$；$a < b \iff a - b < 0$ | 比较大小的基本方法（作差法） |
  | 1.4.3 | $x > y$ 且 $z > 0 \implies xz > yz$ | 不等式两边乘除以正数，方向不变 |
  | 1.4.4 | $x > y \implies -x < -y$ | 乘以 $-1$ 不等号反转（变号法则） |
  | 1.4.5 | $x > y$ 且 $z < 0 \implies xz < yz$ | ⚠️ 不等式两边乘除以负数，方向反转 |

  将“$>$”“$<$”替换为“$\ge$”“$\le$”，上述定理仍成立。

- **Remark：** 以上公理与定理构成不等式运算的全部逻辑基础。本章的核心任务是在此基础上，将不等式的求解系统化。

## 11.2 一元一次不等式 (Linear Inequalities in One Variable)

### 11.2.1 定义及求解法 (Definition and Solving Method)

> 本节定义一元一次不等式，建立其求解算法，并与 Ch05 中一元一次方程的解法进行系统对比。

- **定义 11.1 (一元一次不等式 / Linear Inequality in One Variable)**
  只含有一个未知数，且未知数的最高次数为 $1$ 的不等式。
  - **标准形式：** $ax + b > 0$ 或 $ax + b < 0$，其中 $a, b \in \mathbb{R}$ 且 $a \neq 0$。
  - 包含 $\ge$ 或 $\le$ 的形式归入同一范畴，求解方法一致。

- **定义 11.2 (不等式的解 / Solution of an Inequality)**
  使不等式成立的未知数的取值，称为该不等式的**解**。
  - 与方程不同——方程的解通常为有限个（甚至唯一），而不等式的解通常有**无穷多个**，在数轴上对应一段连续的区间。

- **方法 11.1 (解一元一次不等式 / Solving a Linear Inequality in One Variable)**

  与 Ch05 中解一元一次方程的步骤高度一致，唯一差异在于**系数化为 1** 时须判断系数正负：

  | 步骤 | 操作 | 与解方程的异同 |
  |------|------|---------------|
  | **1. 去分母** | 两边乘各分母的最小公倍数 | 相同（注意：乘的正数，方向不变） |
  | **2. 去括号** | 利用分配律展开 | 相同 |
  | **3. 移项** | 将含未知数的项移至一边，常数项移至另一边 | 相同（基于公理 1.4.1 加法性质，移项本身不变号） |
  | **4. 合并同类项** | 化为 $ax > b$ 或 $ax < b$ 的形式 | 相同 |
  | **5. 系数化为 1** | 两边同除以 $a$ | ⚠️ **不同**——若 $a>0$ 方向不变（定理 1.4.3），若 $a<0$ 方向改变（定理 1.4.5） |

  **Remark：** 第 5 步是本方法的核心。与方程解法相比，$90\%$ 的步骤完全一致，仅在最后的“除以系数”时需多一个判断：系数为正则方向不变，系数为负则方向反转。

- **定理 11.1 (解的公式 / Formula for the Solution)**
  一元一次不等式 $ax + b > 0$（$a \neq 0$）的解为：
  - 若 $a > 0$，则 $\displaystyle x > -\frac{b}{a}$。
  - 若 $a < 0$，则 $\displaystyle x < -\frac{b}{a}$。

  **证明：** 由公理 1.4.1 加法性质移项得 $ax > -b$。再由定理 1.4.3（若 $a>0$）或定理 1.4.5（若 $a<0$）两边除以 $a$ 即得。

  **Remark：** 定理 11.1 同时说明了为何一元一次不等式的解必为一个**无界区间**（向左或向右无限延伸）。

### 11.2.2 几何表示 (Geometric Representation)

> 将不等式解转化为数轴上的图形，以几何直觉辅助理解解的范围。

- **定义 11.3 (区间 / Interval)**
  设 $a, b$ 为实数且 $a < b$。以不等式或数轴上的一段连续范围来表示解的范围，称为**区间**：
  - **开区间 (Open Interval)：** $(a, b)$ 表示 $a < x < b$
  - **闭区间 (Closed Interval)：** $[a, b]$ 表示 $a \le x \le b$
  - **半开半闭区间 (Half-Open Interval)：** $[a, b)$ 表示 $a \le x < b$；$(a, b]$ 表示 $a < x \le b$
  - **无界区间 (Unbounded Interval)：** $(a, \infty)$ 表示 $x > a$；$(-\infty, a]$ 表示 $x \le a$，余此类推。

- **方法 11.2 (数轴上的图示规则 / Rules for Number Line Representation)**

  1. **方向：** $>$ 或 $\ge$ 向右画箭头 $\rightarrow$；$<$ 或 $\le$ 向左画箭头 $\leftarrow$。
  2. **端点：**
     - **空心圆 (Open Circle, $\circ$)：** 表示**不包含**该端点（对应 $>$ 或 $<$，即严格不等式）。
     - **实心点 (Solid Dot, $\bullet$)：** 表示**包含**该端点（对应 $\ge$ 或 $\le$）。

- **图示对照表 (Summary of Representations)**

  | 不等式 | 区间记号 | 端点类型 | 数轴方向 |
  |:------:|:----------------:|:--------:|:--------:|
  | $x > a$ | $(a, \infty)$ | 空心 $\circ$ | 向右 $\rightarrow$ |
  | $x \ge a$ | $[a, \infty)$ | 实心 $\bullet$ | 向右 $\rightarrow$ |
  | $x < a$ | $(-\infty, a)$ | 空心 $\circ$ | 向左 $\leftarrow$ |
  | $x \le a$ | $(-\infty, a]$ | 实心 $\bullet$ | 向左 $\leftarrow$ |
  | $a < x < b$ | $(a, b)$ | 两端空心 $\circ$ | — |
  | $a \le x < b$ | $[a, b)$ | 左实心 $\bullet$，右空心 $\circ$ | — |
  | $a < x \le b$ | $(a, b]$ | 左空心 $\circ$，右实心 $\bullet$ | — |
  | $a \le x \le b$ | $[a, b]$ | 两端实心 $\bullet$ | — |

  **Remark：** 区间记号中的括号与端点类型对应——圆括号 $($ 或 $)$ 对应空心圆（不包含），方括号 $[$ 或 $]$ 对应实心点（包含）。$\infty$ 端永远使用圆括号（因为无穷大并非实数，无法被“包含”）。

## 第十一章总结

1. **序性质回顾（Ch01§1.4）：** 三歧性与传递性构成 $\mathbb{R}$ 上的全序结构。加法保持序（公理 1.4.1-4）、正数乘法保持序（定理 1.4.3）、负数乘法逆转序（定理 1.4.5）——三条性质是不等式运算的全部逻辑基础。
2. **变号法则（定理 1.4.4）：** $x > y \implies -x < -y$。乘以 $-1$（或移项到对面）不等号反转，初学者最频繁错误即遗漏此步。
3. **一元一次不等式（定义 11.1）：** 形如 $ax + b > 0$（$a \neq 0$），其求解步骤（方法 11.1）与解一元一次方程 $90\%$ 一致，仅在“系数化为 1”时须根据 $a$ 的正负决定是否变号。
4. **解的公式（定理 11.1）：** $a > 0$ 时解为 $x > -b/a$；$a < 0$ 时解为 $x < -b/a$。一元一次不等式的解必为**无界区间**。
5. **几何表示（定义 11.3、方法 11.2）：** 不等式解以数轴图示——空心圆 $\circ$ 对应严格不等式（$>$ 或 $<$），实心点 $\bullet$ 对应非严格不等式（$\ge$ 或 $\le$）。区间记号中圆括号对应开端点、方括号对应闭端点。

### 备课建议

1. **解方程 → 解不等式的类比教学法：**
   建议先列出一个学生已熟练的方程（例如 $3x - 5 = 7$），再并列一个相似的不等式（$3x - 5 > 7$），对比求解过程。$90\%$ 的步骤完全一致，学生仅需在最后一步多一个“判断系数正负”的意识。这种类比法能大幅降低陌生感。

2. **“忘记变号”是初学者最大陷阱：**
   建议专门设计一组“陷阱题”——系数为负的一元一次不等式，例如 $-x > 5$、$3 < -2x$、$5 - 3x \le 11$ 等。反复训练学生在“系数化为 1”前先观察系数符号的习惯。可以归纳口诀：“除负变号，除正不变”。

3. **移项的本质理解：**
   学生常机械记忆“移项要变号”，但对不等式而言，移项 $ax + b > 0 \implies ax > -b$ 实质上是两边同时加上 $-b$（公理 1.4.1 加法性质），不等号不变。应强调：移项本身**不引起变号**；仅在乘除负数时才变号。避免学生将“移项”与“变号”混淆。

4. **文字 ↔ 符号的转化训练：**
   DSE 常见考法是以文字描述代替不等式符号，例如：
   - “$x$ 不小于 $3$”$\implies x \ge 3$
   - “$y$ 是非负数”$\implies y \ge 0$
   - “$z$ 是正数”$\implies z > 0$
   - “$t$ 最多为 $10$”$\implies t \le 10$
   建议在课堂中系统性地练习这类转化，培养数学阅读能力。

5. **数轴图示的端点易错提醒：**
   学生容易在 $\ge$ / $\le$ 对应实心点、$>$ / $<$ 对应空心圆的规则上出错。建议设计“给不等式→画数轴”和“看数轴→写不等式”的双向练习。另外，区间记号 $(a,b)$ vs $[a,b]$ 的括号含义与端点类型的对应关系，应在教学中反复强化。

6. **$\infty$ 的使用规范：**
   提醒学生——$\infty$ 不是实数，因此在区间记号中永远使用圆括号（例如 $(a, \infty)$ 而非 $[a, \infty]$）。同样，在数轴上箭头 $\rightarrow$ 表示“无限延伸”，不可在箭头末端画端点记号。

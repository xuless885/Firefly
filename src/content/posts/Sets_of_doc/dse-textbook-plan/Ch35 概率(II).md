## §35.1 概率的集合论基础

回顾 Ch34 定义 34.1 与定义 34.2：一个**试验**的**样本空间** $\Omega$ 为该试验所有可能结果之集合；$\Omega$ 的任意子集 $E \subseteq \Omega$ 称为一个**事件**。本章在此基础上，将“概率”形式化为定义在集合上的一个函数。

### 35.1.1 概率函数与概率空间

- **定义 35.1（概率函数）**
  设 $\Omega$ 为有限样本空间。一个**概率函数**（Probability Function）$P$ 是一个从 $\Omega$ 的幂集到闭区间 $[0, 1]$ 的映射：
  $$P : \mathcal{P}(\Omega) \to [0, 1]$$
  给定每个事件 $A \subseteq \Omega$，$P(A)$ 称为事件 $A$ 的**概率**。

换言之，概率函数的输入是一个**集合**（事件），输出是一个 $0$ 到 $1$ 之间的实数。这是本章的核心观点：概率不是定义在单个结果 $\omega \in \Omega$ 上的，而是定义在事件的集合 $\mathcal{P}(\Omega)$ 上的。

- **定义 35.2（概率公理，Kolmogorov Axioms）**
  设 $\Omega$ 为有限样本空间。一个映射 $P : \mathcal{P}(\Omega) \to [0, 1]$ 称为概率函数，当且仅当满足以下三条**概率公理**：
  - **公理 1（非负性）：** 对任意事件 $A \subseteq \Omega$，$P(A) \ge 0$。
  - **公理 2（规范性）：** 必然事件的概率为 $1$，$P(\Omega) = 1$。
  - **公理 3（可加性）：** 若 $A, B \subseteq \Omega$ 为**互斥**事件（即 $A \cap B = \varnothing$），则 $P(A \cup B) = P(A) + P(B)$。

- **定义 35.3（概率空间）**
  三元组 $(\Omega, \mathcal{P}(\Omega), P)$ 完整地刻画了一个随机试验的概率结构，称为该试验的**概率空间**（Probability Space）。

*注：* 对于有限样本空间，幂集 $\mathcal{P}(\Omega)$ 是概率函数 $P$ 的自然定义域——$\Omega$ 的每个子集都可以被赋予概率。在更一般的理论（处理无限样本空间时）中，定义域需替换为 $\Omega$ 上的一个 **$\sigma$-代数**（$\sigma$-algebra），以确保并非所有子集都需被赋予概率（见 §35.6 延伸阅读）。

- **定理 35.4（概率函数的基本性质）**
  设 $P$ 为概率空间 $(\Omega, \mathcal{P}(\Omega), P)$ 上的概率函数。则：
  1. $P(\varnothing) = 0$（不可能事件的概率为 $0$）。
  2. 对任意事件 $A$，$P(A^c) = 1 - P(A)$。
  3. 若 $A \subseteq B$，则 $P(A) \le P(B)$（单调性）。
  4. 对任意事件 $A$，$0 \le P(A) \le 1$。

*证明：*

1. 取 $A = \Omega$，$B = \varnothing$。因 $\Omega \cap \varnothing = \varnothing$（互斥），由公理 3：$P(\Omega \cup \varnothing) = P(\Omega) + P(\varnothing)$。但 $\Omega \cup \varnothing = \Omega$，故 $P(\Omega) = P(\Omega) + P(\varnothing)$。由公理 2，$P(\Omega) = 1$，代入得 $1 = 1 + P(\varnothing)$，$\implies P(\varnothing) = 0$。

2. $A$ 与 $A^c$ 互斥（$A \cap A^c = \varnothing$），且 $A \cup A^c = \Omega$。由公理 3：$P(A \cup A^c) = P(A) + P(A^c)$。但 $P(A \cup A^c) = P(\Omega) = 1$，故 $P(A) + P(A^c) = 1$，移项即得。

3. 因 $A \subseteq B$，可将 $B$ 分解为两个互斥部分：$B = A \cup (B \setminus A)$，且 $A \cap (B \setminus A) = \varnothing$。由公理 3：$P(B) = P(A) + P(B \setminus A)$。由公理 1，$P(B \setminus A) \ge 0$，故 $P(B) = P(A) + P(B \setminus A) \ge P(A)$。

4. 由公理 1，$P(A) \ge 0$。又 $A \subseteq \Omega$，由 (3) 得 $P(A) \le P(\Omega) = 1$。$\square$

### 35.1.2 古典概型

- **定义 35.5（古典概型）**
  设 $\Omega$ 为有限样本空间。若 $\Omega$ 中的所有基本结果**等可能**——即每个 $\omega \in \Omega$ 满足
  $$P(\{\omega\}) = \frac{1}{|\Omega|}$$
  则称该概率空间为**古典概型**（Classical Probability Model）。

- **定理 35.6（古典概型的计算公式）**
  在古典概型下，对任意事件 $A \subseteq \Omega$，
  $$P(A) = \frac{|A|}{|\Omega|} = \frac{\text{事件 } A \text{ 包含的结果数}}{\text{样本空间的结果总数}}$$

*证明：* 将 $A$ 中的所有基本事件 $\{\omega\}$ 两两并起来。因任意两个不同的基本事件互斥（$\{\omega_1\} \cap \{\omega_2\} = \varnothing$），由公理 3（可加性）的多次应用：
$$P(A) = P\!\left(\bigcup_{\omega \in A} \{\omega\}\right) = \sum_{\omega \in A} P(\{\omega\}) = \sum_{\omega \in A} \frac{1}{|\Omega|} = \frac{|A|}{|\Omega|}$$
其中第三步使用了等可能性定义 $P(\{\omega\}) = 1/|\Omega|$。$\square$

- **注 35.7**
  古典概型的核心前提是**等可能性**。这是学生最易犯错之处：将不等可能的结果误认为等可能。例如：“明天要么下雨、要么不下雨，所以下雨概率是 $1/2$”——错误，因为两种结果并非等可能。务必在每次使用古典概型前确认等可能性是否成立。

## §35.2 概率的加法定律

### 35.2.1 互斥事件

- **定义 35.8（互斥事件）**
  两个事件 $A$ 与 $B$ 称为**互斥**（Mutually Exclusive），若 $A \cap B = \varnothing$。直观而言，互斥事件不可能同时发生。

- **定理 35.9（互斥事件的加法定律）**
  若 $A$ 与 $B$ 互斥，则
  $$P(A \cup B) = P(A) + P(B)$$

这直接来自公理 3（可加性），不需另行证明。

- **推论 35.10（有限多个互斥事件的加法定律）**
  若 $A_1, A_2, \dots, A_k$ 两两互斥（即 $\forall i \neq j,\; A_i \cap A_j = \varnothing$），则
  $$P(A_1 \cup A_2 \cup \dots \cup A_k) = P(A_1) + P(A_2) + \dots + P(A_k)$$

*证明：* 对 $k$ 使用数学归纳法即可。$\square$

### 35.2.2 互补事件

- **定义 35.11（互补事件 / 对立事件）**
  事件 $A$ 的**互补事件**（Complementary Event），记作 $A^c$（或 $\overline{A}$），定义为
  $$A^c = \Omega \setminus A$$
  即“$A$ 不发生”这一事件。

- **定理 35.12（互补事件的概率）**
  $$P(A^c) = 1 - P(A)$$

已在定理 35.4(2) 中证明。

- **注 35.13（“至少一个”问题的策略）**
  遇到求“至少有一个……”的概率时，通常计算其互补事件“一个都没有”的概率更为简便：
  $$P(\text{至少一个}) = 1 - P(\text{一个都没有})$$

### 35.2.3 一般加法定律

- **定理 35.14（一般加法定律，General Addition Law）**
  对任意两个事件 $A$ 与 $B$（不一定互斥），
  $$P(A \cup B) = P(A) + P(B) - P(A \cap B)$$

*证明：* 将 $A \cup B$ 分解为三个两两互斥的部分：
$$A \cup B = (A \setminus B) \;\cup\; (A \cap B) \;\cup\; (B \setminus A)$$

由推论 35.10（有限多个互斥事件的加法定律）：
$$P(A \cup B) = P(A \setminus B) + P(A \cap B) + P(B \setminus A)$$

注意 $P(A) = P(A \setminus B) + P(A \cap B)$（因 $A = (A \setminus B) \cup (A \cap B)$ 且两部分互斥），同理 $P(B) = P(B \setminus A) + P(A \cap B)$。代入：
$$\begin{aligned}
P(A \cup B) &= [P(A) - P(A \cap B)] + P(A \cap B) + [P(B) - P(A \cap B)] \\
            &= P(A) + P(B) - P(A \cap B)
\end{aligned}$$

$\square$

- **注 35.15**
  当 $A$ 与 $B$ 互斥时，$A \cap B = \varnothing$ 且 $P(A \cap B) = 0$，一般加法定律退化为定理 35.9（互斥事件的加法定律）。

- **推论 35.16（三事件的一般加法定律）**
  $$P(A \cup B \cup C) = P(A) + P(B) + P(C) - P(A \cap B) - P(B \cap C) - P(C \cap A) + P(A \cap B \cap C)$$

*证明思路：* 令 $D = A \cup B$，对 $D$ 与 $C$ 应用定理 35.14，再对 $A$ 与 $B$ 应用定理 35.14 展开 $P(D)$。亦可利用文氏图（Venn Diagram）分解为七个互斥的原子区域验证。$\square$

## §35.3 条件概率

### 35.3.1 条件概率的定义

- **定义 35.17（条件概率）**
  设 $A, B$ 为事件，且 $P(B) > 0$。在事件 $B$ **已经发生**的条件下，事件 $A$ 发生的概率，称为**条件概率**（Conditional Probability），记作 $P(A \mid B)$，定义为
  $$P(A \mid B) = \frac{P(A \cap B)}{P(B)}$$

*直观理解：* 条件概率 $P(A \mid B)$ 是将样本空间从 $\Omega$“缩减”到 $B$ 之后，$A$ 在其中所占的比例。

- **定理 35.18（条件概率是一个概率函数）**
  固定 $B \subseteq \Omega$（$P(B) > 0$），定义映射 $Q : \mathcal{P}(\Omega) \to [0, 1]$ 为 $Q(A) = P(A \mid B)$。则 $Q$ 满足概率的三条公理，即 $(\Omega, \mathcal{P}(\Omega), Q)$ 也是一个概率空间。

*证明：*

- **公理 1（非负性）：** 因 $P(A \cap B) \ge 0$ 且 $P(B) > 0$，故 $Q(A) = P(A \cap B)/P(B) \ge 0$。
- **公理 2（规范性）：** $Q(\Omega) = P(\Omega \cap B)/P(B) = P(B)/P(B) = 1$。
- **公理 3（可加性）：** 设 $A_1, A_2$ 互斥（$A_1 \cap A_2 = \varnothing$），则 $(A_1 \cap B) \cap (A_2 \cap B) = \varnothing$。由 $P$ 的可加性：
  $$\begin{aligned}
  Q(A_1 \cup A_2) &= \frac{P((A_1 \cup A_2) \cap B)}{P(B)} = \frac{P((A_1 \cap B) \cup (A_2 \cap B))}{P(B)} \\
  &= \frac{P(A_1 \cap B) + P(A_2 \cap B)}{P(B)} = Q(A_1) + Q(A_2)
  \end{aligned}$$

$\square$

### 35.3.2 乘法定律

- **定理 35.19（乘法定律，Multiplication Law）**
  设 $A, B$ 为事件，且 $P(B) > 0$。则
  $$P(A \cap B) = P(B) \cdot P(A \mid B)$$
  同理，若 $P(A) > 0$，则 $P(A \cap B) = P(A) \cdot P(B \mid A)$。

*证明：* 由定义 35.17 直接移项即得。$\square$

- **注 35.20（乘法定律的推广）**
  对 $n$ 个事件 $A_1, A_2, \dots, A_n$，若 $P(A_1 \cap A_2 \cap \dots \cap A_{n-1}) > 0$，则
  $$\begin{aligned}
  P(A_1 \cap A_2 \cap \dots \cap A_n) = P(A_1) &\cdot P(A_2 \mid A_1) \cdot P(A_3 \mid A_1 \cap A_2) \;\cdots \\
  &\cdot P(A_n \mid A_1 \cap A_2 \cap \dots \cap A_{n-1})
  \end{aligned}$$

*直观理解：* 这与树形图（Tree Diagram）的运算逻辑一致——沿着分支逐步乘以条件概率，最终到达末端节点即为该路径的概率。

## §35.4 独立事件

### 35.4.1 独立的定义

- **定义 35.21（独立事件）**
  两个事件 $A$ 与 $B$ 称为**相互独立**（Independent），若
  $$P(A \cap B) = P(A) \cdot P(B)$$

- **定理 35.22（独立的等价刻划）**
  设 $P(B) > 0$。则 $A$ 与 $B$ 独立 $\iff$ $P(A \mid B) = P(A)$。
  同理，设 $P(A) > 0$，则 $A$ 与 $B$ 独立 $\iff$ $P(B \mid A) = P(B)$。

*证明：*

($\implies$) 若 $A, B$ 独立，则 $P(A \cap B) = P(A) P(B)$。代入条件概率定义：
$$P(A \mid B) = \frac{P(A \cap B)}{P(B)} = \frac{P(A) P(B)}{P(B)} = P(A)$$

($\Longleftarrow$) 若 $P(A \mid B) = P(A)$，由乘法定律：
$$P(A \cap B) = P(B) \cdot P(A \mid B) = P(B) \cdot P(A) = P(A) P(B)$$

故 $A, B$ 独立。$\square$

*直观理解：* 定理 35.22 揭示了“独立”的实质含义——已知 $B$ 发生，并不改变 $A$ 的概率。即 $B$ 的发生对 $A$“没有影响”。

- **定理 35.23（补集的独立性）**
  若 $A$ 与 $B$ 独立，则 $A^c$ 与 $B$ 也独立；$A$ 与 $B^c$ 也独立；$A^c$ 与 $B^c$ 也独立。

*证明（$A^c$ 与 $B$ 独立）：*
$$\begin{aligned}
P(A^c \cap B) &= P(B \setminus A) = P(B) - P(A \cap B) \\
             &= P(B) - P(A) P(B) \quad (\because A, B \text{ 独立}) \\
             &= P(B) \cdot (1 - P(A)) = P(B) \cdot P(A^c)
\end{aligned}$$

其余情形类似可证。$\square$

### 35.4.2 独立与互斥的关系

- **定理 35.24（独立与互斥的辨析）**
  设 $P(A) > 0$ 且 $P(B) > 0$。则 $A$ 与 $B$ 不可能既互斥又独立。

*证明：* 若 $A$ 与 $B$ 互斥，则 $A \cap B = \varnothing$，故 $P(A \cap B) = 0$。若 $A$ 与 $B$ 独立，则 $P(A \cap B) = P(A) P(B) > 0$（因 $P(A), P(B) > 0$）。两者矛盾，故不可能同时成立。$\square$

- **注 35.25（概念辨析）**

| | 互斥 (Mutually Exclusive) | 独立 (Independent) |
|:---|:---|:---|
| **含义** | $A$ 发生 $\implies$ $B$ **绝不**发生 | $A$ 发生与否**不影响** $B$ 的概率 |
| **集合条件** | $A \cap B = \varnothing$ | $P(A \cap B) = P(A) P(B)$ |
| **关系** | 若 $P(A), P(B) > 0$，互斥 $\implies$ **不**独立 | 若 $P(A), P(B) > 0$，独立 $\implies$ **不**互斥 |

*比喻：*

- **互斥**如“生与死”——你活着就不能死，死了就不能活。
- **独立**如“我吃饭”与“下雨”——吃不吃饭跟下不下雨没有关系。

## §35.5 涉及排列组合的概率

回顾 Ch34 的排列数 $P_n^r$ 与组合数 $C^n_r$。在古典概型下，事件的概率计算往往归结为计数问题——分别计算样本空间的基数 $|\Omega|$ 与目标事件的基数 $|A|$。

### 35.5.1 一般方法论

- **方法 35.26（排列组合概率的解题框架）**
  1. **确认古典概型条件：** 样本空间有限，且所有基本结果等可能。
  2. **确定抽样方式：** 有序/无序？有放回/无放回？这决定分母用 $P_n^r$ 还是 $C^n_r$。
  3. **计算 $|\Omega|$（分母）：** 使用 Ch34 的乘法原理、排列或组合公式。
  4. **计算 $|A|$（分子）：** 使用同样的计数框架，但施加事件 $A$ 的额外约束。
  5. **代入 $P(A) = |A|/|\Omega|$。**

- **注 35.27（分母与分子的一致性）**
  分母与分子必须使用**相同的计数框架**（同为有序或同为无序）。若分母按有序计数（如 $P_n^r$），分子也必须按有序计数；若分母按无序计数（如 $C^n_r$），分子也必须按无序计数。这是排列组合概率中最常见的错误来源。

### 35.5.2 典型问题结构

以下以“从 $n$ 个物件中抽取 $r$ 个”为基本模型，整理常见的问题类型：

| 问题类型 | $\|\Omega\|$（分母） | $\|A\|$（分子）结构 |
|:---|:---|:---|
| 无放回、不计顺序（组合） | $C^n_r$ | $C^{n_1}_{r_1} \cdot C^{n_2}_{r_2} \cdots$（按子类划分） |
| 无放回、计顺序（排列） | $P_n^r$ | 按位置逐步施加约束 |
| 有放回、计顺序 | $n^r$（乘法原理） | 按位置逐步施加约束 |

*例：* 袋中有 $3$ 红球、$2$ 白球。随机抽取 $2$ 球（无放回、不计顺序）。

- $|\Omega| = C^5_2 = 10$
- $A =$“恰好一红一白”：从 $3$ 红中选 $1$ 个（$C^3_1$），从 $2$ 白中选 $1$ 个（$C^2_1$）。由乘法原理，$|A| = C^3_1 \cdot C^2_1 = 6$。
- $P(A) = 6/10 = 3/5$。

## §35.6 延伸内容：几何概率

### 35.6.1 基本思想

古典概型要求样本空间有限且等可能。但在许多实际问题中，样本空间是无限的（例如，在一个区间中随机取一点）。此时古典概型的计数方法不再适用，我们需要用“度量”（长度、面积或体积）取代“计数”。

- **定义 35.28（几何概率）**
  设样本空间 $\Omega$ 为某几何区域，具有有限的非零度量（长度、面积或体积），记为 $\mu(\Omega)$。若每个结果出现的可能性与其在 $\Omega$ 中的位置无关，且仅与其“所占的度量”成正比，则对任意事件 $A \subseteq \Omega$，
  $$P(A) = \frac{\mu(A)}{\mu(\Omega)}$$
  其中 $\mu(A)$ 为 $A$ 的度量（长度、面积或体积，视情境而定）。

- **注 35.29**
  几何概率的等可能性体现为“等度量 $\implies$ 等概率”——即 $\Omega$ 中两个度量相等的区域，其对应的事件概率相等。这与古典概型中“等基数 $\implies$ 等概率”的逻辑完全平行。

### 35.6.2 经典模型

**一维模型（长度比）：**

- *例：* 一班巴士在 $8{:}00$ 至 $8{:}15$ 之间随机到达某站。若小明在 $8{:}00$ 准时抵达，求他等候不超过 $5$ 分钟的概率。
- $\Omega$ 为时间区间 $[0, 15]$（分钟），$\mu(\Omega) = 15$。
- 事件 $A =$“等候 $\le 5$ 分钟”$\iff$ 巴士在 $[0, 5]$ 内到达。
- $P(A) = \mu(A)/\mu(\Omega) = 5/15 = 1/3$。

**二维模型（面积比）：**

- *例（相遇问题）：* 甲、乙两人约定在 $12{:}00$ 至 $13{:}00$ 之间到达某地，先到者等待 $15$ 分钟后离开。求两人相遇的概率。
- 设甲到达时刻为 $x$，乙到达时刻为 $y$（均以 $12{:}00$ 起计的分钟数表示，$x, y \in [0, 60]$）。
- $\Omega = [0, 60] \times [0, 60]$，$\mu(\Omega) = 60^2 = 3600$（面积）。
- 两人相遇的条件为 $|x - y| \le 15$，即 $y \in [x-15, x+15]$ 且 $y \in [0, 60]$。
- $A = \{(x, y) \in \Omega \mid |x - y| \le 15\}$，为正方形中夹在 $y = x \pm 15$ 两条线之间的带状区域。
- $\mu(A) = 60^2 - 2 \times \frac{1}{2} \times 45 \times 45 = 3600 - 2025 = 1575$。
- $P(A) = 1575/3600 = 7/16$。

### 延伸阅读：$\sigma$-代数初步

对于**无限样本空间**（如 $[0, 1]$ 区间中随机取一点），并非 $\Omega$ 的**每一个**子集都能被赋予一个合理的概率（这涉及测度论中的不可测集问题）。为此，现代概率论将概率函数 $P$ 的定义域从 $\mathcal{P}(\Omega)$ 替换为 $\Omega$ 上的一个 **$\sigma$-代数** $\mathcal{F} \subseteq \mathcal{P}(\Omega)$，要求 $\mathcal{F}$ 满足三条封闭性条件：

1. $\Omega \in \mathcal{F}$
2. 若 $A \in \mathcal{F}$，则 $A^c \in \mathcal{F}$（对补集封闭）
3. 若 $A_1, A_2, \dots \in \mathcal{F}$，则 $\bigcup_{n=1}^\infty A_n \in \mathcal{F}$（对可列并封闭）

于是概率空间的完整定义为三元组 $(\Omega, \mathcal{F}, P)$。DSE 范围内样本空间均为有限，此时取 $\mathcal{F} = \mathcal{P}(\Omega)$ 即足够。

## 第三十五章总结

1. **概率函数** $P : \mathcal{P}(\Omega) \to [0, 1]$ 是一个从幂集到 $[0, 1]$ 的映射，满足非负性、规范性与可加性三条公理（定义 35.1–35.2）。
2. 概率的基本性质：$P(\varnothing) = 0$、$P(A^c) = 1 - P(A)$、单调性、有界性（定理 35.4）。
3. **古典概型**要求样本空间有限且各基本结果等可能，此时 $P(A) = |A|/|\Omega|$（定理 35.6）。
4. **加法定律：** 互斥事件 $P(A \cup B) = P(A) + P(B)$（定理 35.9）；一般情形 $P(A \cup B) = P(A) + P(B) - P(A \cap B)$（定理 35.14）。
5. **互补事件：** $P(A^c) = 1 - P(A)$。“至少一个”问题优先使用 $1 - P(\text{无})$ 策略（注 35.13）。
6. **条件概率** $P(A \mid B) = P(A \cap B)/P(B)$ 是将样本空间缩减到 $B$ 后重新计算 $A$ 的比例（定义 35.17）。条件概率本身也满足三条概率公理（定理 35.18）。
7. **乘法定律：** $P(A \cap B) = P(B) \cdot P(A \mid B)$（定理 35.19），可推广至 $n$ 个事件的链式乘法（注 35.20）。
8. **独立事件**定义为 $P(A \cap B) = P(A) P(B)$（定义 35.21），等价于 $P(A \mid B) = P(A)$（定理 35.22）。若 $P(A), P(B) > 0$，互斥与独立不可兼得（定理 35.24）。
9. **排列组合概率**需确保分母与分子使用相同的计数框架（有序/无序一致）（方法 35.26，注 35.27）。
10. **几何概率**将古典概型推广至无限样本空间，以长度/面积/体积比取代计数比：$P(A) = \mu(A)/\mu(\Omega)$（定义 35.28）。

### 备课建议

1. **“概率是集合函数”这一核心观点：** 本章从 Ch16 的集合论出发，将概率定义为以事件（集合）为输入、以 $[0,1]$ 中实数为输出的函数。这一观点是贯穿全章的红线——从加法公式的容斥原理证明，到条件概率定义中“缩减样本空间”的直观，都源于此。

2. **加法定律的容斥原理证明：** 定理 35.14 的证明（将 $A \cup B$ 分解为三个互斥部分）直接对应 Ch34 的容斥原理（定理 34.25）及 Ch16 的集合运算。应向学生指出两者的结构同一性——概率加法公式就是容斥原理在概率测度下的体现。

3. **“互斥 vs 独立”的辨析是核心难点：** 定理 35.24 和注 35.25 应重点讲解。许多学生长久混淆两者，需反复训练：
   - 互斥是**集合关系**（$A \cap B = \varnothing$），与概率数值无关。
   - 独立是**概率数值关系**（$P(A \cap B) = P(A) P(B)$），与集合是否相交无关。

4. **条件概率是 DSE 考试重点：** 定义 35.17 和定理 35.19（乘法定律）是高频考点。建议结合树形图进行教学——树形图中每个分支的权重即为条件概率，末端节点的概率即为各分支概率之积（乘法定律）。

5. **排列组合概率的“一致性”原则：** 注 35.27（分母与分子必须使用相同的计数框架）是此类题型中最常见的错误来源。建议在例题中同时展示“有序计数”和“无序计数”两种等价路径，让学生体会两者给出相同结果，从而建立信心。

6. **几何概率的相遇问题：** 经典的“两人约会”问题（§35.6.2）是将代数不等式 $|x-y| \le 15$ 转化为平面几何区域（面积比）的典范。适合在课堂上作为跨课题（不等式 + 坐标几何 + 概率）的综合例题讲解。

7. **与后续章节的衔接：** 本章的条件概率为 M1 的贝叶斯定理（Bayes' Theorem）和离散概率分布打下基础；排列组合概率则直接连接到 M1 的二项分布。

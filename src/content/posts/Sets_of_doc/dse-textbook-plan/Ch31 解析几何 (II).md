## 31.1 圆的方程

### 31.1.1 圆的定义与标准方程

- **定义 31.1（圆 / Circle）**
  平面内到一个定点（圆心）的距离等于一个定长（半径）的所有点所构成的轨迹。

  若圆心为 $C(h, k)$，半径为 $r > 0$，则轨迹条件 $PC = r$ 写成坐标形式即得**圆的标准方程**：
  $$(x - h)^2 + (y - k)^2 = r^2$$

  特别地，当圆心在原点 $(0, 0)$ 时，方程简化为：
  $$x^2 + y^2 = r^2$$

- **Remark：** 圆是五种基本轨迹之一（参见第 30 章定理 30.1），即与定点 $C$ 距离为定长 $r$ 的动点轨迹：$\mathcal{L} = \{P : PC = r\}$。标准方程就是将此几何条件写成代数形式。

### 31.1.2 圆的一般方程

- **定义 31.2（圆的一般方程 / General Equation of a Circle）**
  将标准方程展开并移项，可得形如
  $$x^2 + y^2 + Dx + Ey + F = 0$$
  的方程，称为圆的**一般方程**，其中 $D, E, F \in \mathbb{R}$。

  并非所有此形式的方程都表示圆。需通过配方判定。

- **定理 31.1（一般方程表示圆的条件）**
  对于方程 $x^2 + y^2 + Dx + Ey + F = 0$，令 $\Delta = D^2 + E^2 - 4F$，则：
  $$\begin{aligned}
  \Delta > 0 &\;\Longrightarrow\; \text{图形为一个圆；} \\
  \Delta = 0 &\;\Longrightarrow\; \text{图形为一个点 }\left(-\frac{D}{2}, -\frac{E}{2}\right)\text{；} \\
  \Delta < 0 &\;\Longrightarrow\; \text{无实数解，无图形。}
  \end{aligned}$$

  **证明：** 配方：
  $$\begin{aligned}
  x^2 + y^2 + Dx + Ey + F &= 0 \\[2pt]
  \left(x^2 + Dx + \frac{D^2}{4}\right) + \left(y^2 + Ey + \frac{E^2}{4}\right) &= \frac{D^2}{4} + \frac{E^2}{4} - F \\[2pt]
  \left(x + \frac{D}{2}\right)^2 + \left(y + \frac{E}{2}\right)^2 &= \frac{D^2 + E^2 - 4F}{4}
  \end{aligned}$$
  令 $R^2 = \dfrac{\Delta}{4}$。由 $R^2 > 0 \iff \Delta > 0$ 即得结论。$\square$

- **定理 31.2（一般方程化为标准式）**
  当 $D^2 + E^2 - 4F > 0$ 时，圆 $x^2 + y^2 + Dx + Ey + F = 0$ 的参数为：
  $$\begin{aligned}
  \text{圆心：}&\quad \left(-\frac{D}{2},\; -\frac{E}{2}\right) \\[6pt]
  \text{半径：}&\quad r = \frac{1}{2}\sqrt{D^2 + E^2 - 4F}
  \end{aligned}$$

  **证明：** 由定理 31.1 证明过程中的配方结果直接读出。$\square$

- **推论 31.1（一般方程的代数特征）**
  圆的一般方程 $x^2 + y^2 + Dx + Ey + F = 0$ 具有以下两个必要（但不充分）的代数特征：
  1. $x^2$ 项与 $y^2$ 项的系数相等（均为 $1$，或可化为相等）；
  2. 不含 $xy$ 项。

  **证明：** 由标准方程展开即知——$(x-h)^2+(y-k)^2=r^2$ 产生 $x^2+y^2$ 而无 $xy$。$\square$

### 31.1.3 涉及圆的一些问题

本节讨论由不同已知条件确定圆的方程的常见方法。

- **方法 31.1（已知三点求圆方程）**
  设不共线的三点 $A(x_1, y_1)$、$B(x_2, y_2)$、$C(x_3, y_3)$，则存在唯一的圆经过这三点。求法有两种：

  **途径一（待定系数法）：** 设一般方程 $x^2 + y^2 + Dx + Ey + F = 0$。将三点坐标代入，得关于 $D, E, F$ 的三元一次方程组，解之即得。

  **途径二（几何法）：** 分别求 $AB$ 和 $BC$ 的垂直平分线，其交点即为圆心（圆心到 $A, B, C$ 等距）；半径 $r = CA$。代入标准式即得。

  两个途径的原理：途径一利用一般方程有三个未知数、需要三个条件（三点）来确定；途径二利用“圆心到圆上任一点等距”，故圆心在任两点连线的垂直平分线上。

- **方法 31.2（以直径端点确定的圆）**
  以 $A(x_1, y_1)$ 和 $B(x_2, y_2)$ 为直径两端点的圆方程为：
  $$(x - x_1)(x - x_2) + (y - y_1)(y - y_2) = 0$$

  **推导：** 设 $P(x, y)$ 为圆上任意点（$P \neq A, B$）。由“直径所对的圆周角为直角”（定理 29.7），有 $PA \perp PB$。由垂直线斜率条件（定理 20.8）：
  $$\frac{y - y_1}{x - x_1} \cdot \frac{y - y_2}{x - x_2} = -1$$
  整理即得。当 $P = A$ 或 $P = B$ 时可直接验证方程成立。

  此圆的圆心为 $AB$ 的中点，半径为 $\dfrac{AB}{2}$。展开上式可得一般式。

- **方法 31.3（已知圆心与切线求圆方程）**
  若圆心为 $C(h, k)$，且圆与直线 $L: Ax + By + C_0 = 0$ 相切，则半径 $r$ 等于圆心到 $L$ 的距离：
  $$r = \frac{|Ah + Bk + C_0|}{\sqrt{A^2 + B^2}}$$

  代入标准方程即得 $(x-h)^2 + (y-k)^2 = r^2$。

## 31.2 圆的切线与公切线

### 31.2.1 直线与圆的位置关系

- **定义 31.3（点与圆的位置关系）**
  设圆心为 $C(h, k)$，半径为 $r$，点 $P(x_0, y_0)$ 到圆心的距离为 $d = PC$。则：
  $$\begin{aligned}
  d < r &\iff P \text{ 在圆内；} \\
  d = r &\iff P \text{ 在圆上；} \\
  d > r &\iff P \text{ 在圆外。}
  \end{aligned}$$

- **回顾：点到直线的距离**
  点 $P(x_0, y_0)$ 到直线 $L: Ax + By + C = 0$ 的距离为（定理 20.10）：
  $$d = \frac{|Ax_0 + By_0 + C|}{\sqrt{A^2 + B^2}}$$

- **方法 31.4（判定直线与圆位置关系的策略）**
  处理直线与圆位置关系的首选策略是**几何法**：计算圆心到直线的距离 $d$，与半径 $r$ 比较。
  $$\begin{aligned}
  d > r &\;\Longrightarrow\; \text{相离（0 个交点）} \\
  d = r &\;\Longrightarrow\; \text{相切（1 个交点）} \\
  d < r &\;\Longrightarrow\; \text{相交（2 个交点）}
  \end{aligned}$$

  几何法只需代入一次距离公式，比联立方程后计算判别式 $\Delta$ 更简便。仅在需要交点具体坐标（而非仅位置关系）时才使用判别式法作为补充。

  此策略的几何依据和判别式法的具体形式见以下两个定理。

- **定理 31.5（直线与圆的位置关系 — 几何法）**
  设圆心为 $C(h, k)$，半径为 $r > 0$，直线 $L: Ax + By + C = 0$，圆心到 $L$ 的距离为 $d$。则：

  | 位置关系 | 条件 | 交点数 |
  |:--------:|:----:|:------:|
  | **相离** | $d > r$ | 0 |
  | **相切** | $d = r$ | 1 |
  | **相交** | $d < r$ | 2 |

  **证明：** 过圆心 $C$ 作 $L$ 的垂线，垂足为 $H$，则 $CH = d$。在 $L$ 上任意取点 $Q$，由毕氏定理：
  $CQ^2 = CH^2 + HQ^2 = d^2 + HQ^2$。$Q$ 在圆上 $\iff CQ = r \iff HQ = \sqrt{r^2 - d^2}$。由此：
  - 若 $d > r$，则 $\sqrt{r^2 - d^2}$ 无实数值，无交点；
  - 若 $d = r$，则 $HQ = 0$，$H$ 即为唯一交点（切点）；
  - 若 $d < r$，则 $HQ = \sqrt{r^2 - d^2}$，$H$ 两侧各有一交点。$\square$

- **定理 31.6（直线与圆的交点 — 判别式法）** ⬆️
  将直线方程 $L$ 与圆方程联立，消去一个变量后得一元二次方程 $ax^2 + bx + c = 0$（或 $ay^2 + by + c = 0$）。设其判别式为 $\Delta = b^2 - 4ac$，则：

  | 条件 | 交点数 |
  |:----:|:------:|
  | $\Delta > 0$ | 2 |
  | $\Delta = 0$ | 1（相切） |
  | $\Delta < 0$ | 0 |

  **Remark：** 定理 31.5 的几何法更简便——距离 $d = r$ 直接给出相切条件，无需展开联立方程。但在涉及交点具体坐标（而非仅交点数量）的问题中，判别式法仍有用。

### 31.2.2 求解圆的切线方程

- **方法 31.5（求切线方程的通法）**
  求圆的切线方程的共同核心条件只有一个：
  > **圆心到直线的距离等于半径（$d = r$）。**

  无论切点是否已知，此条件均为求切线的根本依据。具体流程视已知条件而定：

  | 已知条件 | 策略 |
  |---------|------|
  | **切点 $P$ 在圆上** | 可用通法 $d = r$，亦可直接用定理 31.7 的公式（更快捷） |
  | **仅知圆外一点 $P$** | 设斜率 $\to$ 写出直线方程 $\to$ 代入 $d = r$ 解 $m$（定理 31.8） |
  | **仅知切线斜率 $m$** | 设切线为 $y = mx + c$ $\to$ 代入 $d = r$ 解 $c$ |

  此方法的深层原理：半径垂直于切线（定理 29.7），故圆心到切线的垂足即为切点，垂线长度即为半径。以下的定理 31.7–31.8 是本通法在不同情境下的具体展开。

- **定理 31.7（过圆上一点的切线方程）**
  设点 $P(x_0, y_0)$ 在圆 $(x - h)^2 + (y - k)^2 = r^2$ 上，则过 $P$ 的切线方程为：
  $$(x_0 - h)(x - h) + (y_0 - k)(y - k) = r^2$$

  **证明：** 设 $Q(x, y)$ 为切线上任意点（$Q \neq P$）。圆心 $C(h, k)$ 到 $P$ 的半径 $CP$ 垂直于切线 $PQ$（定理 29.7），故 $\triangle CPQ$ 为直角三角形，$CQ$ 为斜边。由毕氏定理：
  $$CQ^2 = CP^2 + PQ^2$$
  即
  $$(x - h)^2 + (y - k)^2 = r^2 + (x - x_0)^2 + (y - y_0)^2$$

  展开并消去 $x^2, y^2$：
  $$-2xh + h^2 - 2yk + k^2 = r^2 - 2xx_0 + x_0^2 - 2yy_0 + y_0^2$$

  移项整理：
  $$2x(x_0 - h) + 2y(y_0 - k) = x_0^2 + y_0^2 - h^2 - k^2 + r^2$$

  利用 $r^2 = (x_0 - h)^2 + (y_0 - k)^2$ 将右端化简：
  $$\begin{aligned}
  x_0^2 + y_0^2 - h^2 - k^2 + r^2 &= x_0^2 + y_0^2 - h^2 - k^2 + (x_0^2 - 2x_0h + h^2 + y_0^2 - 2y_0k + k^2) \\
  &= 2x_0^2 + 2y_0^2 - 2x_0h - 2y_0k \\
  &= 2x_0(x_0 - h) + 2y_0(y_0 - k)
  \end{aligned}$$

  因此：
  $$2x(x_0 - h) + 2y(y_0 - k) = 2x_0(x_0 - h) + 2y_0(y_0 - k)$$
  即
  $$(x_0 - h)(x - x_0) + (y_0 - k)(y - y_0) = 0$$

  将 $x - x_0 = (x - h) - (x_0 - h)$、$y - y_0 = (y - k) - (y_0 - k)$ 代入：
  $$(x_0 - h)\big[(x - h) - (x_0 - h)\big] + (y_0 - k)\big[(y - k) - (y_0 - k)\big] = 0$$
  $$(x_0 - h)(x - h) + (y_0 - k)(y - k) = (x_0 - h)^2 + (y_0 - k)^2 = r^2$$

  当 $Q = P$ 时可直接验证等式成立。$\square$

- **推论 31.2（原点为圆心的切线公式）**
  当圆心在原点 $(0, 0)$ 时，过圆 $x^2 + y^2 = r^2$ 上一点 $P(x_0, y_0)$ 的切线方程为：
  $$x_0 x + y_0 y = r^2$$

- **Remark（替代证法）：** 过圆上一点的切线亦可通过“半径斜率 × 切线斜率 = −1”（定理 20.8）求得。此即定理 31.7 的斜率形式。

- **定理 31.8（过圆外一点的切线方程）**
  设点 $P(x_0, y_0)$ 在圆 $(x - h)^2 + (y - k)^2 = r^2$ 的外部（即 $PC > r$）。求过 $P$ 的切线方程步骤：

  1. 设切线斜率为 $m$，方程为 $y - y_0 = m(x - x_0)$，即 $mx - y + (y_0 - mx_0) = 0$。
  2. 利用“圆心到切线的距离等于半径”（定理 31.5，$d = r$）：
     $$\frac{|mh - k + (y_0 - mx_0)|}{\sqrt{m^2 + 1}} = r$$
  3. 解此方程求 $m$，得切线斜率。每解出一个 $m$ 即对应一条切线。

  **警示：** 步骤 1 假设切线的斜率存在（即使用了点斜式），这排除了铅垂切线 $x = x_0$。若步骤 3 只解出一个 $m$，须检查 $x = x_0$ 是否也为切线——即检查点 $(x_0, k \pm r)$ 是否满足 $PC = r$（或直接检验圆心到 $x = x_0$ 的距离 $|x_0 - h|$ 是否等于 $r$）。

- **推论 31.3（圆外一点到圆的切线长度）**
  设点 $P$ 在圆的外侧（$PC > r$），则从 $P$ 到圆的任一切点 $T$ 的距离（即切线长）为：
  $$PT = \sqrt{PC^2 - r^2} = \sqrt{(x_0 - h)^2 + (y_0 - k)^2 - r^2}$$

  **证明：** 由半径垂直于切线，$\triangle PCT$ 为直角三角形，$PT^2 + CT^2 = PC^2$ 且 $CT = r$。$\square$

### 31.2.3 公切线

- **定义 31.4（公切线 / Common Tangent）**
  同时与两个圆相切的直线称为此两圆的**公切线**。
  - **外公切线：** 两圆在公切线的**同侧**（公切线不穿过两圆之间）；
  - **内公切线：** 两圆在公切线的**异侧**（公切线穿过两圆之间）。

  <br>

  | 示意 | 外离（4 条） | 外切（3 条） | 相交（2 条） | 内切（1 条） | 内含（0 条） |
  |:---:|:----------:|:----------:|:----------:|:----------:|:----------:|
  | 外 | 2 | 2 | 2 | 1 | 0 |
  | 内 | 2 | 1 | 0 | 0 | 0 |

- **方法 31.6（求公切线的通法——缩圆／扩圆法）**
  两圆公切线问题的核心策略是**化归**：将两圆的公切线问题转化为一个点到一个圆的切线问题（即回归到方法 31.5）。

  具体操作是固定其中一个圆，对另一个圆进行半径变换：
  $$\begin{aligned}
  \text{求外公切线：}&\quad \text{将小圆“缩为”一点，大圆半径变为 } r_1 - r_2 \\[4pt]
  \text{求内公切线：}&\quad \text{将小圆“反转”，大圆半径变为 } r_1 + r_2
  \end{aligned}$$

  缩圆／扩圆后，问题变为“求点 $O_2$ 到圆心为 $O_1$、半径为 $r_1 \mp r_2$ 的圆的切线”，即为方法 31.5 的标准情境。求出这些切线后，再将它们平行移回原位，即得原两圆的公切线。

  此策略不仅给出公切线的存在性判定（定理 31.9），也直接给出公切线长度公式（定理 31.10）——后者本质上就是缩圆／扩圆后直角三角形的毕氏定理。

- **定理 31.9（两圆位置与公切线数量）**
  设两圆的圆心距为 $d = |O_1O_2|$，半径分别为 $r_1$ 和 $r_2$，不妨设 $r_1 \ge r_2 > 0$。则：

  | 位置关系 | 几何条件 | 外公切线数 | 内公切线数 | 总数 |
  |:--------:|:--------:|:----------:|:----------:|:----:|
  | **外离** | $d > r_1 + r_2$ | 2 | 2 | **4** |
  | **外切** | $d = r_1 + r_2$ | 2 | 1 | **3** |
  | **相交** | $r_1 - r_2 < d < r_1 + r_2$ | 2 | 0 | **2** |
  | **内切** | $d = r_1 - r_2$ | 1 | 0 | **1** |
  | **内含** | $0 \le d < r_1 - r_2$ | 0 | 0 | **0** |

  **证明思路：**
  - **外公切线：** 将小圆“缩为”一点（将其半径从大圆半径中扣除），相当于求点 $O_2$ 到半径为 $r_1 - r_2$、圆心为 $O_1$ 的圆的切线。外公切线存在 $\iff O_1O_2 \ge r_1 - r_2$。当 $d > r_1 - r_2$ 时有两条，$d = r_1 - r_2$ 时有一条（两圆内切情况）。
  - **内公切线：** 将小圆“放大”（将其半径加到大圆半径上），相当于求点 $O_2$ 到半径为 $r_1 + r_2$、圆心为 $O_1$ 的圆的切线。内公切线存在 $\iff O_1O_2 \ge r_1 + r_2$。当 $d > r_1 + r_2$ 时有两条，$d = r_1 + r_2$ 时有一条。$\square$

- **定理 31.10（公切线长度公式）**
  若两圆的圆心距为 $d$，半径为 $r_1, r_2$（$r_1 \ge r_2$），且对应的公切线存在，则：
  $$\begin{aligned}
  \text{外公切线长度：}&\quad L_{\text{外}} = \sqrt{d^2 - (r_1 - r_2)^2} \\[6pt]
  \text{内公切线长度：}&\quad L_{\text{内}} = \sqrt{d^2 - (r_1 + r_2)^2}
  \end{aligned}$$

  **证明：** 以其中一个圆心为起点，作平行于公切线的辅助线。对于外公切线：自小圆圆心 $O_2$ 向大圆的与外公切线对应的半径作垂线，构造直角三角形，其斜边为 $d$，一腰为 $r_1 - r_2$，另一腰即为外公切线长度。由毕氏定理（定理 8.10）即得。内公切线同理，仅将 $r_1 - r_2$ 换为 $r_1 + r_2$。$\square$

## 第三十一章总结

1. **圆的标准方程：** 圆心 $C(h, k)$、半径 $r$ $\iff$ $(x-h)^2 + (y-k)^2 = r^2$（定义 31.1）。圆本质上是到定点距离为定长的动点轨迹（Ch30）。
2. **圆的一般方程：** $x^2 + y^2 + Dx + Ey + F = 0$（定义 31.2）。图形为圆的充要条件为 $D^2 + E^2 - 4F > 0$（定理 31.1），圆心为 $(-D/2, -E/2)$，半径为 $\frac{1}{2}\sqrt{D^2+E^2-4F}$（定理 31.2）。
3. **确定圆的方程：** 可通过三点（方法 31.1）、直径端点（方法 31.2）、圆心与切线（方法 31.3）等条件求出。
4. **点与圆的位置：** $PC < r$（圆内）、$PC = r$（圆上）、$PC > r$（圆外）（定义 31.3）。
5. **直线与圆的位置关系（通法：$d \gtrless r$）：** 优先使用几何法——计算圆心到直线距离 $d$，与半径 $r$ 比较（方法 31.4、定理 31.5）。仅在需要交点坐标时使用判别式法 $\Delta$（定理 31.6，⬆️）。
6. **切线方程（通法：$d = r$）：** 所有切线问题的根本条件是圆心到直线距离等于半径（方法 31.5）。过圆上一点的切线可用公式 $(x_0-h)(x-h)+(y_0-k)(y-k)=r^2$（定理 31.7）；过圆外一点则设斜率 $m$，代入 $d=r$ 求解，注意检查铅垂切线（定理 31.8）。切线长度为 $\sqrt{PC^2-r^2}$（推论 31.3）。
7. **公切线（通法：缩圆／扩圆法）：** 将两圆公切线问题化归为点到圆的切线问题（方法 31.6）。外公切线对应缩圆（$r_1-r_2$），内公切线对应扩圆（$r_1+r_2$）。公切线数量由 $d$ 与 $r_1 \pm r_2$ 的大小关系决定（定理 31.9），长度由毕氏定理导出（定理 31.10）。

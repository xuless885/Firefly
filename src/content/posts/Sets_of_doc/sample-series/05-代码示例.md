### 代码块

```python
def quadratic_formula(a, b, c):
    """求解二次方程 ax² + bx + c = 0"""
    discriminant = b**2 - 4*a*c
    if discriminant < 0:
        return None
    elif discriminant == 0:
        return -b / (2*a)
    else:
        import math
        sqrt_d = math.sqrt(discriminant)
        return (
            (-b + sqrt_d) / (2*a),
            (-b - sqrt_d) / (2*a)
        )
```

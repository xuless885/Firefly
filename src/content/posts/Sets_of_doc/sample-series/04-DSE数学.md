## 集合论基础

集合是数学中最基本的概念之一。一个**集合**（set）是一些确定的、互不相同的对象的全体。

### 常见集合记号

| 符号 | 含义 |
|------|------|
| $\mathbb{N}$ | 自然数集 |
| $\mathbb{Z}$ | 整数集 |
| $\mathbb{Q}$ | 有理数集 |
| $\mathbb{R}$ | 实数集 |
| $\varnothing$ | 空集 |

### 集合运算

两个集合 $A$ 和 $B$ 的**并集**（union）定义为：

$$
A \cup B = \{ x \mid x \in A \text{ 或 } x \in B \}
$$

**交集**（intersection）定义为：

$$
A \cap B = \{ x \mid x \in A \text{ 且 } x \in B \}
$$

### 例题

已知 $A = \{1, 2, 3, 4\}$，$B = \{3, 4, 5, 6\}$，求 $A \cup B$ 和 $A \cap B$。

:::ad-solution
**解：**

$$
\begin{aligned}
A \cup B &= \{1, 2, 3, 4, 5, 6\} \\
A \cap B &= \{3, 4\}
\end{aligned}
$$
:::
